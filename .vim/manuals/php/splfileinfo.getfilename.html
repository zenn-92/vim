SplFileInfo::getFilename

(PHP 5 >= 5.1.2)

SplFileInfo::getFilename — Получает имя файла

Описание

public string SplFileInfo::getFilename ( void )

Получает имя файла без информации о пути к нему.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Имя файла.

Примеры

Пример #1 Пример использования SplFileInfo::getFilename()

<?php
$info = new SplFileInfo('foo.txt');
var_dump($info->getFilename());

$info = new SplFileInfo('/path/to/foo.txt');
var_dump($info->getFilename());

$info = new SplFileInfo('http://www.php.net/');
var_dump($info->getFilename());

$info = new SplFileInfo('http://www.php.net/svn.php');
var_dump($info->getFilename());
?>

Результатом выполнения данного примера будет что-то подобное:

string(7) "foo.txt"
string(7) "foo.txt"
string(0) ""
string(7) "svn.php"
