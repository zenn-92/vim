mysqli_client_encoding

(PHP 5 < 5.4.0)

mysqli_client_encoding — Синоним функции mysqli_character_set_name()

Описание

Эта функция является синонимом функции mysqli_character_set_name().

Внимание

Данная функция была помечена УСТАРЕВШЕЙ начиная с версии PHP 5.3.0 и была УДАЛЕНА в версии PHP 5.4.0.
