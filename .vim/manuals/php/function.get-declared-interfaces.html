get_declared_interfaces

(PHP 5)

get_declared_interfaces — Возвращает массив всех объявленных интерфейсов

Описание

array get_declared_interfaces ( void )

Возвращает объявленные интерфейсы.

Возвращаемые значения

Возвращает массив имен всех объявленных интерфейсов в текущем скрипте.

Примеры

Пример #1 Пример использования get_declared_interfaces()

<?php
print_r(get_declared_interfaces());
?>

Результатом выполнения данного примера будет что-то подобное:

Array
(
    [0] => Traversable
    [1] => IteratorAggregate
    [2] => Iterator
    [3] => ArrayAccess
    [4] => reflector
    [5] => RecursiveIterator
    [6] => SeekableIterator
)

Смотрите также

  • interface_exists() - Проверяет, определен ли интерфейс
  • get_declared_classes() - Возвращает массив с именами объявленных классов
  • class_implements() - Возвращает список интерфейсов, реализованных в заданном классе
