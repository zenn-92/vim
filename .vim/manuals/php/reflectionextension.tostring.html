ReflectionExtension::__toString

(PHP 5)

ReflectionExtension::__toString — Преобразование в строку

Описание

public string ReflectionExtension::__toString ( void )

Выполняет экспорт расширения и возвращает его в виде строки string. Это то же самое, что и вызов ReflectionExtension::export() с аргументом return равным TRUE.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает экспортированное расширение в виде строки тем же образом, что и ReflectionExtension::export().
