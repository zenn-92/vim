imagefontwidth

(PHP 4, PHP 5)

imagefontwidth — Получение ширины шрифта

Описание

int imagefontwidth ( int $font )

Возвращает ширину символа шрифта.

Список параметров

font

Может принимать значения 1, 2, 3, 4, 5 для встроенных шрифтов в кодировке latin2 (более высокое число соответствует большему шрифту) или любому из ваших собственных идентификаторов шрифтов, зарегистрированных с помощью imageloadfont().

Возвращаемые значения

Возвращает ширину в пикселах

Примеры

Пример #1 Использование imagefontwidth() на встроенных шрифтах

<?php
echo 'Ширина шрифта: ' . imagefontwidth(4);
?>

Результатом выполнения данного примера будет что-то подобное:

Font width: 8

Пример #2 Использование imagefontwidth() вместе с imageloadfont()

<?php
// загрузка .gdf шрифта
$font = imageloadfont('anonymous.gdf');

echo 'Ширина шрифта: ' . imagefontwidth($font);
?>

Результатом выполнения данного примера будет что-то подобное:

Font width: 23

  • imageloadfont() - Загрузка шрифта
