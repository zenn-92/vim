GearmanTask::data

(PECL gearman >= 0.5.0)

GearmanTask::data — Получение данных из обработчика

Описание

public string GearmanTask::data ( void )

Получает данные, которые обработчик вернул по завершении работы.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Сериализованные данные или FALSE, если обработчик не предоставил данных.
