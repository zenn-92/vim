ArrayIterator::key

(PHP 5 >= 5.0.0)

ArrayIterator::key — Возвращает ключ текущего элемента массива

Описание

public mixed ArrayIterator::key ( void )

Эта функция возвращает ключ текущего элемента массива

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Ключ текущего элемента в массиве array.

Примеры

Пример #1 Пример использования ArrayIterator::key()

<?php
$array = array('key' => 'value');

$arrayobject = new ArrayObject($array);
$iterator = $arrayobject->getIterator();

echo $iterator->key(); //key
?>
