gmp_sign

(PHP 4 >= 4.0.4, PHP 5)

gmp_sign — Знак числа

Описание

int gmp_sign ( resource $a )

Проверяет знак числа.

Список параметров

a

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

Возвращаемые значения

Возвращает 1, если a положительное; -1, если a отрицательное; и 0, если a равно нулю.

Примеры

Пример #1 Пример использования gmp_sign()

<?php
// положительное
echo gmp_sign("500") . "\n";

// отрицательное
echo gmp_sign("-500") . "\n";

// ноль
echo gmp_sign("0") . "\n";
?>

Результат выполнения данного примера:

1
-1
0
