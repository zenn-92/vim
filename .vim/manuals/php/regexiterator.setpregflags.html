RegexIterator::setPregFlags

(PHP 5 >= 5.2.0)

RegexIterator::setPregFlags — Задание флагов регулярного выражения

Описание

public void RegexIterator::setPregFlags ( int $preg_flags )

Устанавливает флаги регулярного выражения.

Список параметров

preg_flags

Флаги настройки регулярного выражения. Список возможных флагов приведен в описании метода RegexIterator::__construct().

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования RegexIterator::setPregFlags()

Создает новый объект RegexIterator, отбирающий элементы массива, ключи которых начинаются со слова 'test'.

<?php
$test = array ('test 1', 'another test', 'test 123');

$arrayIterator = new ArrayIterator($test);
$regexIterator = new RegexIterator($arrayIterator, '/^test/', RegexIterator::GET_MATCH);

$regexIterator->setPregFlags(PREG_OFFSET_CAPTURE);

foreach ($regexIterator as $key => $value) {
    var_dump($value);
}
?>

Результатом выполнения данного примера будет что-то подобное:

array(1) {
  [0]=>
  array(2) {
    [0]=>
    string(4) "test"
    [1]=>
    int(0)
  }
}
array(1) {
  [0]=>
  array(2) {
    [0]=>
    string(4) "test"
    [1]=>
    int(0)
  }
}
