imageaffinematrixconcat

(PHP 5 >= 5.5.0)

imageaffinematrixconcat — Concat two matrices (as in doing many ops in one go)

Описание

array imageaffinematrixconcat ( array $m1 , array $m2 )

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

m1

Array with keys 0 to 5.

m2

Array with keys 0 to 5.

Возвращаемые значения

Array with keys 0 to 5 and float values или FALSE в случае возникновения ошибки.
