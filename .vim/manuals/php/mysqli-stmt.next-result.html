mysqli_stmt::next_result

mysqli_stmt_next_result

(PHP 5)

mysqli_stmt::next_result -- mysqli_stmt_next_result — Читает следующий набор строк из мультизапроса

Описание

Объектно-ориентированный стиль (method):

public bool mysqli_stmt::next_result ( void )

Процедурный стиль:

bool mysqli_stmt_next_result ( mysql_stmt $stmt )

Читает следующий набор строк из мультизапроса.

Список параметров

stmt

Только для процедурного стиля: Идентификатор выражения, полученный с помощью mysqli_stmt_init().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Ошибки

Вызывает ошибку уровня E_STRICT, если нет доступного результирующего набора. Следует проверять наличие доступных наборов функцией mysqli_stmt::more_results() непосредственно перед вызовом mysqli_stmt::next_result().
