DOMDocument::createComment

(PHP 5)

DOMDocument::createComment — Создает новый узел-комментарий

Описание

public DOMComment DOMDocument::createComment ( string $data )

Эта функция создает новый экземпляр класса DOMComment. Этот узел не будет отображаться в документе до тех пор, пока он не будет вставлен, например, функцией DOMNode::appendChild().

Список параметров

data

Содержимое комментария.

Возвращаемые значения

Новый объект класса DOMComment лиюо FALSE в случае ошибки.
