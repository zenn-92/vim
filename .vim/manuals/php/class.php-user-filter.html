Класс php_user_filter

(PHP 5)

Введение

Потомки этого класса передаются в функцию stream_filter_register().

Обзор классов

php_user_filter {
/* Свойства */
public $filtername ;
public $params ;
/* Методы */
public int filter ( resource $in , resource $out , int &$consumed , bool $closing )
public void onClose ( void )
public bool onCreate ( void )
}

Свойства

filtername

Имя фильтра, зарегистрированного функцией stream_filter_append().

params

Содержание

  • php_user_filter::filter — Вызывается, как только применяется фильтр
  • php_user_filter::onClose — Вызывается при закрытии фильтра
  • php_user_filter::onCreate — Вызывается при создании объекта фильтра
