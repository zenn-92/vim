Изменения в обработке INI-файлов

Следующие php.ini директивы были удалены:

  • register_globals и register_long_arrays
  • magic_quotes_gpc, magic_quotes_runtime и magic_quotes_sybase
  • allow_call_time_pass_reference
  • define_syslog_variables
  • highlight.bg
  • session.bug_compat_42 и session.bug_compat_warn
  • y2k_compliance
  • safe_mode, safe_mode_gid, safe_mode_include_dir, safe_mode_exec_dir, safe_mode_allowed_env_vars и safe_mode_protected_env_vars

Следующие php.ini директивы были добавлены:

  • cli.pager и cli.prompt для CLI SAPI, используя функцию readline в интерактивном режиме.
  • cli_server.color для включения ANSI-раскраски в консольном выводе встроенного веб-сервера для разработки.
  • max_input_vars - указывает, какое количество входных переменных GET/POST/COOKIE может быть принято.
  • zend.multibyte - для управления новой возможностью автоопределения закодированных Unicode скриптов и файлов.
  • zend.script_encoding - Это значение будет использоваться до тех пор, пока не встретится директива "declare(encoding=...)" в начале скрипта.
  • zend.signal_check - для проверки замененных обработчиков сигналов при завершении.
  • session.upload_progress.enabled, session.upload_progress.cleanup, session.upload_progress.prefix, session.upload_progress.name, session.upload_progress.freq, session.upload_progress.min_freq
  • enable_post_data_reading - Когда отключено, данные POST не читаются (и не обрабатываются)
  • windows_show_crt_warning - Эта директива включает отображение предупреждений Windows CRT. До этого момента отображение было включено по умолчанию.

Следующие php.ini директивы были изменены:

  • session.entropy_file теперь принимает значение по умолчанию /dev/random или /dev/urandom в зависимости от того, что было определено на этапе компиляции.
  • session.entropy_length теперь принимает значение по умолчанию 32.
