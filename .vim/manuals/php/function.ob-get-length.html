ob_get_length

(PHP 4 >= 4.0.2, PHP 5)

ob_get_length — Возвращает размер буфера вывода

Описание

int ob_get_length ( void )

Вернет размер содержимого в буфере вывода.

Возвращаемые значения

Возвращает размер содержимого буфера вывода или FALSE если буферизация не активна.

Примеры

Пример #1 Простой пример использования функции ob_get_length()

<?php

ob_start();

echo "Привет ";

$len1 = ob_get_length();

echo "Мир";

$len2 = ob_get_length();

ob_end_clean();

echo $len1 . ", ." . $len2;
?>

Результат выполнения данного примера:

7, 10
