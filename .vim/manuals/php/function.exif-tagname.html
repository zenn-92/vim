exif_tagname

(PHP 4 >= 4.2.0, PHP 5)

exif_tagname — Получение имени заголовка по его индексу

Описание

string exif_tagname ( int $index )

Список параметров

index

ID тэга, имя которого требуется получить.

Возвращаемые значения

Возвращает имя заголовка или FALSE, если index не является ранее определенным ID EXIF тэга.

Примеры

Пример #1 Пример использования exif_tagname()

<?php
echo "256: ".exif_tagname(256).PHP_EOL;
echo "257: ".exif_tagname(257).PHP_EOL;
?>

Результат выполнения данного примера:

256: ImageWidth
257: ImageLength
