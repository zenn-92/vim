SplObjectStorage::getInfo

(PHP 5 >= 5.3.0)

SplObjectStorage::getInfo — Возвращает данные ассоциированные с текущим объектом

Описание

public mixed SplObjectStorage::getInfo ( void )

Возвращает данные ассоциированные с объектом, на который указывает итератор.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Данные ассоциированные с объектом.

Примеры

Пример #1 Пример использования SplObjectStorage::getInfo()

<?php
$s = new SplObjectStorage();

$o1 = new StdClass;
$o2 = new StdClass;

$s->attach($o1, "d1");
$s->attach($o2, "d2");

$s->rewind();
while($s->valid()) {
    $index  = $s->key();
    $object = $s->current(); // аналогично current($s)
    $data   = $s->getInfo();

    var_dump($object);
    var_dump($data);
    $s->next();
}
?>

Результатом выполнения данного примера будет что-то подобное:

object(stdClass)#2 (0) {
}
string(2) "d1"
object(stdClass)#3 (0) {
}
string(2) "d2"

Смотрите также

  • SplObjectStorage::current() - Возвращает текущий объект
  • SplObjectStorage::rewind() - Переводит итератор на первый элемент контейнера
  • SplObjectStorage::key() - Возвращает индекс текущего положения итератора
  • SplObjectStorage::next() - Переход к следующему объекту
  • SplObjectStorage::valid() - Определяет, допустимо ли текущее значение итератора
  • SplObjectStorage::setInfo() - Ассоциирует данные с текущим объектом контейнера
