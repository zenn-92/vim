ReflectionClass::getExtensionName

(PHP 5)

ReflectionClass::getExtensionName — Возвращает имя расширения, определенного в классе

Описание

public string ReflectionClass::getExtensionName ( void )

Возвращает имя расширения, определенного в классе.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Имя расширения, определенного в классе, или FALSE для пользовательских классов.

Примеры

Пример #1 Пример использования ReflectionClass::getExtensionName()

<?php
$class = new ReflectionClass('ReflectionClass');
$extension = $class->getExtensionName();
var_dump($extension);
?>

Результат выполнения данного примера:

string(10) "Reflection"
