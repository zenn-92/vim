oci_field_type_raw

(PHP 5, PECL OCI8 >= 1.1.0)

oci_field_type_raw — Возвращает исходный Oracle тип поля

Описание

int oci_field_type_raw ( resource $statement , int $field )

Возвращает тип Oracle поля field.

Однако, если вы хотите получить тип поля, то oci_field_type() подойдет вам больше.

Список параметров

statement

Идентификатор допустимого OCI запроса.

field

Может быть номером поля (нумерация начинается с 1) или именем.

Возвращаемые значения

Возвращает исходный Oracle тип в виде строки, или FALSE при ошибках.

Примечания

Замечание:

В версиях PHP ниже 5.0.0 эта функция называлась ocicolumntyperaw(). В PHP 5.0.0 и выше ocicolumntyperaw() является алиасом oci_field_type_raw(), поэтому вы можете продолжать использовать это имя, однако это не рекомендуется.
