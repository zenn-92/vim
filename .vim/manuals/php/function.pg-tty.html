pg_tty

(PHP 4, PHP 5)

pg_tty — Возвращает имя терминала TTY, связанное с соединением

Описание

string pg_tty ([ resource $connection ] )

pg_tty() возвращает имя терминала, связанного с подключением PostgreSQL connection, на который выводится отладочная информация.

Замечание:

pg_tty() устарела, но оставлена для обратной совместимости.

Список параметров

connection

Ресурс подключения к базе данных PostgreSQL. Если параметр connection не задан, будет использовано подключение по умолчанию - последнее соединение, открытое функцией pg_connect() или pg_pconnect().

Возвращаемые значения

Имя TTY для подключения connection в виде строки (string) либо FALSE в случае ошибки.

Примеры

Пример #1 Пример использования pg_tty()

<?php
$pgsql_conn = pg_connect("dbname=mark host=localhost");

if ($pgsql_conn) {
   print "Server debug TTY is: " . pg_tty($pgsql_conn) . "<br/>\n";
} else {
   print pg_last_error($pgsql_conn);
   exit;
}
?>
