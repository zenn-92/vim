ReflectionFunction::__toString

(PHP 5)

ReflectionFunction::__toString — Представление в виде строки

Описание

public string ReflectionFunction::__toString ( void )

Представление в виде строки.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Результат выполнения схож с выводом ReflectionFunction::export().

Примеры

Пример #1 Пример использования ReflectionFunction::__toString()

<?php
function title($title, $name)
{
    return sprintf("%s. %s\r\n", $title, $name);
}

echo new ReflectionFunction('title');
?>

Результатом выполнения данного примера будет что-то подобное:

Function [ <user> function person ] {
  @@ Command line code 1 - 1

  - Parameters [2] {
    Parameter #0 [ <required> $prefix ]
    Parameter #1 [ <required> $name ]
  }
}
