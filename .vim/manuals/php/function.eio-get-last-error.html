eio_get_last_error

(PECL eio >= 1.0.0)

eio_get_last_error — Возвращает последнюю ошибку, связанную с указателем на ресурс

Описание

string eio_get_last_error ( resource $req )

eio_get_last_error() возвращает последнюю ошибку, связанную с указателем на ресурс req.

Список параметров

req

Указатель на ресурс

Возвращаемые значения

eio_get_last_error() возвращает строку, содержащую последнюю ошибку, связанную с указателем на ресурс req.

Внимание

Эта функция является ЭКСПЕРИМЕНТАЛЬНОЙ. Поведение этой функции, ее имя и относящаяся к ней документация могут измениться в последующих версиях PHP без уведомления. Используйте эту функцию на свой страх и риск.
