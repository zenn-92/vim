xslt_free

(PHP 4 >= 4.0.3)

xslt_free — Освобождение XSLT-процессора

Описание

void xslt_free ( resource $xh )

Освобождает ресурс, занятый XSLT-процессором.

Список параметров

xh

Идентификатор ссылки XSLT-процессора, созданный с помощью функции xslt_create().

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
