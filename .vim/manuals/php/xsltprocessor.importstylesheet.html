XSLTProcessor::importStylesheet

(PHP 5)

XSLTProcessor::importStylesheet — Импортирует таблицу стилей

Описание

void XSLTProcessor::importStylesheet ( object $stylesheet )

Этот метод импортирует таблицу стилей в XSLTProcessor для трансформации.

Список параметров

stylesheet

Импортируемая таблица стилей в виде объекта DOMDocument или SimpleXMLElement.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Список изменений

Версия Описание
5.2.8 Принимает SimpleXMLElement вновь, что было невозможно в версии PHP 5.2.6 и более поздних.
