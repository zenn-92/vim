GearmanTask::create

(PECL gearman <= 0.5.0)

GearmanTask::create — Создает задание (устаревший метод)

Описание

public GearmanTask GearmanTask::create ( void )

Возвращает новый объект класса GearmanTask.

Замечание:

Этот метод был удален в версии 0.6.0 расширения Gearman.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Объект класса GearmanTask или FALSE в случае возникновения ошибки.
