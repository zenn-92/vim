socket_clear_error

(PHP 4 >= 4.2.0, PHP 5)

socket_clear_error — Очищает ошибку на сокете или последний код ошибки

Описание

void socket_clear_error ([ resource $socket ] )

Эта функция очищает код ошибки на указанном сокете или последнюю глобальную ошибку сокета, если сокет не указан.

Эта функция позволяет принудительно сбросить значение кода ошибки для сокета или последний глобальный код ошибки расширения. Это может быть полезно для определения в части приложения, произошла ошибка или нет.

Список параметров

socket

Действующий ресурс сокета, созданный при помощи функции socket_create().

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Смотрите также

  • socket_last_error() - Возвращает последнюю ошибку на сокете
  • socket_strerror() - Возвращает строку, описывающую ошибку сокета
