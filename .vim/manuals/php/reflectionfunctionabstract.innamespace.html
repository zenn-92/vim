ReflectionFunctionAbstract::inNamespace

(PHP 5 >= 5.3.0)

ReflectionFunctionAbstract::inNamespace — Проверяет, находится ли функция в пространстве имен

Описание

public bool ReflectionFunctionAbstract::inNamespace ( void )

Проверяет, что функция определена в пространстве имен.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если функция определена в пространстве имен, FALSE в противном случае.
