Новые расширения

Следующие расширения были добавлены (по умолчанию) в PHP 5.2.0:

  • Filter - Проверяет и фильтрует данные. Разработан для работы с небезопасными данными, такими, как пользовательский ввод. Это расширение включено по умолчанию. Режим RAW (по умолчанию) не влияет на входные данные.
  • JSON - Реализует работу с форматом обмена данными the JavaScript Object Notation (JSON). Это расширение включено по умолчанию.
  • Zip - Позволяет прозрачно читать и писать ZIP-архивы и файлы в них.
