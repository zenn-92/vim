dbase_numrecords

(PHP 4, PHP 5)

dbase_numrecords — Получает количество записей в базе данных

Описание

int dbase_numrecords ( int $dbase_identifier )

Получает количество записей (строк) в указанной базе данных.

Замечание:

Записи базы данных нумеруются от 1 dbase_numrecords($db), тогда как поля от 0 до dbase_numfields($db)-1.

Список параметров

dbase_identifier

Идентификатор базы данных, возвращаемый функцией dbase_open() или dbase_create().

Возвращаемые значения

Количество записей в базе данных, или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Цикл по всем записям базы данных

<?php

// открываем в режиме для чтения
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
  $record_numbers = dbase_numrecords($db);
  for ($i = 1; $i <= $record_numbers; $i++) {   
      //выполнение каких-либо действий с записью
  }
}

?>
