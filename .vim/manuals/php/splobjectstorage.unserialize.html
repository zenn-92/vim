SplObjectStorage::unserialize

(PHP 5 >= 5.2.2)

SplObjectStorage::unserialize — Восстанавливает сериализованый контейнер из строки

Описание

public void SplObjectStorage::unserialize ( string $serialized )

Восстанавливает данные контейнера из строки и добавляет их в текущий контейнер.

Список параметров

serialized

Сериализованное представление контейнера.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования SplObjectStorage::unserialize()

<?php
$s1 = new SplObjectStorage;
$s2 = new SplObjectStorage;
$o = new StdClass;
$s1[$o] = "data";

$s2->unserialize($s1->serialize());

var_dump(count($s2));
?>

Результатом выполнения данного примера будет что-то подобное:

int(1)
