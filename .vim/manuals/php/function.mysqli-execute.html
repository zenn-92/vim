mysqli_execute

(PHP 5)

mysqli_execute — Синоним для mysqli_stmt_execute()

Описание

Эта функция является синонимом mysqli_stmt_execute().

Примечания

Замечание:

Функция mysqli_execute() устарела и будет удалена.
