SplFixedArray::getSize

(PHP 5 >= 5.3.0)

SplFixedArray::getSize — Получает размер массива

Описание

public int SplFixedArray::getSize ( void )

Получает размер массива.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает размер массива как integer.

Примеры

Пример #1 Пример использования SplFixedArray::getSize()

<?php
$array = new SplFixedArray(5);
echo $array->getSize()."\n";
$array->setSize(10);
echo $array->getSize()."\n";
?>

Результат выполнения данного примера:

5
10

Примечания

Замечание:

Данный метод эквивалентен SplFixedArray::count()
