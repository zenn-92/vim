DOMDocumentFragment::appendXML

(PHP 5 >= 5.1.0)

DOMDocumentFragment::appendXML — Добавление необработанных XML данных

Описание

public bool DOMDocumentFragment::appendXML ( string $data )

Добавляет необработанные XML данные в объект класса DOMDocumentFragment.

Это метод не входит в стандарт DOM. Он был разработан для упрощения процедуры добавления XML DocumentFragment в DOMDocument.

Если следовать стандарту, необходимо создавать временный DOMDocument с фиктивным корнем, а затем проходить по всем узлам потомкам корневого узла XML документа и добавлять их по очереди.

Список параметров

data

XML для добавления.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Добавление XML данных в документ

<?php
$doc = new DOMDocument();
$doc->loadXML("<root/>");
$f = $doc->createDocumentFragment();
$f->appendXML("<foo>text</foo><bar>text2</bar>");
$doc->documentElement->appendChild($f);
echo $doc->saveXML(); 
?>

Результат выполнения данного примера:

<?xml version="1.0"?>
<root><foo>text</foo><bar>text2</bar></root>
