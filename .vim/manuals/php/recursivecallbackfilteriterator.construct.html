RecursiveCallbackFilterIterator::__construct

(PHP 5 >= 5.4.0)

RecursiveCallbackFilterIterator::__construct — Создает объект класса RecursiveCallbackFilterIterator на основе объекта RecursiveIterator

Описание

public RecursiveCallbackFilterIterator::__construct ( RecursiveIterator $iterator , string $callback )

Создает фильтрующий итератор на основе объекта-итератора RecursiveIterator, используя фунцкию callback для отбора нужных элементов.

Список параметров

iterator

Рекурсивный итератор, к которому нужно применить фильтр.

callback

Callback-функция, которая должна возвращать TRUE, если текущий элемент подходит под условия фильтра, и FALSE, если элемент нужно исключить из перебора. См. Примеры.

Может быть любым допустимым callable значением.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
