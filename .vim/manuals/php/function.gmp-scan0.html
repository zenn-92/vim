gmp_scan0

(PHP 4 >= 4.0.4, PHP 5)

gmp_scan0 — Поиск нуля в числе

Описание

int gmp_scan0 ( resource $a , int $start )

Сканирует a, начиная с бита start, пока не найдет бит установленный в 0.

Список параметров

a

Число для сканирования.

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

start

Бит начала сканирования.

Возвращаемые значения

Возвращает индекс найденного бита в виде числа типа integer. Индексация начинается с нуля.

Примеры

Пример #1 Пример использования gmp_scan0()

<?php
// "0" бит найдет на позиции 3. поиск начинается с 0
$s1 = gmp_init("10111", 2);
echo gmp_scan0($s1, 0) . "\n";

// "0" бит найдет на позиции 7. поиск начинается с 5
$s2 = gmp_init("101110000", 2);
echo gmp_scan0($s2, 5) . "\n";
?>

Результат выполнения данного примера:

3
7
