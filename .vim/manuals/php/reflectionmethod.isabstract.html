ReflectionMethod::isAbstract

(PHP 5)

ReflectionMethod::isAbstract — Проверяет, является ли метод абстрактным

Описание

public bool ReflectionMethod::isAbstract ( void )

Проверяет, является ли метод абстрактным.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если метод абстрактный, FALSE в противном случае.
