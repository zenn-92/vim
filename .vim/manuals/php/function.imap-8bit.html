imap_8bit

(PHP 4, PHP 5)

imap_8bit — Конвертирует 8-битную строку в quoted-printable строку

Описание

string imap_8bit ( string $string )

Конвертирует 8-битную строку в quoted-printable строку (в соответствии с » RFC2045, раздел 6.7).

Список параметров

string

8-битная строка для конвертации

Возвращаемые значения

Возвращает quoted-printable строку.
