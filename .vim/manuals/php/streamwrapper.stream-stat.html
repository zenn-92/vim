streamWrapper::stream_stat

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::stream_stat — Получение информации о файловом ресурсе

Описание

public array streamWrapper::stream_stat ( void )

Этот метод вызывается в результате вызова функции fstat().

Список параметров

У этой функции нет параметров.

Возвращаемые значения

См. stat().

Ошибки

Вызывает ошибку уровня E_WARNING, если вызов к этому методу не удался (например, не реализован).

Смотрите также

  • stat() - Возвращает информацию о файле
  • streamwrapper::url_stat() - Получение информации о файле
