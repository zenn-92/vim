GearmanClient::error

(PECL gearman >= 0.5.0)

GearmanClient::error — Вернуть строку ошибки для последней встретившейся ошибки

Описание

public string GearmanClient::error ( void )

Возвращает строку ошибки для последней встретившейся ошибки.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Удобочитаемая строка ошибки.
