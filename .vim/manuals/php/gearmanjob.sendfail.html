GearmanJob::sendFail

(PECL gearman >= 0.6.0)

GearmanJob::sendFail — Отправка статуса провала операции

Описание

public bool GearmanJob::sendFail ( void )

Посылает сообщение о провале обработки, причем причина этого провала известна (в отличие от провала в результате возникновения исключительной ситуации).

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
