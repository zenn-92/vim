LimitIterator::seek

(PHP 5 >= 5.1.0)

LimitIterator::seek — Перемещает итератор на заданную позицию

Описание

public int LimitIterator::seek ( int $position )

Перемещает указатель на заданную позицию position.

Список параметров

position

Позиция, на которую требуется переместить указатель.

Возвращаемые значения

Возвращает смещение от начальной позиции после перемещения.

Ошибки

Выбрасывает исключение OutOfBoundsException, если заданная позиция выходит за пределы, переданные конструктору LimitIterator::__construct().
