ArrayObject::count

(PHP 5 >= 5.0.0)

ArrayObject::count — Возвращает количество публичных свойств ArrayObject

Описание

public int ArrayObject::count ( void )

Возвращает количество публичных свойств ArrayObject.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Количество публичных свойств ArrayObject.

Замечание:

Если ArrayObject получен из массива, то все свойства являются общедоступными (public).

Примеры

Пример #1 Пример использования ArrayObject::count()

<?php
class Example {
    public $public = 'prop:public';
    private $prv   = 'prop:private';
    protected $prt = 'prop:protected';
}

$arrayobj = new ArrayObject(new Example());
var_dump($arrayobj->count());

$arrayobj = new ArrayObject(array('first','second','third'));
var_dump($arrayobj->count());
?>

Результат выполнения данного примера:

int(1)
int(3)
