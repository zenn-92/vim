ReflectionFunctionAbstract::returnsReference

(PHP 5)

ReflectionFunctionAbstract::returnsReference — Проверяет, что функция возвращает ссылку

Описание

public bool ReflectionFunctionAbstract::returnsReference ( void )

Проверяет, что функция возвращает ссылку.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если функция возвращает ссылку, FALSE в противном случае.
