SplFixedArray::__construct

(PHP 5 >= 5.3.0)

SplFixedArray::__construct — Создает новый массив фиксированной длины

Описание

public SplFixedArray::__construct ([ int $size = 0 ] )

Создает фиксированный массив с числом NULL значений, равных size.

Список параметров

size

Размер фиксированного массива. Ожидается значение между 0 и PHP_INT_MAX.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Ошибки

Выбрасывает исключение InvalidArgumentException, когда параметр size отрицателен.

Вызывает ошибку уровня E_WARNING, когда size нельзя распознать как число.

Примеры

Пример #1 Пример использования SplFixedArray::__construct()

<?php
$array = new SplFixedArray(5);

$array[1] = 2;
$array[4] = "foo";

foreach($array as $v) {
  var_dump($v);
}
?>

Результат выполнения данного примера:

NULL
int(2)
NULL
NULL
string(3) "foo"
