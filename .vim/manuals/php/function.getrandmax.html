getrandmax

(PHP 4, PHP 5)

getrandmax — Вовзращает максимально возможное случайное число

Описание

int getrandmax ( void )

Возвращает максимальное значение, которое может быть получено функцией rand().

Возвращаемые значения

Максимально возможное случайное значение, возвращаемое функцией rand()

Смотрите также

  • rand() - Генерирует случайное число
  • srand() - Изменяет начальное число генератора псевдослучайных чисел
  • mt_getrandmax() - Показывает максимально возможное значение случайного числа
