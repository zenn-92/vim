ParentIterator::__construct

(PHP 5 >= 5.1.0)

ParentIterator::__construct — Конструктор класса ParentIterator

Описание

public ParentIterator::__construct ( RecursiveIterator $iterator )

Создает новый объект класса ParentIterator на основе объекта-итератора.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

iterator

Объект-итератор, на основе которого создается новый объект.
