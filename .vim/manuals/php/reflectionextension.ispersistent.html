ReflectionExtension::isPersistent

(PHP >= 5.4.0)

ReflectionExtension::isPersistent — Определяет, является ли расширение постоянным

Описание

public void ReflectionExtension::isPersistent ( void )

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE для расширений, загруженных посредствомextension, FALSE в противном случае.
