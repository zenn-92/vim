xslt_set_scheme_handler

(PHP 4 >= 4.0.5)

xslt_set_scheme_handler — Задает обработчики схемы для XSLT-процессора

Описание

void xslt_set_scheme_handler ( resource $xh , array $handlers )

Задает обработчики схемы для ресурса XSLT-процессора xh.

Список параметров

xh

Идентификатор ссылки XSLT-процессора, созданный с помощью функции xslt_create().

handlers

Обработчики схемы должны быть представлены в виде массива в следующем формате (все элементы являются необязательными): 

array(
[get_all] => обработчик get all,
[open] => обработчик open,
[get] => обработчик get,
[put] => обработчик put,
[close] => обработчик close
)

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
