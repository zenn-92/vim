SimpleXMLIterator::key

(PHP 5 >= 5.1.0)

SimpleXMLIterator::key — Возвращает текущий ключ

Описание

mixed SimpleXMLIterator::key ( void )

Этот метод получает имя XML-тега текущего элемента.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает имя XML-тега элемента, на который ссылается текущий объект SimpleXMLIterator, или FALSE

Примеры

Пример #1 Получает имя текущего XML-тега

<?php
$xmlIterator = new SimpleXMLIterator('<books><book>Основы PHP</book><book>Основы XML</book></books>');

echo var_dump($xmlIterator->key());
$xmlIterator->rewind(); // возврат к первому элементу
echo var_dump($xmlIterator->key());

?>

Результат выполнения данного примера:

bool(false)
string(4) "book"
