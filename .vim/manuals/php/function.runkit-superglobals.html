runkit_superglobals

(PECL runkit >= 0.7.0)

runkit_superglobals — Возвращает индексный массив зарегистрированных суперглобальных переменных.

Описание

array runkit_superglobals ( void )

Возвращаемые значения

Возвращает индексный массив зарегистрированных переменных, например: _GET, _POST, _REQUEST, _COOKIE, _SESSION, _SERVER, _ENV, _FILES
