php_ini_loaded_file

(PHP 5 >= 5.2.4)

php_ini_loaded_file — Получение адреса загруженного php.ini файла

Описание

string php_ini_loaded_file ( void )

Проверяет, загружен ли файл php.ini, и возвращает его адрес.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Путь к загруженному файлу php.ini или FALSE, если файл не загружен.

Примеры

Пример #1 Пример использования php_ini_loaded_file()

<?php
$inipath = php_ini_loaded_file();

if ($inipath) {
    echo 'Загруженный php.ini: ' . $inipath;
} else {
    echo 'Файл php.ini не загружен';
}
?>

Результатом выполнения данного примера будет что-то подобное:

Loaded php.ini: /usr/local/php/php.ini
