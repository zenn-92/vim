stream_set_chunk_size

(PHP 5 >= 5.4.0)

stream_set_chunk_size — Задание размера фрагмента данных потока

Описание

int stream_set_chunk_size ( resource $fp , int $chunk_size )

Устанавливает размер фрагмента обрабатываемых данных потока.

Список параметров

fp

Поток.

chunk_size

Желаемый размер фрагмента.

Возвращаемые значения

Возвращает предыдущий размер фрагмента в случае успеха.

Вернет FALSE, если chunk_size меньше 1 или больше PHP_INT_MAX.

Ошибки

Вызывает ошибку уровня E_WARNING, если chunk_size меньше 1 или больше PHP_INT_MAX.
