php_ini_scanned_files

(PHP 4 >= 4.3.0, PHP 5)

php_ini_scanned_files — Возвращает список .ini файлов найденных в дополнительной ini директории

Описание

string php_ini_scanned_files ( void )

php_ini_scanned_files() возвращает список разделенных запятыми конфигурационных файлов, проанализированных после php.ini. Это файлы, обнаруженные в директории, которая была задана при компиляции настройкой --with-config-file-scan-dir .

Возвращаемые имена файлов включают путь в том же виде, как он задан в настройке --with-config-file-scan-dir .

Возвращаемые значения

При успехе возвращает строку с разделенными запятыми именами .ini файлов За каждой запятой следует новая строка. Если директива --with-config-file-scan-dir не задана, функция вернет FALSE. Если директива задана, и директория пуста, будет возвращена пустая строка. Если файл не удается распознать, он все же будет включен в список, но при этом будет PHP сообщит об ошибке. Эта ошибка будет возникать как при компиляции, так и во время работы php_ini_scanned_files().

Примеры

Пример #1 Простой пример вывода списка ini файлов

<?php
if ($filelist = php_ini_scanned_files()) {
    if (strlen($filelist) > 0) {
        $files = explode(',', $filelist);

        foreach ($files as $file) {
            echo "<li>" . trim($file) . "</li>\n";
        }
    }
}
?>
