Классы для работы с потоками

Пользовательские обертки можно разрабатывать, используя определения классов показанные на этой странице руководства, а затем регистрировать функцией stream_wrapper_register().

Класс php_user_filter предопределен и является абстрактным базовым классом для создания пользовательских фильтров. Дополнительную информацию, касающуюся их реализации, можно почерпнуть из руководства к функции stream_filter_register().
