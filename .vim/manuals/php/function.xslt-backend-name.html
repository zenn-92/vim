xslt_backend_name

(PHP 4 >= 4.3.0)

xslt_backend_name — Возвращает имя расширения

Описание

string xslt_backend_name ( void )

xslt_backend_name() получает имя расширения (backend).

Возвращаемые значения

Возвращает Sablotron.

Примеры

Пример #1 Пример использования xslt_backend_name()

<?php

echo xslt_backend_name(); // Sablotron

?>

Смотрите также

  • xslt_backend_info() - Возвращает информацию о настройках, заданных при компиляции расширения (backend)
  • xslt_backend_version() - Возвращает номер версии Sablotron
