sqlite_valid

SQLiteResult::valid

SQLiteUnbuffered::valid

(PHP 5 < 5.4.0)

sqlite_valid -- SQLiteResult::valid -- SQLiteUnbuffered::valid — Проверяет, есть ли еще доступные ряды

Описание

bool sqlite_valid ( resource $result )

Объектно-ориентированный стиль (метод):

bool SQLiteResult::valid ( void )
bool SQLiteUnbuffered::valid ( void )

Проверяет, есть ли еще доступные ряды в указанном результирующем наборе.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Возвращаемые значения

Возвращает TRUE, если в результирующем наборе result еще есть доступные ряды, иначе возвращается FALSE.

Смотрите также

  • sqlite_num_rows() - Возвращает количество записей в результате запроса
  • sqlite_changes() - Возвращает количество записей, которые были изменены последним SQL-запросом
