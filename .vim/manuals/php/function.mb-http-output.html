mb_http_output

(PHP 4 >= 4.0.6, PHP 5)

mb_http_output — Установка/получение кодировки символов HTTP вывода

Описание

mixed mb_http_output ([ string $encoding ] )

Установка/получение кодировки символов HTTP вывода. Выходные данные после работы этой функции будут преобразованы к кодировке encoding.

Список параметров

encoding

Если задан аргумент encoding, mb_http_output() устанавливает кодировку выходных символов HTTP в encoding.

Если аргумент encoding опущен, mb_http_output() возвращает текущую кодировку символов HTTP вывода.

Возвращаемые значения

Если аргумент encoding опущен, mb_http_output() возвращает текущую кодировку символов HTTP вывода. В противном случае, Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
