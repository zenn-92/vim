call_user_method_array

(PHP 4 >= 4.0.5, PHP 5)

call_user_method_array — Вызывает метод объекта с массивом параметров [устаревшее]

Описание

mixed call_user_method_array ( string $method_name , object &$obj , array $params )
Внимание

Функция call_user_method_array() устарела с выходом 4.1.0.

Список параметров

method_name

Имя вызываемого метода.

obj

Объект (object), содержащий вызываемый метод method_name.

params

Массив параметров.

Примеры

Пример #1 Альтернатива call_user_method_array()

<?php
call_user_func_array(array($obj, $method_name), $params);
call_user_func_array(array(&$obj, $method_name), $params); // PHP 4
?>

  • call_user_func() - Вызывает пользовательскую функцию, указанную в первом параметре
