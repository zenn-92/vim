DirectoryIterator::valid

(PHP 5)

DirectoryIterator::valid — Проверяет, является ли текущий элемент DirectoryIterator допустимым файлом

Описание

public bool DirectoryIterator::valid ( void )

Проверяет, является ли текущий элемент DirectoryIterator допустимым файлом.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE если элемент ялвяется допустимым, иначе FALSE

Примеры

Пример #1 Пример использования DirectoryIterator::valid()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));

// Loop to end of iterator
while($iterator->valid()) {
    $iterator->next();
}

$iterator->valid(); // FALSE
$iterator->rewind(); 
$iterator->valid(); // TRUE

?>
