SphinxClient::getLastWarning

(PECL sphinx >= 0.1.0)

SphinxClient::getLastWarning — Получение последнего предупреждения

Описание

public string SphinxClient::getLastWarning ( void )

Возвращает последнее сообщение с предупреждением. Если во время предыдущего вызова API предупреждений не было, возвращается пустая строка. Этот метод не обнуляет сообщение о предупреждении, таким образом, его можно вызывать несколько раз.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает последнее сообщение с предупреждением или пустую строку, если предупреждений не было.
