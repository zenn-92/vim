SplFileInfo::getLinkTarget

(PHP 5 >= 5.2.2)

SplFileInfo::getLinkTarget — Получение пути по ссылке

Описание

public string SplFileInfo::getLinkTarget ( void )

Получает целевой путь по ссылке файловой системы.

Замечание:

Этот путь может не быть реальным расположением в файловой системе. Чтобы получить реальный путь файловой системы, используйте метод SplFileInfo::getRealPath().

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает целевой путь по ссылке файловой системы.

Ошибки

В случае ошибки выбрасывает исключение RuntimeException.

Примеры

Пример #1 Пример использования SplFileInfo::getLinkTarget()

<?php
$info = new SplFileInfo('/Users/bbieber/workspace');
if ($info->isLink()) {
    var_dump($info->getLinkTarget());
    var_dump($info->getRealPath());
}
?>

Результатом выполнения данного примера будет что-то подобное:

string(19) "Documents/workspace"
string(34) "/Users/bbieber/Documents/workspace"

Смотрите также

  • SplFileInfo::isLink() - Сообщает, является ли файл ссылкой
  • SplFileInfo::getRealPath() - Определяет абсолютный путь к файлу
