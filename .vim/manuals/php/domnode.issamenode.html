DOMNode::isSameNode

(PHP 5)

DOMNode::isSameNode — Проверяет, являются ли два узла одним и тем же узлом

Описание

public bool DOMNode::isSameNode ( DOMNode $node )

Эта функция проверяет, являются ли два узла одним и тем же узлом. Данное сравнение происходит не на основе содержимого узлов.

Список параметров

node

Сравниваемый узел.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
