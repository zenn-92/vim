Предопределенные константы

Перечисленные ниже константы всегда доступны как часть ядра PHP.

PHP_OUTPUT_HANDLER_START (integer)

Служит признаком того, что буферирование вывода началось.

PHP_OUTPUT_HANDLER_WRITE (integer)

Служит признаком того, что буфер вывода очищается и в нем находятся данные для вывода.

Доступно начиная с PHP 5.4.

PHP_OUTPUT_HANDLER_FLUSH (integer)

Обозначает, что буфер был очищен.

Доступно начиная с PHP 5.4.

PHP_OUTPUT_HANDLER_CLEAN (integer)

Обозначает, что буфер был опустошен.

Доступно начиная с PHP 5.4.

PHP_OUTPUT_HANDLER_FINAL (integer)

Обозначает, что это последняя операция буферирования.

Доступно начиная с PHP 5.4.

PHP_OUTPUT_HANDLER_CONT (integer)

Обозначает, что буфер был очищен, но буферирование вывода будет продолжено.

Начиная с PHP 5.4 это синоним для PHP_OUTPUT_HANDLER_WRITE.

PHP_OUTPUT_HANDLER_END (integer)

Обозначает, что буферирование вывода завершено.

Начиная с PHP 5.4 это синоним для PHP_OUTPUT_HANDLER_FINAL.
