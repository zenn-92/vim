OCI-Lob::read

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob::read — Возвращает часть объекта LOB

Описание

string OCI-Lob::read ( int $length )

Читает и возвращает length байт данных с текущей позиции объекта LOB.

Чтение останавливается, если длина прочтенных данных равна length или если достигнут конец объекта. Внутренний указатель перемещается на столько байт, сколько было прочитано.

Список параметров

length

Длина считываемых данных, в байтах.

Возвращаемые значения

Возвращает содержимое в виде строки, или FALSE в случае возникновения ошибки.
