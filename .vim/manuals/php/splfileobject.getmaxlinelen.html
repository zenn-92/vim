SplFileObject::getMaxLineLen

(PHP 5 >= 5.1.0)

SplFileObject::getMaxLineLen — Получает максимальную длину строки

Описание

public int SplFileObject::getMaxLineLen ( void )

Получает максимальную длину строки, установленную с помощью SplFileObject::setMaxLineLen().

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает максимальную длину строки, если она была установлена с помощью SplFileObject::setMaxLineLen(), по умолчанию 0.

Примеры

Пример #1 Пример использования SplFileObject::getMaxLineLen()

<?php
$file = new SplFileObject("file.txt");
var_dump($file->getMaxLineLen());

$file->setMaxLineLen(20);
var_dump($file->getMaxLineLen());
?>

Результатом выполнения данного примера будет что-то подобное:

int(0)
int(20)
