PDOStatement::getAttribute

(PHP 5 >= 5.1.0, PECL pdo >= 0.2.0)

PDOStatement::getAttribute — Получение значения атрибута запроса PDOStatement

Описание

mixed PDOStatement::getAttribute ( int $attribute )

Получает значение атрибута запроса. На данный момент не существует общих атрибутов, есть только характерные для конкретного драйвера:

  • PDO::ATTR_CURSOR_NAME (характерен для Firebird и ODBC): Получение имени курсора для запроса UPDATE ... WHERE CURRENT OF.

Возвращаемые значения

Возвращает значение атрибута.

Смотрите также

  • PDO::getAttribute() - Получить атрибут соеденения с базой данных
  • PDO::setAttribute() - Присвоение атрибута
  • PDOStatement::setAttribute() - Присваивает атрибут объекту PDOStatement
