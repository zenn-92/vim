realpath_cache_size

(PHP 5 >= 5.3.2)

realpath_cache_size — Получает размер кэша реального пути

Описание

int realpath_cache_size ( void )

Получает объем памяти, используемой при кэшировании реального пути.

Возвращаемые значения

Возвращает объем использованной памяти для кэширования реального пути.

Примеры

Пример #1 Пример использования realpath_cache_size()

<?php
var_dump(realpath_cache_size());
?>

Результатом выполнения данного примера будет что-то подобное:

int(412)
