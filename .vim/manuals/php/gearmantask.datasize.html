GearmanTask::dataSize

(PECL gearman >= 0.5.0)

GearmanTask::dataSize — Получение размера возвращаемых данных

Описание

public int GearmanTask::dataSize ( void )

Возвращает размер данных, которые выдал обработчик.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Разме данных или FALSE, если данных предоставлено не было.
