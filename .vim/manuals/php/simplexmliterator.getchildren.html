SimpleXMLIterator::getChildren

(PHP 5 >= 5.1.0)

SimpleXMLIterator::getChildren — Возвращает вложенные элементы текущего элемента

Описание

SimpleXMLIterator SimpleXMLIterator::getChildren ( void )

Этот метод возвращает объект SimpleXMLIterator, содержащий вложенные элементы текущего элемента SimpleXMLIterator.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает объект SimpleXMLIterator, содержащий вложенные элементы текущего элемента.

Примеры

Пример #1 Возвращает вложенные элементы текущего элемента

$xml = <<<XML
<books>
    <book>
        <title>Основы PHP</title>
        <author>Джим Смит</author>
    </book>
    <book>Основы XML</book>
</books>
XML;

$xmlIterator = new SimpleXMLIterator($xml);
for( $xmlIterator->rewind(); $xmlIterator->valid(); $xmlIterator->next() ) {
    foreach($xmlIterator->getChildren() as $name => $data) {
    echo "$name: '$data' из класса " . get_class($data) . "\n";
    }
}
?>

Результат выполнения данного примера:

title: 'Основы PHP' из класса SimpleXMLIterator
author: 'Джим Смит' из класса SimpleXMLIterator
