Введение

С помощью расширения EXIF вы можете работать с мета-данными изображений. Например, вы можете использовать EXIF функции для чтения метаданных из фотографий снятых на цифровые камеры. Этим функции работают с информацией, хранящейся в заголовках JPEG и TIFF изображений.
