Imagick::getVersion

(PECL imagick 2.0.0)

Imagick::getVersion — Возвращает версию API ImageMagick

Описание

array Imagick::getVersion ( void )

Возвращает версию API ImageMagick как строку и число.

Возвращаемые значения

Возвращает версию API ImageMagick как строку и число.

Ошибки

Вызывает ImagickException при ошибке.
