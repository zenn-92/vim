Введение

Этот модуль содержит интерфейс к функциям, определенным в стандарте IEEE 1003.1 (POSIX.1), которые не доступны с помощью других средств.

Внимание

Конфиденциальные данные могут быть получены с помощью POSIX функций, например posix_getpwnam() и других. Ни одна из POSIX функций не осуществляет какой либо проверки доступа когда включен безопасный режим. Поэтому строго рекомендуется отключать POSIX расширения вообще (используйте --disable-posix в вашей строке конфигурации) если вы работаете в таких условиях.

Замечание: Для Windows-платформ это расширение недоступно.
