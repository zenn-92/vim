php_user_filter::onClose

(PHP 5)

php_user_filter::onClose — Вызывается при закрытии фильтра

Описание

public void php_user_filter::onClose ( void )

Этот метод вызывается, когда фильтр завершает работу (обычно вместе с закрытием потока), и выполняется после вызова метода flush. Если какие-либо ресурсы были выделены во время выполнения метода onCreate(), то это время, когда их пора уничтожать.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращаемое значение игнорируется.
