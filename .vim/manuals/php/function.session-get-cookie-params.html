session_get_cookie_params

(PHP 4, PHP 5)

session_get_cookie_params — Возвращает параметры cookie сессии

Описание

array session_get_cookie_params ( void )

Возвращает параметры cookie сессии.

Возвращаемые значения

Возвращает массив с информацией о cookie текущей сессии, который содержит следующие элементы:

  • "lifetime" - Время жизни cookie в секундах.
  • "path" - Путь, где размещена хранимая информация.
  • "domain" - Домен cookie.
  • "secure" - cookie должны передаваться только через безопасные соединения.
  • "httponly" - cookie могут быть доступны только по протоколу HTTP.

Список изменений

Версия Описание
5.2.0 Добавлен элемент "httponly" в возвращаемый массив.
4.0.4 Добавлен элемент "secure" в возвращаемый массив.
