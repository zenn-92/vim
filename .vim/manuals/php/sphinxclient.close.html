SphinxClient::close

(PECL sphinx >= 1.0.3)

SphinxClient::close — Закрытие ранее открытого постоянного соединения

Описание

public bool SphinxClient::close ( void )

Закрывает ранее открытое постоянное соединение.

Список параметров

У этой функции нет параметров.

Список изменений

PECL/sphinx Версия Описание
1.0.3 Добавлен метод SphinxClient::close(). Доступен только если модуль cкомпилирован с версией libsphinxclient >= 0.9.9.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
