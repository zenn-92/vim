xslt_set_object

(PHP 4 >= 4.3.0)

xslt_set_object — Задает объект, в котором будут реализованы callback-функции

Описание

bool xslt_set_object ( resource $processor , object &$obj )

Эта функция позволяет использовать processor внутри объекта object и реализовать все callback-функции как методы этого объекта.

Callback-функции могут задаваться с помощью функций xslt_set_sax_handlers(), xslt_set_scheme_handlers() или xslt_set_error_handler() и быть объявлены как методы объекта object.

Список параметров

xh

Идентификатор ссылки XSLT-процессора, созданный с помощью функции xslt_create().

obj

Объект.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Использование собственного обработчика ошибок в качестве метода

<?php

class my_xslt_processor {

    var $_xh; // наш XSLT-процессор

    function my_xslt_processor()
    {
        $this->_xh = xslt_create();

        // внедряем процессор в объект $this
        xslt_set_object($this->_xh, $this);

        // задаем обработчик ошибок
        xslt_set_error_handler($this->_xh, "my_xslt_error_handler");
    }

    function my_xslt_error_handler($handler, $errno, $level, $info)
    {
        // проверим список аргументов
        var_dump(func_get_args());
    }
}

?>
