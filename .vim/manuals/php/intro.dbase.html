Введение

Эти функции позволяют получить доступ к записям, хранящимся в DBase-формате (DBF) баз данных.

База данных dBase является просто последовательностью файлов с фиксированной длиной записи. Записи добавляются в конец файла и удаленные записи будут (физически) хранится до вызова dbase_pack().

Доступные типы полей dBase:

Типы полей
Поле dBase тип Формат Дополнительная информация
M Memo n/a Тип не поддерживается PHP, такое поле будет игнорироваться
D Date YYYYMMDD Длина поля установлена в 8
N Number Число Вы должны указать длину и точность (количество цифр после десятичной точки)
C String A string Вы должны указать длину. При получении данных, строка будет дополнена справа пробелами до заявленной длины.
L Boolean T или Y соответствует TRUE, F или N соответствует FALSE Хранится и возвращается как целое число (1 или 0)
F Float Число с плавающей точкой Поддержка этого типа поля была добавлена в PHP 5.2.0

Замечание:

Это расширение было перемещено в репозиторий » PECL и больше не поставляется с PHP 5.3.0.

Внимание

Нет никакой поддержки для индексов или мемо-полей. Также нет поддержки захвата записи (lock). Два параллельных процесса веб-сервера изменив один файл DBase, весьма вероятно, разрушат вашу базу данных.

Рекомендуется не использовать DBase файлы в качестве основной базы данных приложения. Выберете любой SQL сервер, например: » MySQL или » Postgres поддерживаемые PHP. Поддержка dBase в PHP реализована для импорта и экспорта данных в основную базу данных приложения, потому что формат файлов dBase обычно хорошо понимается Windows, известными электронными таблицами и вашим руководством.
