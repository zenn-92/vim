GearmanTask::uuid

(PECL gearman <= 0.5.0)

GearmanTask::uuid — Получение уникального идентификатора задания (устаревший метод)

Описание

public string GearmanTask::uuid ( void )

Возвращает уникальный идентификатор этого задания. Этот идентификатор присваивает GearmanClient, в отличие от идентификатора объекта "работы", который присваивает сервер задач.

Замечание:

Этот метод был заменен на GearmanTask::unique() в версии 0.6.0 расширения Gearman.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Уникальный идентификатор или FALSE, если никакого идентификатора не присвоено.
