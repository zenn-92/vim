OCI-Collection::max

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Collection::max — Возвращает максимальное количество элементов в коллекции

Описание

int OCI-Collection::max ( void )

Возвращает максимальное количество элементов в коллекции.

Возвращаемые значения

Возвращает максимальное количество как целое число или FALSE в случае ошибки.

Если будет возвращен 0, то количество элементов не ограничено.
