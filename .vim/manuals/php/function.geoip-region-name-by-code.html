geoip_region_name_by_code

(PECL geoip >= 1.0.4)

geoip_region_name_by_code — Возвращает название региона для выбраной страны и кода региона

Описание

string geoip_region_name_by_code ( string $country_code , string $region_code )

Функция geoip_region_name_by_code() возвращает название региона соответствующей страны и кода региона.

В США, код региона соответствует двухбувенной абревиации названия штата. В Канаде, код региона соответствует двухбуквенной абривиации имени провинции или территории, присвоенный почтовой службой Канады.

Для остального мира, GeoIP использует коды FIPS 10-4 для представления региона. См. также » http://www.maxmind.com/app/fips10_4 -- детальный список кодов FIPS 10-4 .

Данная функция всегда доступна при использовании библиотеки GeoIP версии 1.4.1 и выше. Данные берутся непосредственно из библиотеки GeoIP, а не из какой-либо базы.

Список параметров

country_code

Двухбуквенный код страны geoip_country_code_by_name())

region_code

Двухбуквенный (или циферный) код региона geoip_region_by_name())

Возвращаемые значения

Возвращает название региона в случае успехе или FALSE, если страна и код региона не найдены в базе данных.

Примеры

Пример #1 Пример использования geoip_region_name_by_code() (Для Канады/США)

Будет выведеено название региона для страны CA (Канада), региона QC (Квебек).

<?php
$region = geoip_region_name_by_code('CA', 'QC');
if ($region) {
    echo 'Region name for CA/QC is: ' . $region;
}
?>

Результат выполнения данного примера:

Region name for CA/QC is: Quebec

Пример #2 Пример использования geoip_region_name_by_code()(коды FIPS)

Будет выведено имя региона для страны JP (Япония), региона 01 (Айти).

<?php
$region = geoip_region_name_by_code('JP', '01');
if ($region) {
    echo 'Region name for JP/01 is: ' . $region;
}
?>

Результат выполнения данного примера:

Region name for JP/01 is: Aichi
