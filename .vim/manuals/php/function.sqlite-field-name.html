sqlite_field_name

SQLiteResult::fieldName

SQLiteUnbuffered::fieldName

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_field_name -- SQLiteResult::fieldName -- SQLiteUnbuffered::fieldName — Возвращает имя указанного поля

Описание

string sqlite_field_name ( resource $result , int $field_index )

Объектно-ориентированный стиль (метод):

string SQLiteResult::fieldName ( int $field_index )
string SQLiteUnbuffered::fieldName ( int $field_index )

При передаче номера поля field_index функция sqlite_field_name() возвращает имя это поля из результата запроса result.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

field_index

Порядковый номер поля в результирующем наборе.

Возвращаемые значения

Возвращает имя поля в результирующем наборе SQLite по указанному порядковому номеру поля, или FALSE в случае ошибки.

Имена столбцов, возвращаемые SQLITE_ASSOC и SQLITE_BOTH, будут приведены к нужному регистру согласно значению конфигурационной опции sqlite.assoc_case.
