DOMDocument::save

(PHP 5)

DOMDocument::save — Сохраняет XML дерево из внутреннего представления в файл

Описание

public int DOMDocument::save ( string $filename [, int $options ] )

Создает XML документ из DOM представления. Эту функцию обычно вызывают после построения нового DOM документа, как показано в примере ниже.

Список параметров

filename

Путь к файлу в который будет сохранен XML доумент.

options

Дополнительные настройки. На данный момент поддерживается только LIBXML_NOEMPTYTAG.

Возвращаемые значения

Возвращает число записанных байт или FALSE в случае ошибки.

Список изменений

Версия Описание
5.1.0 Добавлен аргумент options

Примеры

Пример #1 Сохранение DOM дерева в файл

<?php

$doc = new DOMDocument('1.0');
// мы хотим красивый вывод
$doc->formatOutput = true;

$root = $doc->createElement('book');
$root = $doc->appendChild($root);

$title = $doc->createElement('title');
$title = $root->appendChild($title);

$text = $doc->createTextNode('This is the title');
$text = $title->appendChild($text);

echo 'Записано: ' . $doc->save("/tmp/test.xml") . ' байт'; // Записано: 72 байт

?>
