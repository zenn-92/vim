swf_labelframe

(PHP 4)

swf_labelframe — Label the current frame

Описание

void swf_labelframe ( string $name )

Labels the current frame with the given name.

Список параметров

name

The frame label.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
