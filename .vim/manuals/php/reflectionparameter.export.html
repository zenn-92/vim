ReflectionParameter::export

(PHP 5)

ReflectionParameter::export — Экспорт

Описание

public static string ReflectionParameter::export ( string $function , string $parameter [, bool $return ] )

Экспорт.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

function

Имя функции.

parameter

Имя аргумента.

return

Установка в TRUE вернет экспортируемое значение, в отличие от поведения, где этот параметр опущен. Установка в FALSE (по умолчанию) сделает противоположное.

Возвращаемые значения

Экспорт отображения.
