PDF_close

(PHP 4, PECL pdflib >= 1.0.0)

PDF_close — Закрывает pdf-документ [устаревшее]

Описание

bool PDF_close ( resource $p )

Закрывает сгенерированный PDF-файл и освобождает все относящиеся к документу ресурсы. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Эта функция устарела начиная с 6-ой версии PDFlib, вместо нее используйте PDF_end_document().
