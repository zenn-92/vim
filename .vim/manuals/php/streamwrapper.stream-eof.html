streamWrapper::stream_eof

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::stream_eof — Проверяет достижение конца файла по файловому указателю

Описание

public bool streamWrapper::stream_eof ( void )

Этот метод вызывается в результате вызова feof().

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Должен вернуть TRUE, если позиция чтения/записи находится в конце потока и доступных для чтения данных больше нет. В остальных случаях возвращается FALSE.

Примечания

Внимание

При чтении файла целиком (например, функцией file_get_contents()), PHP будет вызываеть streamWrapper::stream_read() и вместе с ним streamWrapper::stream_eof() в цикле, пока streamWrapper::stream_read() возвращает непустую строку. Возвращаемое из streamWrapper::stream_eof() значение при этом игнорируется.
