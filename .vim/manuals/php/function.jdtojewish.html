jdtojewish

(PHP 4, PHP 5)

jdtojewish — Переводит число дней в Юлианском летоисчислении в дату по Иудейскому календарю

Описание

string jdtojewish ( int $juliandaycount [, bool $hebrew = false [, int $fl = 0 ]] )

Переводит число дней в Юлианском летоисчислении в дату по Иудейскому календарю.

Список параметров

julianday

Номер дня в Юлианском летоисчислении

hebrew

Если аргумент hebrew установлен в TRUE, дата будет возвращена в виде строки в Иудейском формате заданном аргументом fl.

fl

Доступные форматы вывода даты: CAL_JEWISH_ADD_ALAFIM_GERESH, CAL_JEWISH_ADD_ALAFIM, CAL_JEWISH_ADD_GERESHAYIM.

Возвращаемые значения

Дата по Иудейскому календарю в виде строки формата "месяц/день/год"

Список изменений

Версия Описание
5.0.0 Добавлен аргумент fl.
4.3.0 Добавлен аргумент hebrew.

Примеры

Пример #1 Пример использования jdtojewish()

<?php
echo jdtojewish(gregoriantojd(10, 8, 2002), true,
       CAL_JEWISH_ADD_GERESHAYIM + CAL_JEWISH_ADD_ALAFIM + CAL_JEWISH_ADD_ALAFIM_GERESH); 
?>
