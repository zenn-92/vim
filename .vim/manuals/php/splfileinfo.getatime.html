SplFileInfo::getATime

(PHP 5 >= 5.1.2)

SplFileInfo::getATime — Получает время последнего обращения к файлу

Описание

public int SplFileInfo::getATime ( void )

Получает время последнего обращения к файлу.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает время последнего доступа к файлу.

Ошибки

Выбрасывает RunTimeException в случае ошибки.

  • fileatime() - Возвращает время последнего доступа к файлу
