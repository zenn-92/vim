Требования

Расширение Hash не требует дополнительных библиотек, включено по умолчанию в PHP 5.1.2. Оно может быть отключено с использованием --disable-hash при конфигурации. Для включения расширения Hash в более ранних версиях PHP необходимо установить » модуль PECL.
