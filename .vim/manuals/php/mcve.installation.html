Установка

Это расширение » PECL не поставляется вместе с PHP.

Информация по установке этого расширения PECL может быть найдена в главе руководства Установка PECL расширений. Дополнительная информация, такая как новые версии, скачивание, исходные файлы, информация о разработчике и CHANGELOG, может быть найдена здесь: » http://pecl.php.net/package/mcve.

DLL для этого расширения PECL в данный момент недоступна. См. также раздел сборка на Windows.

При установке без указания пути к libmonetra, PHP будет пытаться использовать путь LibMonetra по умолчанию (/usr/local). Если Monetra (MCVE) установлена по нестандартному пути, то сконфигурируйте установку следующим образом: --with-mcve=$mcve_path , где $mcve_path это путь установки MCVE/Monetra. Обратите внимание, что MCVE/Monetra требует существование каталогов $mcve_path/lib и $mcve_path/include, и включенный mcve.h, или monetra.h в каталоге include и libmcve.so, и/или libmcve.a, и/или libmonetra.so, и/или libmonetra.a в каталоге lib.

Так как MCVE/Monetra имеет разделение на сервер/клиент, то нет дополнительных требований для запуска PHP с MCVE. Для проверки вашего расширения MCVE/Monetra в PHP, можно подключиться к testbox.monetra.com на порт 8333 для IP, или на порт 8444 для SSL, используя MCVE/Monetra PHP API. Используйте 'vitale' как имя пользователя и 'test' как пароль. Дополнительная информация доступна на » http://www.mainstreetsoftworks.com/.
