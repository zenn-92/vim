ReflectionClass::getFileName

(PHP 5)

ReflectionClass::getFileName — Возвращает имя файла, в котором объявлен класс

Описание

public string ReflectionClass::getFileName ( void )

Возвращает имя файла, в котором объявлен класс.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает имя файла, в котором объявлен класс. Если класс объявлен в ядре PHP или в расширении PHP, возвращает FALSE.
