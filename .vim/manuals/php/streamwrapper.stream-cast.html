streamWrapper::stream_cast

(PHP 5 >= 5.3.0)

streamWrapper::stream_cast — Получает ресурс уровнем ниже

Описание

public resource streamWrapper::stream_cast ( int $cast_as )

Этот метод вызывается в результате вызова stream_select().

Список параметров

cast_as

Может быть STREAM_CAST_FOR_SELECT, когда stream_select() вызывает stream_cast(), либо STREAM_CAST_AS_STREAM, когда stream_cast() вызывается для других целей.

Возвращаемые значения

Должен вернуть ресурс потока, лежащего уровнем ниже, или FALSE.
