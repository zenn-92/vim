SphinxClient::getLastError

(PECL sphinx >= 0.1.0)

SphinxClient::getLastError — Получение последнего сообщения об ошибке

Описание

public string SphinxClient::getLastError ( void )

Возвращает строку, содержащую последнее сообщение об ошибке. Если во время предыдущего вызова API ошибок не было, возвращается пустая строка. Этот метод не обнуляет сообщение об ошибке, таким образом, его можно вызывать несколько раз.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает последнее сообщение об ошибке или пустую строку в случае их отсутствия.
