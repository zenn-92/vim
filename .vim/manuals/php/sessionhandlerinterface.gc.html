SessionHandlerInterface::gc

(PHP 5 >= 5.4.0)

SessionHandlerInterface::gc — Очищает старые сессии

Описание

abstract public bool SessionHandlerInterface::gc ( string $maxlifetime )

Очищает истекшие сессии. Вызывается функцией session_start(), и основывается на опциях session.gc_divisor, session.gc_probability и session.gc_lifetime.

Список параметров

maxlifetime

Сессии, которые не обновлялись в течении maxlifetime секунд будут удалены.

Возвращаемые значения

Возвращаемое значение сессионного хранилища (обычно TRUE в случае успеха, FALSE в случае ошибки). Данное значение возвращается обратно в PHP для внутренней обработки.
