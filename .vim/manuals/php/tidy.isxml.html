tidy::isXml

tidy_is_xml

(PHP 5, PECL tidy >= 0.5.2)

tidy::isXml -- tidy_is_xml — Определяет, является ли документ общим XML-документом (не HTML/XHTML)

Описание

Объектно-ориентированный стиль

bool tidy::isXml ( void )

Процедурный стиль

bool tidy_is_xml ( tidy $object )

Определяет, является ли документ общим XML-документом (не HTML/XHTML).

Список параметров

object

Объект Tidy.

Возвращаемые значения

Эта функция возвращает TRUE если указанный tidy-объект object является общим XML-документом (не HTML/XHTML), или FALSE в противном случае.

Внимание

Эта функция еще не реализована в самом Tidylib, поэтому всегда возвращается FALSE.
