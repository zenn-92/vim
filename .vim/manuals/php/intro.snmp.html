Введение

Расширение SNMP предоставляет простой и удобный набор инструментов для управления удаленными устройствами через Simple Network Management Protocol (простой протокол управления сетями).

Так как данное расширение является оберткой над библиотекой Net-SNMP (или UCD-SNMP на старых системах), все основные концепции идентичны и функции PHP меняют их поведение в зависимости от конфигурации Net-SNMP и переменных среды.

Начиная с версии PHP 5.4, доступен как OpenOffice API.

Больше информации о Net-SNMP можно найти по адресу » http://www.net-snmp.org/.
