Поведение этих функций зависит от установок в php.ini.

Краткое разъяснение конфигурационных директив.

define_syslog_variables boolean

Позволяет задать, необходимо ли определять различные переменные syslog (н-р. $LOG_PID, $LOG_CRON, и т.д.). Отключение определения этих переменных позволяет повысить производительность. Вы можете определить эти переменные при запуске с помощью функции define_syslog_variables().
