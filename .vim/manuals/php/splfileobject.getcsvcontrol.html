SplFileObject::getCsvControl

(PHP 5 >= 5.2.0)

SplFileObject::getCsvControl — Получает символы разделителя и ограничителя для CSV

Описание

public array SplFileObject::getCsvControl ( void )

Получает символы разделителя и ограничителя, используемые для разбора данных CSV.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает индексный массив, содержащий символы разделителя и ограничителя.

Примеры

Пример #1 Пример использования SplFileObject::getCsvControl()

<?php
$file = new SplFileObject("data.txt");
print_r($file->getCsvControl());
?>

Результатом выполнения данного примера будет что-то подобное:

Array
(
    [0] => ,
    [1] => "
)
