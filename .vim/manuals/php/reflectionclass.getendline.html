ReflectionClass::getEndLine

(PHP 5)

ReflectionClass::getEndLine — Возвращает номер последней строки

Описание

public int ReflectionClass::getEndLine ( void )

Возвращает номер последней строки определения пользовательского класса.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Номер последней строки определения пользовательского класса или FALSE, если номер неизвестен.

Примеры

Пример #1 Пример использования ReflectionClass::getEndLine()

<?php
// Тестовый класс
class TestClass { }

$rc = new ReflectionClass('TestClass');

echo $rc->getEndLine();
?>
