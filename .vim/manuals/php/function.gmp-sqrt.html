gmp_sqrt

(PHP 4 >= 4.0.4, PHP 5)

gmp_sqrt — Вычисление квадратного корня

Описание

resource gmp_sqrt ( resource $a )

Вычисляет квадратный корень числа a.

Список параметров

a

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

Возвращаемые значения

Целая часть корня в виде GMP числа.

Примеры

Пример #1 Пример использования gmp_sqrt()

<?php
$sqrt1 = gmp_sqrt("9");
$sqrt2 = gmp_sqrt("7");
$sqrt3 = gmp_sqrt("1524157875019052100");

echo gmp_strval($sqrt1) . "\n";
echo gmp_strval($sqrt2) . "\n";
echo gmp_strval($sqrt3) . "\n";
?>

Результат выполнения данного примера:

3
2
1234567890
