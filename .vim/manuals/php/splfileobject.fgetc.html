SplFileObject::fgetc

(PHP 5 >= 5.1.0)

SplFileObject::fgetc — Читает символ из файла

Описание

public string SplFileObject::fgetc ( void )

Читает символ из файла.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает строку, содержащую один символ, прочитанный из файла, или FALSE при достижении EOF.

Внимание

Эта функция может возвращать как boolean FALSE, так и не-boolean значение, которое приводится к FALSE. За более подробной информацией обратитесь к разделу Булев тип. Используйте оператор === для проверки значения, возвращаемого этой функцией.

Примеры

Пример #1 Пример использования SplFileObject::fgetc()

<?php
$file = new SplFileObject('file.txt');
while (false !== ($char = $file->fgetc())) {
    echo "$char\n";
}
?>
