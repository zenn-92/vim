apache_get_version

(PHP 4 >= 4.3.2, PHP 5)

apache_get_version — Возвращает версию Apache

Описание

string apache_get_version ( void )

Возвращает версию Apache.

Возвращаемые значения

Возвращает версию Apache или FALSE в случае возникновения ошибки.

Список изменений

Версия Описание
4.3.4 Стала доступной при использовании Apache 1.
5.0.0 Стала доступной при использовании Apache 2 filter API.

Примеры

Пример #1 Пример apache_get_version()

<?php
$version = apache_get_version();
echo "$version\n";
?>

Результатом выполнения данного примера будет что-то подобное:

Apache/1.3.29 (Unix) PHP/4.3.4
