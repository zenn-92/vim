RecursiveTreeIterator::getEntry

(PHP 5 >= 5.3.0)

RecursiveTreeIterator::getEntry — Получение поддерева, корнем которого является текущий элемент

Описание

public string RecursiveTreeIterator::getEntry ( void )

Получает поддерево, корнем которого является текущий элемент.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает поддерево, корнем которого является текущий элемент.
