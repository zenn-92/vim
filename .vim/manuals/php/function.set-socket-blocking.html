set_socket_blocking

(PHP 4, PHP 5)

set_socket_blocking — Псевдоним stream_set_blocking()

Описание

Эта функция является псевдонимом: stream_set_blocking()

Внимание

Этот псевдоним УСТАРЕЛ начиная с версии PHP 5.3.0. Крайне не рекомендуется полагаться на этот псевдоним.
