ImagickPixelIterator::newPixelIterator

(PECL imagick 2.0.0)

ImagickPixelIterator::newPixelIterator — Возвращает новый итератор пикселей

Описание

bool ImagickPixelIterator::newPixelIterator ( Imagick $wand )
Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Возвращает новый итератор пикселей.

Возвращаемые значения

В случае успешной работы возвращает TRUE. Исключение ImagickPixelIteratorException.
