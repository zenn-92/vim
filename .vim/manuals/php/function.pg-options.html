pg_options

(PHP 4, PHP 5)

pg_options — Получение параметров соединения с сервером баз данных

Описание

string pg_options ([ resource $connection ] )

pg_options() возвращает строку, содержащую параметры соединения PostgreSQL connection.

Список параметров

connection

Ресурс подключения к базе данных PostgreSQL. Если параметр connection не задан, будет использовано подключение по умолчанию - последнее соединение, открытое функцией pg_connect() или pg_pconnect().

Возвращаемые значения

Строка, содержащая параметры соединения connection, либо FALSE в случае ошибки.

Примеры

Пример #1 Пример использования pg_options()

<?php
   $pgsql_conn = pg_connect("dbname=mark host=localhost");
   echo pg_options($pgsql_conn);
?>
