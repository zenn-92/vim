spl_autoload_extensions

(PHP 5 >= 5.1.2)

spl_autoload_extensions — Регистрация и вывод расширений файлов для spl_autoload

Описание

string spl_autoload_extensions ([ string $file_extensions ] )

Эта функция может задавать расширения файлов, в которых callback-фунция __autoload() будет искать описания классов и интерфейсов. spl_autoload() будет вызывать метод __autoload и передавать ему эти расширения. Также эта фунция может выводить уже зарегистрированные расширения файлов.

Список параметров

file_extensions

При вызове без параметров фунция просто выведет список зарегистрированных на данный момент расширений, перечисленных через запятую. Чтобы изменить этот список, необходимо вызывать фунцию, передав ей строку с расширениями, также перечисленными через запятую.

Возвращаемые значения

Список перечисленных через запятую расширений файлов для функции spl_autoload().
