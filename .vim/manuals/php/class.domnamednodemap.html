Класс DOMNamedNodeMap

(PHP 5)

Обзор классов

DOMNamedNodeMap implements Traversable {
/* Свойства */
readonly public int $length ;
/* Методы */
DOMNode getNamedItem ( string $name )
DOMNode getNamedItemNS ( string $namespaceURI , string $localName )
DOMNode item ( int $index )
}

Свойства

length

Количество узлов в отображении. Диапазон допустимых индексов узлов-потомков от 0 до length - 1 включительно.

Содержание

  • DOMNamedNodeMap::getNamedItem — Извлекает узел с заданным именем
  • DOMNamedNodeMap::getNamedItemNS — Извлекает узел с заданным локальным именем и URI пространства имен
  • DOMNamedNodeMap::item — Извлекает узел с заданным индексом
