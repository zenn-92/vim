ReflectionExtension::__construct

(PHP 5)

ReflectionExtension::__construct — Создаёт объект класса ReflectionExtension

Описание

ReflectionExtension::__construct ( string $name )

Создаёт объект (object) класса ReflectionExtension.

Список параметров

name

Имя модуля.

Возвращаемые значения

Объект (object) класса ReflectionExtension.

Примеры

Пример #1 Пример использования ReflectionExtension

<?php
$ext = new ReflectionExtension('Reflection');

printf('Модуль: %s (версия: %s)', $ext->getName(), $ext->getVersion());
?>

Результатом выполнения данного примера будет что-то подобное:

Модуль: Reflection (версия: $Revision: 316929 $)
