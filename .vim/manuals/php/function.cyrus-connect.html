cyrus_connect

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_connect — Установить соединение с сервером Cyrus IMAP

Описание

resource cyrus_connect ([ string $host [, string $port [, int $flags ]]] )

Устанавливает соединение с сервером Cyrus IMAP.

Список параметров

host

Имя хоста Cyrus IMAP сервера.

port

Номер порта.

flags

Возвращаемые значения

Возвращает дескриптор соединения в случае успеха или FALSE в случае возникновения ошибки.
