GearmanTask::taskNumerator

(PECL gearman >= 0.5.0)

GearmanTask::taskNumerator — Получение числителя дроби, отвечающей за процент выполненной работы

Описание

public int GearmanTask::taskNumerator ( void )

Возвращает числитель дроби, которая используется для вычисления процента выполненной работы.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Число от 0 до 100 или FALSE, если значение определить не удалось.
