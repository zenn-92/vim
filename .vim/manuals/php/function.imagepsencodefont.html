imagepsencodefont

(PHP 4, PHP 5)

imagepsencodefont — Изменение вектора кодировки шрифта

Описание

bool imagepsencodefont ( resource $font_index , string $encodingfile )

Загружает вектор кодировки из файла и подменяет им текущий вектор шрифта. Так как вектор по умолчанию шрифтов PostScript не знает о символах после 127го, функция может оказаться полезной, когда используется язык отличный от английского.

Если вы заметили, что пользуетесь этой функцией довольно часто, лучшей альтернативой будет задать кодировку в настройке ps.default_encoding файла конфигурации. В этом случае все загружаемые шрифты будут иметь нужную кодировку автоматически.

Список параметров

font_index

Ресурс шрифта, возвращаемый функцией imagepsloadfont().

encodingfile

Формат этого файла описан в документации к T1lib библиотеке. Эта библиотека представлена двумя файлами: IsoLatin1.enc и IsoLatin2.enc.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования imagepsencodefont()

<?php
// Загрузка файла шрифта .pfb 
$font = imagepsloadfont('./px3l.pfb');

// Скажем T1lib использовать ISO Latin 1 кодировку
imagepsencode($font, './IsoLatin1.enc');

// тут могут быть какие-либо операции с шрифтом

// Освобождение памяти
imagepsfreefont($font);
?>

Примечания

Замечание: Эта функция доступна только в случае, если PHP был скомпилирован с опцией --with-t1lib[=DIR] .
