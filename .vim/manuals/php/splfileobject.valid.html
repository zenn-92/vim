SplFileObject::valid

(PHP 5 >= 5.1.0)

SplFileObject::valid — Проверяет, достигнут ли конец файла (EOF)

Описание

public bool SplFileObject::valid ( void )

Проверяет, был ли достигнут конец файла (EOF).

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE если EOF не достигнут, FALSE - в противном случае.

Примеры

Пример #1 Пример использования SplFileObject::valid()

<?php
// Цикл по файлу
$file = new SplFileObject("file.txt");
while ($file->valid()) {
    echo $file->fgets();
}
?>
