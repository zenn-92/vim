SessionHandlerInterface::close

(PHP 5 >= 5.4.0)

SessionHandlerInterface::close — Закрывает сессию

Описание

abstract public bool SessionHandlerInterface::close ( void )

Закрывает текущую сессию. Эта функция автоматически выполняется при закрытии сессии или явно через session_write_close().

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращаемое значение сессионного хранилища (обычно TRUE в случае успеха, FALSE в случае ошибки). Данное значение возвращается обратно в PHP для внутренней обработки.
