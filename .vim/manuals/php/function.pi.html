pi

(PHP 4, PHP 5)

pi — Возвращает число Пи

Описание

float pi ( void )

Возвращает приближённое значение числа Пи с точностью, определяемой директивой precision в php.ini, значение по умолчанию которой 14. Константа M_PI даёт идентичный результат.

Возвращаемые значения

Приближенное значение числа Пи, представленное как float.

Примеры

Пример #1 Пример использования pi()

<?php
echo pi(); // 3.1415926535898
echo M_PI; // 3.1415926535898
?>
