Iterator::valid

(PHP 5 >= 5.0.0)

Iterator::valid — Проверка корректности позиции

Описание

abstract public boolean Iterator::valid ( void )

Этот метод вызывается после Iterator::rewind() и Iterator::next() чтобы проверить, корректна ли текущая позиция.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращаемое значение будет приведено к типу boolean и затем использовано. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
