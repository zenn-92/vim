DirectoryIterator::isWritable

(PHP 5)

DirectoryIterator::isWritable — Определяет, доступен ли текущий элемент DirectoryIterator для записи

Описание

public bool DirectoryIterator::isWritable ( void )

Определяет, доступен ли текущий элемент DirectoryIterator для записи.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE, если файл или директория доступны для записи, иначе FALSE

Примеры

Пример #1 Пример использования DirectoryIterator::isWritable()

Этот пример показывает список файлов и директорий, которые могут быть открыты для записи, расположенные в директории, содержащей выполняемый скрипт.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    if ($fileinfo->isWritable()) {
        echo $fileinfo->getFilename() . "\n";
    }
}
?>

Результатом выполнения данного примера будет что-то подобное:

apples.txt
bananas.html
pears
