sqlite_prev

SQLiteResult::prev

(PHP 5 < 5.4.0)

sqlite_prev -- SQLiteResult::prev — Переходит на предыдущую запись

Описание

bool sqlite_prev ( resource $result )

Объектно-ориентированный стиль (метод):

bool SQLiteResult::prev ( void )

sqlite_prev() перемещает внутренний указатель в дескрипторе результирующего набора result на предыдущую запись.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Возвращаемые значения

Возвращает TRUE в случае успеха, или FALSE, если предыдущих записей больше нет.

Смотрите также

  • sqlite_has_prev() - Определяет, доступна ли предыдущая запись
  • sqlite_rewind() - Переходит к первой записи
  • sqlite_next() - Переходит на следующую запись
