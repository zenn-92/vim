Установка

Расширение SQLite3 доступно по умолчанию начиная с версии PHP 5.3.0. Расширение можно отключить помощью --without-sqlite3 во время компиляции.

Пользователи Windows должны включить поддержку php_sqlite3.dll для того, чтобы использовать это расширение. Эта DLL входит в состав Windows-дистрибутивов РНР начиная с версии PHP 5.3.0.

Замечание:

Некоторое время данное расширение было так называемым PECL-расширением, версии которого рекомендуется использовать только в целях эксперимента.
