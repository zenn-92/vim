ArrayIterator::count

(PHP 5 >= 5.0.0)

ArrayIterator::count — Посчитать количество элементов

Описание

public int ArrayIterator::count ( void )

Возвращает количество элементов в массиве (array), или количество публичных свойств объекта (object).

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Количество элементов или публичных свойств в ассоциативном массиве (array) или объекте (object) соответственно.
