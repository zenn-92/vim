ReflectionClass::isInterface

(PHP 5)

ReflectionClass::isInterface — Проверяет, является ли класс интерфейсом

Описание

public bool ReflectionClass::isInterface ( void )

Проверяет, является ли класс интерфейсом или нет.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования ReflectionClass::isInterface()

<?php
interface SomeInterface {
    public function interfaceMethod();
}

$class = new ReflectionClass('SomeInterface');
var_dump($class->isInterface());
?>

Результат выполнения данного примера:

bool(true)
