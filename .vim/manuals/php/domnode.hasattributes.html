DOMNode::hasAttributes

(PHP 5)

DOMNode::hasAttributes — Проверяет, содержит ли данный узел атрибуты

Описание

public bool DOMNode::hasAttributes ( void )

Этот метод проверяет, содержит ли данный узел атрибуты. Проверяемый узел должен быть XML_ELEMENT_NODE.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
