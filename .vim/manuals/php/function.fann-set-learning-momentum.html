fann_set_learning_momentum

(PECL fann >= 1.0.0)

fann_set_learning_momentum — Sets the learning momentum

Описание

bool fann_set_learning_momentum ( resource $ann , float $learning_momentum )

Sets the learning momentum.

More info available in fann_get_learning_momentum().

Список параметров

ann

Ресурс (resource) нейронной сети.

learning_momentum

The learning momentum.

Возвращаемые значения

Возвращает TRUE в случае успешного выполнения, или FALSE в противном случае.

Смотрите также

  • fann_get_learning_momentum() - Returns the learning momentum
  • fann_set_training_algorithm() - Sets the training algorithm
