imagepsloadfont

(PHP 4, PHP 5)

imagepsloadfont — Загрузка шрифта PostScript Type 1 из файла

Описание

resource imagepsloadfont ( string $filename )

Загружает шрифт PostScript Type 1 из файла filename.

Список параметров

filename

Путь к файлу шрифта Postscript.

Возвращаемые значения

Если все в порядке, будет возвращен индекс шрифта для дальнейшего использования. В противном случае будет возвращено значение FALSE.

Примеры

Пример #1 Пример использования imagepsloadfont()

<?php
// Создание изображения 
$im = imagecreatetruecolor(350, 45);

// Создание цвета и закрашивание фона
$black = imagecolorallocate($im, 0, 0, 0);
$white = imagecolorallocate($im, 255, 255, 255);
imagefilledrectangle($im, 0, 0, 349, 44, $white);

// Загрузка шрифта, запись текста на изображении, очистка памяти
$font = imagepsloadfont("bchbi.pfb");
imagepstext($im, "Тест... Работает!", $font, 32, $white, $black, 32, 32);
imagepsfreefont($font);

// Вывод изображения
header('Content-type: image/png');

imagepng($im);
imagedestroy($im);
?>

Примечания

Замечание: Эта функция доступна только в случае, если PHP был скомпилирован с опцией --with-t1lib[=DIR] .
