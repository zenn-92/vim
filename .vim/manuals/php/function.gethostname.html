gethostname

(PHP >= 5.3.0)

gethostname — Получает имя хоста

Описание

string gethostname ( void )

Функция gethostname() получает стандартное имя хоста для локального компьютера.

Возвращаемые значения

Возвращает строку с именем хоста либо FALSE в случае ошибки.

Примеры

Пример #1 Пример использования gethostname()

<?php
echo gethostname(); // может вывести, н-р.: sandie

// Или вариант, который также работает в версиях до PHP 5.3
echo php_uname('n'); // может вывести, н-р.: sandie
?>

Смотрите также

  • gethostbyname() - Получает адрес IPv4, соответствующий переданному имени узла интернета
  • gethostbyaddr() - Получает доменное имя узла, соответствующее переданному IP-адресу
  • php_uname() - Выдает информацию об операционной системе, на которой запущен PHP
