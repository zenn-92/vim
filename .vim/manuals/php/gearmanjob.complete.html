GearmanJob::complete

(PECL gearman <= 0.5.0)

GearmanJob::complete — Отправка результата и статуса завершения (устаревший метод)

Описание

public bool GearmanJob::complete ( string $result )

Отправляет результаты обработки клиенту и обновляет статус работы на завершенный.

Замечание:

Этот метод был заменен на GearmanJob::sendComplete() в выпуске расширения Gearman 0.6.0.

Список параметров

result

Сериализованные результаты обработки.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Смотрите также

  • GearmanJob::sendFail() - Отправка статуса провала операции
  • GearmanJob::setReturn() - Установка возвращаемого значения
