RecursiveIteratorIterator::callGetChildren

(PHP 5 >= 5.1.0)

RecursiveIteratorIterator::callGetChildren — Получение дочерних элементов

Описание

public RecursiveIterator RecursiveIteratorIterator::callGetChildren ( void )

Получает дочерние элементы текущего элемента итератора.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Объект класса RecursiveIterator().
