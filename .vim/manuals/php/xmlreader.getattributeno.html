XMLReader::getAttributeNo

(PHP 5 >= 5.1.0)

XMLReader::getAttributeNo — Получить значение атрибута по индексу

Описание

string XMLReader::getAttributeNo ( int $index )

Возвращает значение атрибута на основании позиции или пустую строку, если атрибут не существует или не расположен на узле элемента.

Список параметров

index

Позиция атрибута.

Возвращаемые значения

Значение атрибута или пустая строка, если атрибут с заданным параметром index не существует или нет позиции элемента.
