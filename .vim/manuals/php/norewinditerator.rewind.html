NoRewindIterator::rewind

(PHP 5 >= 5.1.0)

NoRewindIterator::rewind — Предотвращает возврат внутреннего итератора в начало

Описание

public void NoRewindIterator::rewind ( void )

Предотвращает возврат внутреннего итератора в начало.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования NoRewindIterator::rewind()

Этот пример показывает, что вызов операции возврата в начало над объектом NoRewindIterator не имеет никакого эффекта.

<?php
$fruits = array("лимон", "апельсин", "яблоко", "груша");

$noRewindIterator = new NoRewindIterator(new ArrayIterator($fruits));

echo $noRewindIterator->current() . "\n";
$noRewindIterator->next();
// возврат итератора в начало (ничего не должно случиться)
$noRewindIterator->rewind();
echo $noRewindIterator->current() . "\n";
?>

Результат выполнения данного примера:

лимон
апельсин
