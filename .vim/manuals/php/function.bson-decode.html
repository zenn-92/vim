bson_decode

(PECL mongo >=1.0.1)

bson_decode — Преобразовует объект BSON в массив PHP

Описание

array bson_decode ( string $bson )

Функция нестабильная и совершенно бесполезна для 99% пользователей. Может быть полезна только в случае если вы пишете что-нибудь странное, как, например, написание собственного драйвера поверх стандартного драйвера PHP.

Список параметров

bson

Преобразуемый BSON объект.

Возвращаемые значения

Возвращает десериализованный BSON объект.
