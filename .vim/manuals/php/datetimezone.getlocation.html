DateTimeZone::getLocation

timezone_location_get

(PHP 5 >= 5.3.0)

DateTimeZone::getLocation -- timezone_location_get — Возвращает информацию о местоположении для временной зоны

Описание

Объектно-ориентированный стиль

public array DateTimeZone::getLocation ( void )

Процедурный стиль

array timezone_location_get ( DateTimeZone $object )

Возвращает информацию о местоположении временной зоны, включающую код страны, широту/долготу и комментарии.

Список параметров

object

Только для процедурного стиля: Объект DateTimeZone, возвращаемый timezone_open()

Возвращаемые значения

Массив, содержащий информацию о местоположении временной зоны.

Примеры

Пример #1 Пример использования DateTimeZone::getLocation()

<?php
$tz = new DateTimeZone("Europe/Prague");
print_r($tz->getLocation());
print_r(timezone_location_get($tz));
?>
