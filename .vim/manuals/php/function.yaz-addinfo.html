yaz_addinfo

(PHP 4 >= 4.0.1, PECL yaz >= 0.9.0)

yaz_addinfo — Возвращает дополнительную информацию в случае ошибки

Описание

string yaz_addinfo ( resource $id )

Возвращает дополнительную информацию в случае ошибки последнего запроса к серверу.

С некоторыми серверами эта функция может вернуть ту же строку, что и функция yaz_error().

Список параметров

id

Дескриптор соединения, возвращаемый функцией yaz_connect().

Возвращаемые значения

Строка содержит дополнительную информацию об ошибке или пустое значение, если последняя операция прошла успешно или нет дополнительной информации от сервера.

Смотрите также

  • yaz_error() - Returns error description
  • yaz_errno() - Returns error number
