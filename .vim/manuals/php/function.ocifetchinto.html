ocifetchinto

(PHP 4, PHP 5, PECL OCI8 >= 1.0.0)

ocifetchinto — Возвращает следующий ряд в массив (устарела)

Описание

int ocifetchinto ( resource $statement , array &$result [, int $mode = OCI_ASSOC + OCI_NUM ] )

Данная функция устарела. Рекомендуемые альтернативы: oci_fetch_array(), oci_fetch_object(), oci_fetch_assoc() и oci_fetch_row().

Внимание

Эта функция УСТАРЕЛА начиная с версии PHP 5.4.0. Использование этой функции не рекомендуется.
