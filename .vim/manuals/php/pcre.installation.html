Установка

Начиная с PHP 4.2.0, эти функции включены по умолчанию. Вы можете отключить их при помощи опции --without-pcre-regex . В случае если вы хотите использовать библиотеку, отличную от идущей в стандартной поставке РНР, используйте опцию --with-pcre-regex=DIR для указания директории, содержащей необходимые файлы. Если у вас версия PHP менее, чем 4.2.0, вам необходимо сконфигурировать и пересобрать PHP с опцией --with-pcre-regex[=DIR] , чтобы включить поддержку PCRE-функций.

Версия PHP для Windows имеет встроенную поддержку данного расширения. Это означает, что для использования данных функций не требуется загрузка никаких дополнительных расширений.

Замечание:

Начиная с версии PHP 5.3.0 это расширение не может быть отключено и поэтому всегда присутствует.

Также еще возможно собрать данное расширение с внешней библиотекой PCRE с помощью опции --with-pcre-regex=DIR

PCRE - активный проект и по мере того как он меняется, меняется и функционал PHP, который полагается на него. Возможно, некоторые части документации PHP устарели в том плане, что они не покрывает новейшие возможности, которые предоставляет PCRE.
