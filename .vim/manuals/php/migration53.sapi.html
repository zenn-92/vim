Изменения в модулях SAPI

  • Доступен новый модуль SAPI, называемый litespeed.
  • Поддержка FastCGI в CGI SAPI теперь всегда включена и не может быть отключена. Для более детальной информации см. sapi/cgi/CHANGES.
  • Новая CGI SAPI опция -T может быть использована для измерения времени повторяющегося исполнения скрипта.
  • В CGI/FastCGI теперь есть поддержка для .htaccess-стиля файлов php.ini, определяемых пользователем.
  • Функция dl() теперь отключена по умолчанию, и доступна только для CLI, CGI и встраиваемых SAPI.
