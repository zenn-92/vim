ldap_get_dn

(PHP 4, PHP 5)

ldap_get_dn — Получить DN результирующей записи

Описание

string ldap_get_dn ( resource $link_identifier , resource $result_entry_identifier )

Обнаружить DN записи в результате.

Список параметров

link_identifier

Идентификатор ссылки LDAP, возвращенный ldap_connect().

result_entry_identifier

Возвращаемые значения

Возвращает DN записи результата и FALSE в случае ошибки.
