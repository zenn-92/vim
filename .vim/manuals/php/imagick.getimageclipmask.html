Imagick::getImageClipMask

(Информация о версии неизвестна, возможно, только в SVN)

Imagick::getImageClipMask — Gets image clip mask

Описание

Imagick Imagick::getImageClipMask ( void )

Returns the image clip mask. The clip mask is an Imagick object containing the clip mask. Этот метод доступен, если Imagick был скомпилирован с версией ImageMagick 6.3.6 или старше.

Возвращаемые значения

Returns an Imagick object containing the clip mask.

Ошибки

Вызывает ImagickException при ошибке.
