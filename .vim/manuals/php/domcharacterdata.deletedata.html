DOMCharacterData::deleteData

(PHP 5)

DOMCharacterData::deleteData — Удаление диапазона символов из узла

Описание

void DOMCharacterData::deleteData ( int $offset , int $count )

Удаляет count символов, начиная с позиции offset.

Список параметров

offset

Позиция, с которой будут удаляться символы.

count

Количество удаляемых символов. Если offset и count в сумме окажется больше значения длины строки данных узла, будут удалены все символы до конца данных.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Ошибки

DOM_INDEX_SIZE_ERR

Возникает, если offset меньше нуля или больше количества 16-битных блоков в области данных, либо если count имеет отрицательное значение.

Смотрите также

  • DOMCharacterData::appendData() - Добавляет строку в конец символьных данных узла
  • DOMCharacterData::insertData() - Вставляет строку после заданного отступа из 16-битных блоков
  • DOMCharacterData::replaceData() - Заменяет подстроку в узле типа DOMCharacterData
  • DOMCharacterData::substringData() - Извлекает определенный диапазон данных из узла
