SphinxClient::setMaxQueryTime

(PECL sphinx >= 0.1.0)

SphinxClient::setMaxQueryTime — Установка максимального времени выполнения запроса

Описание

public bool SphinxClient::setMaxQueryTime ( int $qtime )

Устанавливает максимальное время выполнения запроса.

Список параметров

qtime

Максимальное время выполнения запроса, в миллисекундах. Должно быть положительным целым числом. Значение по умолчанию 0, то есть без ограничений.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
