DOMElement::setAttributeNodeNS

(PHP 5)

DOMElement::setAttributeNodeNS — Добавляет новый атрибут к элементу

Описание

public DOMAttr DOMElement::setAttributeNodeNS ( DOMAttr $attr )

Добавляет новый узел атрибута attr в элемент.

Список параметров

attr

Узел атрибута.

Возвращаемые значения

Возвращает старый узел, если атрибут был заменен.

Ошибки

DOM_NO_MODIFICATION_ALLOWED_ERR

Возникает, если узел доступен только для чтения.

Смотрите также

  • DOMElement::hasAttributeNS() - Проверяет, существует ли заданный атрибут
  • DOMElement::getAttributeNodeNS() - Возвращает узел атрибута
  • DOMElement::removeAttributeNode() - Удаляет атрибут
