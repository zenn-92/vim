PDF_get_font

(PHP 4, PECL pdflib >= 1.0.0)

PDF_get_font — Возвращает шрифт [устаревшее]

Описание

Эта функция устарела начиная с 3-ей версии PDFlib, вместо нее используйте функцию PDF_get_value() с параметром font.
