stream_context_set_params

(PHP 4 >= 4.3.0, PHP 5)

stream_context_set_params — Устанавливает параметры для потока/обертки/контекста

Описание

bool stream_context_set_params ( resource $stream_or_context , array $params )

Устанавливает параметры для указанного контекста.

Список параметров

stream_or_context

Поток или контекст, к которому будут применены параметры.

params

Массив параметров, которые будут установлены.

Замечание:

Массив params должен быть ассоциативным массивом со следующей структурой: $params['paramname'] = "paramvalue";.

Поддерживаемые параметры

Параметры Назначение
notification Название определённой пользователем callback-функции, которая будет вызвана, когда потоком посылается уведомление.
options Массив опций и параметров контекста.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
