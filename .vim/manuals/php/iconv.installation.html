Установка

Это расширение по умолчанию включено, чтобы его отключить используйте опцию --without-iconv .

Также может быть задана необязательная директива --with-iconv-dir для указания альтернативного расположения iconv в системе, в которой компилируется PHP, иначе будет использовано расположение по умолчанию.

Замечание: Примечание для пользователей PHP 4
Включение этого расширения требует сборки с флагом --with-iconv=[DIR] . Windows-пользователи могут подключить его, разместив iconv.dll в системной папке (PATH), а также, включив php_iconv.dll в php.ini.
