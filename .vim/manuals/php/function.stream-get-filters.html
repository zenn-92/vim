stream_get_filters

(PHP 5)

stream_get_filters — Извлекает список зарегистрированных фильтров

Описание

array stream_get_filters ( void )

Извлекает список зарегистрированных фильтров на запущенной системе.

Возвращаемые значения

Возвращает индексированный массив, содержащий названия всех доступных системных фильтров потока.

Примеры

Пример #1 Пример использования функции stream_get_filters()

<?php
$streamlist = stream_get_filters();
print_r($streamlist);
?>

Результатом выполнения данного примера будет что-то подобное:

Array (
  [0] => string.rot13
  [1] => string.toupper
  [2] => string.tolower
  [3] => string.base64
  [4] => string.quoted-printable
)

  • stream_get_wrappers() - Извлекает список зарегистрированных потоков
