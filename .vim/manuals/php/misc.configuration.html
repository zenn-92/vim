Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Опции настроек разных функций
Имя По умолчанию Меняемо Список изменений
ignore_user_abort "0" PHP_INI_ALL  
highlight.string "#DD0000" PHP_INI_ALL  
highlight.comment "#FF8000" PHP_INI_ALL  
highlight.keyword "#007700" PHP_INI_ALL  
highlight.bg "#FFFFFF" PHP_INI_ALL Удалено в PHP 5.4.0.
highlight.default "#0000BB" PHP_INI_ALL  
highlight.html "#000000" PHP_INI_ALL  
browscap NULL PHP_INI_SYSTEM  
Для подробного описания констант PHP_INI_*, обратитесь к разделу Где могут быть установлены параметры конфигурации.

Краткое разъяснение конфигурационных директив.

ignore_user_abort boolean

FALSE по умолчанию. Если изменяется на TRUE, то скрипты не будут остановлены после того как клиент разорвет соединение.

См. также ignore_user_abort().

highlight.bg string
highlight.comment string
highlight.default string
highlight.html string
highlight.keyword string
highlight.string string

Цвета для режима подсвечивания (Syntax Highlighting). Всё, что приемлемо в <font color="??????">, будет работать.

browscap string

Имя (н-р: browscap.ini) и расположение файла возможностей браузера. См. также get_browser().
