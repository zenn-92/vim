printer_delete_dc

(PECL printer SVN)

printer_delete_dc — Удаляет контекст устройства

Описание

bool printer_delete_dc ( resource $printer_handle )

Эта функция удаляет контекст устройства. За примером обращайтесь к описанию функции printer_create_dc().

Список параметров

printer_handle

printer_handle должен быть активным дескриптором устройства.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
