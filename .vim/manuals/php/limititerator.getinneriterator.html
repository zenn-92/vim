LimitIterator::getInnerIterator

(PHP 5 >= 5.1.0)

LimitIterator::getInnerIterator — Получение внутреннего объекта-итератора

Описание

public Iterator LimitIterator::getInnerIterator ( void )

Возвращает объект-итератор Iterator скрытый внутри объекта.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Внутренний объект-итератор, переданный конструктору LimitIterator::__construct().
