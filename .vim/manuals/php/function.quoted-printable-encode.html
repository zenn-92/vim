quoted_printable_encode

(PHP 5 >= 5.3.0)

quoted_printable_encode — Кодирует 8-битную строку в с помощью метода quoted-printable

Описание

string quoted_printable_encode ( string $str )

Возвращает строку, закодированную в формат quoted-printable в соответствии с разделом 6.7 » RFC2045.

Эта функция подобна функции imap_8bit(), за исключением того что не требует для своей работы модуля IMAP.

Список параметров

str

Входная строка.

Возвращаемые значения

Возвращает закодированную строку.

Смотрите также

  • quoted_printable_decode() - Преобразует строку, закодированную методом quoted-printable в 8-битовую строку
  • iconv_mime_encode() - Сжимает поле MIME-заголовка
