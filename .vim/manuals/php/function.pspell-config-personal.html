pspell_config_personal

(PHP 4 >= 4.0.2, PHP 5)

pspell_config_personal — Устанавливает файл, содержащий персональный список слов

Описание

bool pspell_config_personal ( int $dictionary_link , string $file )

Устанавливает файл, содержащий персональный список слов. Персональный список слов будет загружен и использован в дополнение к стандартному после тго, как вы вызовите pspell_new_config(). Это тот же самый файл, в который функция pspell_save_wordlist() сохранит персональный список слов.

pspell_config_personal() должна быть использвана для конфигурации до вызова pspell_new_config().

Список параметров

dictionary_link

file

Персональный список слов. Если файл не существует, он будет создан. Файл должен быть доступен для записи любому, кто запускает PHP (например, nobody).

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 pspell_config_personal()

<?php
$pspell_config = pspell_config_create("en");
pspell_config_personal($pspell_config, "/var/dictionaries/custom.pws");
$pspell_link = pspell_new_config($pspell_config);
pspell_check($pspell_link, "thecat");
?>

Примечания

Замечание:

Эта функция не будет работать, если у вас нет pspell .11.2 и aspell .32.5 или выше.
