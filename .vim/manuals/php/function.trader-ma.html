trader_ma

(PECL trader >= 0.2.0)

trader_ma — Moving average

Описание

array trader_ma ( array $real [, integer $timePeriod [, integer $mAType ]] )

Список параметров

real

Массив, содержащий реальные значения.

timePeriod

Номер периода. Допустимы значения от 2 до 100000.

mAType

Тип скользящей средней. Следует использовать серию констант TRADER_MA_TYPE_*.

Возвращаемые значения

Returns an array with calculated data or false on failure.
