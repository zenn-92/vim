bind_textdomain_codeset

(PHP 4 >= 4.2.0, PHP 5)

bind_textdomain_codeset — Устанавливает кодировку, в которой будут возвращаться сообщения из каталога сообщений домена

Описание

string bind_textdomain_codeset ( string $domain , string $codeset )

Функция bind_textdomain_codeset() устанавливает кодировку, в которой будут возвращаться сообщения из domain, такими функциями как gettext().

Список параметров

domain

Домен.

codeset

Кодировка.

Возвращаемые значения

Строка в случае успеха.
