stream_set_read_buffer

(PHP 5 >= 5.3.3)

stream_set_read_buffer — Устанавливает буферизацию чтения файла на указанном потоке

Описание

int stream_set_read_buffer ( resource $stream , int $buffer )

Устанавливает буфер чтения. Это эквивалент функции stream_set_write_buffer(), но для операций чтения.

Список параметров

stream

Файловый указатель.

buffer

Число байт для буферизации. Если аргумент buffer равен 0, то операции чтения не буферизуются. Это гарантирует, что все операции чтения при помощи функции fread() были завершены до того, как остальным процессам будет позволено читать из входного потока.

Возвращаемые значения

Возвращает 0 в случае удачи, или EOF, если запрос не может быть выполнен.
