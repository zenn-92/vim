ReflectionParameter::isDefaultValueAvailable

(PHP 5 >= 5.0.3)

ReflectionParameter::isDefaultValueAvailable — Проверяет доступно ли значение по умолчанию аргумента

Описание

public bool ReflectionParameter::isDefaultValueAvailable ( void )

Проверяет доступно ли значение по умолчанию аргумента.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если доступно значение по умолчанию, FALSE в противном случае.
