pspell_new_config

(PHP 4 >= 4.0.2, PHP 5)

pspell_new_config — Загружает новый словарь с установками на основе заданной конфигурации

Описание

int pspell_new_config ( int $config )

pspell_new_config() открывает новый словарь с параметрами, заданными в конфигурации, созданной с помощью pspell_config_create() и модифицированной с помощью функций pspell_config_*(). Этот метод дает наибольшую гибкость и имеет все функциональные возможности, предоставляемые pspell_new() и pspell_new_personal().

Список параметров

config

Параметр config - это параметр возвращённый функцией pspell_config_create() после того, как конфигурация была создана.

Возвращаемые значения

В случае успеха возвращает ссылку на словарь.

Примеры

Пример #1 pspell_new_config()

<?php
$pspell_config = pspell_config_create("en");
pspell_config_personal($pspell_config, "/var/dictionaries/custom.pws");
pspell_config_repl($pspell_config, "/var/dictionaries/custom.repl");
$pspell_link = pspell_new_config($pspell_config);
?>
