pg_num_fields

(PHP 4 >= 4.2.0, PHP 5)

pg_num_fields — Возвращает количество полей в выборке

Описание

int pg_num_fields ( resource $result )

pg_num_fields() возвращает количество полей (столбцов) в результате запроса PostgreSQL.

Замечание:

Ранее функция называлась pg_numfields().

Список параметров

result

Ресурс результата запроса PostgreSQL, возвращаемый функциями pg_query(), pg_query_params() или pg_execute() (и прочих).

Возвращаемые значения

Количество полей (столбцов) в выборке. В случае ошибки возвращает -1.

Примеры

Пример #1 Пример использования pg_num_fields()

<?php
$result = pg_query($conn, "SELECT 1, 2");

$num = pg_num_fields($result);

echo "Возвращено полей: " . $num . ".\n";
?>

Результат выполнения данного примера:

Возвращено полей: 2.
