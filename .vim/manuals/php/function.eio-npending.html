eio_npending

(PECL eio >= 0.0.1dev)

eio_npending — Возвращает число завершенных, но необработанных процессов

Описание

int eio_npending ( void )

eio_npending() возвращает число завершенных, но необработанных процессов

Список параметров

У этой функции нет параметров.

Возвращаемые значения

eio_npending() возвращает число завершенных, но необработанных процессов.
