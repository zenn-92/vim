stream_get_wrappers

(PHP 5)

stream_get_wrappers — Извлекает список зарегистрированных потоков

Описание

array stream_get_wrappers ( void )

Извлекает список зарегистрированных потоков, доступных на запущенной системе.

Возвращаемые значения

Возвращает индексированный массив, содержащий названия все обёрток потоков, доступных на запущенной системе.

Примеры

Пример #1 Пример использования stream_get_wrappers()

<?php
print_r(stream_get_wrappers());
?>

Результатом выполнения данного примера будет что-то подобное:

Array
(
    [0] => php
    [1] => file
    [2] => http
    [3] => ftp
    [4] => compress.bzip2
    [5] => compress.zlib
)

Пример #2 Проверка существования обёртки потока

<?php
// Проверяет существование обёртки потока bzip2
if (in_array('compress.bzip2', stream_get_wrappers())) {
    echo 'compress.bzip2:// поддержка включена.';
} else {
    echo 'compress.bzip2:// поддержка не включена.';
}
?>
