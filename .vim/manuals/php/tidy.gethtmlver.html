tidy::getHtmlVer

tidy_get_html_ver

(PHP 5, PECL tidy >= 0.5.2)

tidy::getHtmlVer -- tidy_get_html_ver — Получает обнаруженную HTML версию для указанного документа

Описание

Объектно-ориентированный стиль

int tidy::getHtmlVer ( void )

Процедурный стиль

int tidy_get_html_ver ( tidy $object )

Возвращает обнаруженную HTML версию для указанного объекта tidy object.

Список параметров

object

Объект Tidy.

Возвращаемые значения

Возвращает обнаруженную HTML версию.

Внимание

Эта функция еще не реализована в самом Tidylib, поэтому всегда возвращается 0.
