SplPriorityQueue::valid

(PHP 5 >= 5.3.0)

SplPriorityQueue::valid — Проверяет, есть ли в очереди еще элементы

Описание

bool SplPriorityQueue::valid ( void )

Проверяет, есть ли в очереди еще элементы.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE, если в очереди еще есть доступные элементы, и FALSE в противном случае.
