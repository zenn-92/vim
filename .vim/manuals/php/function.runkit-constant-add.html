runkit_constant_add

(PECL runkit >= 0.7.0)

runkit_constant_add — Объявляет константу. Схожа с функцией define(), но позволяет создавать локальные константы внутри классов.

Описание

bool runkit_constant_add ( string $constname , mixed $value )

Список параметров

constname

Имя объявляемой константы. Строка для объявления глобальной константы или выражение classname::constname для добавления локальной константы в классе.

value

NULL, Bool, Long, Double, String, или тип Resource для сохранения в новой константе.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Смотрите также

  • define() - Определяет именованную константу
  • runkit_constant_redefine() - Переназначить уже определенную константу.
  • runkit_constant_remove() - Удаляет уже определенную константу.
