GearmanJob::sendComplete

(PECL gearman >= 0.6.0)

GearmanJob::sendComplete — Отправка результата и статуса завершения

Описание

public bool GearmanJob::sendComplete ( string $result )

Отправляет результаты работы клиенту и обновляет статус объекта на завершенный.

Список параметров

result

Сериализованные результаты работы.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Смотрите также

  • GearmanJob::sendFail() - Отправка статуса провала операции
  • GearmanJob::setReturn() - Установка возвращаемого значения
