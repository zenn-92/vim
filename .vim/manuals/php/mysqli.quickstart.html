Данное практическое руководство поможет освоить основные моменты API-интерфейса MySQL сервера.

В этом руководстве приводится краткий обзор расширения mysqli. Для всех основных особенностей API-интерфейса приведены примеры PHP кода. Также объясняются принципы работы баз данных в целом, чтобы иметь возможность описать какие-то отдельные моменты, присущие конкретно MySQL.

Требования: Знакомство с языком программирования PHP, SQL, а также знание основ работы с MySQL сервером.
