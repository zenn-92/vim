XMLReader::getAttributeNs

(PHP 5 >= 5.1.0)

XMLReader::getAttributeNs — Получить значение атрибута по localname и URI

Описание

string XMLReader::getAttributeNs ( string $localName , string $namespaceURI )

Возвращает значение атрибута по имени и URI пространства имен или пустую строку, если атрибут не существует или не расположен на узле элемента.

Список параметров

localName

Локальное имя.

namespaceURI

URI пространства имён.

Возвращаемые значения

Значение атрибута или пустая строка, если атрибут с заданным localName и namespaceURI не найден или нет позиции элемента.
