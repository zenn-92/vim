SphinxClient::resetFilters

(PECL sphinx >= 0.1.0)

SphinxClient::resetFilters — Обнуление всех фильтров

Описание

public void SphinxClient::resetFilters ( void )

Обнуляет все выставленные на данный момент фильтры. Вызов этого метота обычно требуется при использовании очереди запросов. При необходимости установки различных фильтров для различных запросов в очереди следует вызвать SphinxClient::resetFilters() и добавить новые фильтмы, используя соответствующие вызовы.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
