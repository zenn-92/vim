DOMElement::__construct

(PHP 5)

DOMElement::__construct — Создание нового объекта класса DOMElement

Описание

public DOMElement::__construct ( string $name [, string $value [, string $namespaceURI ]] )

Создает новый объект класса DOMElement. Этот объект доступен только для чтения. Он может быть добавлен в документ, но в него нельзя добавить дочерние узлы пока данный узел связан с документом. Для того чтобы создавать узлы, которые могут быть изменены, воспользуйтесь следующими методами: DOMDocument::createElement or DOMDocument::createElementNS.

Список параметров

name

Имя тэга создаваемого элемента. Если передан параметр namespaceURI, имя элемента может получить префикс для связи с URI.

value

Значение элемента.

namespaceURI

Пространство имен URI, передается для создания элемента со отдельным пространством имен.

Примеры

Пример #1 Создание DOMElement

<?php

$dom = new DOMDocument('1.0', 'iso-8859-1');
$element = $dom->appendChild(new DOMElement('root'));
$element_ns = new DOMElement('pr:node1', 'thisvalue', 'http://xyz');
$element->appendChild($element_ns);
echo $dom->saveXML(); /* <?xml version="1.0" encoding="utf-8"?>
<root><pr:node1 xmlns:pr="http://xyz">thisvalue</pr:node1></root> */

?>
