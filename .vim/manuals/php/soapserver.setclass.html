SoapServer::setClass

(PHP 5 >= 5.0.1)

SoapServer::setClass — Устанавливает класс, который будет обрабатывать SOAP-запросы

Описание

public void SoapServer::setClass ( string $class_name [, mixed $args [, mixed $... ]] )

Экспортирует все методы из заданного класса

Чтобы сохранить объект для каждого последующего запроса в рамках данной PHP-сессии, можно использовать метод SoapServer::setPersistence()

Список параметров

class_name

Имя передаваемого класса.

args

Необязательные параметры, которые будут переданы в конструктор класса в процессе создания объекта.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Смотрите также

  • SoapServer::SoapServer() - SoapServer constructor
  • SoapServer::addFunction() - Adds one or more functions to handle SOAP requests
  • SoapServer::setPersistence() - Sets SoapServer persistence mode
