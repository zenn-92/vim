getimagesizefromstring

(PHP 5 >= 5.4.0)

getimagesizefromstring — Получение размера изображения из строки данных

Описание

array getimagesizefromstring ( string $imagedata [, array &$imageinfo ] )

Функция работает идентично функции getimagesize() с той разницей, что getimagesizefromstring() принимает данные изображения в виде строки в качестве первого аргумента вместо имени файла.

Для уточнения, как функция работает, обращайтесь к документации к функции getimagesize().

Список параметров

imagedata

Данные изображения в виде строки.

imageinfo

См. getimagesize().

Возвращаемые значения

См. getimagesize().

Примеры

Пример #1 Пример использования getimagesizefromstring()

<?php
$img = '/path/to/test.png';

// Открыть как файл
$size_info1 = getimagesize($img);

// Открыть как строку
$data       = file_get_contents($img);
$size_info2 = getimagesizefromstring($data);
?>
