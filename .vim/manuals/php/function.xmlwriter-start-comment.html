XMLWriter::startComment

xmlwriter_start_comment

(PHP 5 >= 5.1.2, PECL xmlwriter >= 1.0.0)

XMLWriter::startComment -- xmlwriter_start_comment — Создает стартовый комментарий

Описание

Объектно-ориентированный стиль

bool XMLWriter::startComment ( void )

Процедурный стиль

bool xmlwriter_start_comment ( resource $xmlwriter )

Начинает комментарий.

Список параметров

xmlwriter

Только для процедурных вызовов. Изменяемый resource XMLWriter. Этот ресурс может быть получен из xmlwriter_open_uri() или xmlwriter_open_memory().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
