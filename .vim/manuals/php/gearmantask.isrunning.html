GearmanTask::isRunning

(PECL gearman >= 0.5.0)

GearmanTask::isRunning — Проверяет, выполняется ли задание в данный момент

Описание

public bool GearmanTask::isRunning ( void )

Указывает, выполняется сейчас задание или нет.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если задание выполняется, FALSE в противном случае.
