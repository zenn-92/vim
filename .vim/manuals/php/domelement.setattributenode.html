DOMElement::setAttributeNode

(PHP 5)

DOMElement::setAttributeNode — Добавляет новый узел атрибута к элементу

Описание

public DOMAttr DOMElement::setAttributeNode ( DOMAttr $attr )

Добавляет новый узел атрибута attr в элемент.

Список параметров

attr

Узел атрибута.

Возвращаемые значения

Возвращает старый узел, если атрибут был заменен, или NULL.

Ошибки

DOM_NO_MODIFICATION_ALLOWED_ERR

Возникает, если узел доступен только для чтения.

Смотрите также

  • DOMElement::hasAttribute() - Проверяет наличие атрибута
  • DOMElement::getAttributeNode() - Возвращает узел атрибута
  • DOMElement::removeAttributeNode() - Удаляет атрибут
