SplFileInfo::getGroup

(PHP 5 >= 5.1.2)

SplFileInfo::getGroup — Получает группу файла

Описание

public int SplFileInfo::getGroup ( void )

Получает группу файла. Числовой идентификатор группы возвращается в числовом формате.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

ID группы в числовом формате.

Ошибки

Выбрасывает RuntimeException в случае ошибки.

Примеры

Пример #1 Пример использования SplFileInfo::getGroup()

<?php
$info = new SplFileInfo(__FILE__);
print_r(posix_getgrgid($info->getGroup()));
?>
