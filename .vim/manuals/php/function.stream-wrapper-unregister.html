stream_wrapper_unregister

(PHP 5 >= 5.1.0)

stream_wrapper_unregister — Отменяет регистрацию обертки URL

Описание

bool stream_wrapper_unregister ( string $protocol )

Позволяет вам отключить уже определённую обёртку потока. Как только обёртка будет отключена, вы можете перезаписать её пользовательской обёрткой, используя stream_wrapper_register() или включить её повторно, используя stream_wrapper_restore().

Список параметров

protocol

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
