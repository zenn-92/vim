DOMDocument::getElementsByTagName

(PHP 5)

DOMDocument::getElementsByTagName — Ищет все элементы с заданным локальным именем

Описание

public DOMNodeList DOMDocument::getElementsByTagName ( string $name )

Эта функция возвращает новый объект класса DOMNodeList, содержащий элементы с заданным локальным именем.

Список параметров

name

Локальное имя тэга (без указания пространства имен), по которому будет проводиться поиск. Специальное значение * означает, что поиск будет произведен по всем тэгам.

Возвращаемые значения

Новый объект класса DOMNodeList содержащий все найденные элементы.

Примеры

Пример #1 Простой пример использования

<?php
$xml = <<< XML
<?xml version="1.0" encoding="utf-8"?>
<books>
 <book>Patterns of Enterprise Application Architecture</book>
 <book>Design Patterns: Elements of Reusable Software Design</book>
 <book>Clean Code</book>
</books>
XML;

$dom = new DOMDocument;
$dom->loadXML($xml);
$books = $dom->getElementsByTagName('book');
foreach ($books as $book) {
    echo $book->nodeValue, PHP_EOL;
}
?>

Результат выполнения данного примера:

Patterns of Enterprise Application Architecture
Design Patterns: Elements of Reusable Software Design
Clean Code
