streamWrapper::dir_closedir

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::dir_closedir — Обработчик закрытия директории

Описание

public bool streamWrapper::dir_closedir ( void )

Этот метод вызывается в результате вызова closedir().

Все ресурсы, выделенные или заблокированные во время открытия и использования потока директории, необходимо освободить.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
