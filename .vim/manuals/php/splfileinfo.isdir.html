SplFileInfo::isDir

(PHP 5 >= 5.1.2)

SplFileInfo::isDir — Сообщает, является ли файл каталогом

Описание

public bool SplFileInfo::isDir ( void )

Этот метод может быть использован для определения того, является ли файл каталогом.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE, если каталог; в противном случае - FALSE.

Примеры

Пример #1 Пример использования SplFileInfo::isDir()

<?php
$d = new SplFileInfo(dirname(__FILE__));
var_dump($d->isDir());

$d = new SplFileInfo(__FILE__);
var_dump($d->isDir());
?>

Результатом выполнения данного примера будет что-то подобное:

bool(true)
bool(false)
