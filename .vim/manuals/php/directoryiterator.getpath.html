DirectoryIterator::getPath

(PHP 5)

DirectoryIterator::getPath — Возвращает путь к текущему элементу DirectoryIterator без имени файла

Описание

public string DirectoryIterator::getPath ( void )

Возвращает путь к текущему элементу DirectoryIterator.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает путь до файла (без имени файла и завершающего слэша).

Примеры

Пример #1 Пример использования DirectoryIterator::getPath()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
echo $iterator->getPath();
?>

Результатом выполнения данного примера будет что-то подобное:

/home/examples/public_html
