mysqli_stmt::more_results

mysqli_stmt_more_results

(PHP 5)

mysqli_stmt::more_results -- mysqli_stmt_more_results — Проверяет, есть ли еще наборы строк в результате мультизапроса

Описание

Объектно-ориентированный стиль (method):

public bool mysqli_stmt::more_results ( void )

Процедурный стиль:

bool mysqli_stmt_more_results ( mysql_stmt $stmt )

Проверяет, есть ли еще наборы строк в результате мультизапроса.

Список параметров

stmt

Только для процедурного стиля: Идентификатор выражения, полученный с помощью mysqli_stmt_init().

Возвращаемые значения

Возвращает TRUE, если еще есть доступные результирующие наборы, FALSE в противном случае.
