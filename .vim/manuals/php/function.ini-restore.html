ini_restore

(PHP 4, PHP 5)

ini_restore — Восстанавливает значение настройки конфигурации

Описание

void ini_restore ( string $varname )

Восстанавливает исходное значение настройки конфигурации.

Список параметров

varname

Имя настройки.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования ini_restore()

<?php
$setting = 'y2k_compliance';

echo 'Текущее значение настройки \'' . $setting . '\': ' . ini_get($setting), PHP_EOL;

ini_set($setting, ini_get($setting) ? 0 : 1);
echo 'Новое значение настройки \'' . $setting . '\': ' . ini_get($setting), PHP_EOL;

ini_restore($setting);
echo 'Исходное значение настройки \'' . $setting . '\': ' . ini_get($setting), PHP_EOL;
?>

Результат выполнения данного примера:

Текущее значение настройки 'y2k_compliance': 1
Новое значение настройки 'y2k_compliance': 0
Исходное значение настройки 'y2k_compliance': 1
