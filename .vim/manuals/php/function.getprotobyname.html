getprotobyname

(PHP 4, PHP 5)

getprotobyname — Получает номер протокола по имени

Описание

int getprotobyname ( string $name )

Функция getprotobyname() возвращает номер протокола по имени, указанном в параметре name как в /etc/protocols.

Список параметров

name

Имя протокола.

Возвращаемые значения

Возвращает номер протокола или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования getprotobyname()

<?php
$protocol = 'tcp';
$get_prot = getprotobyname($protocol);
if ($get_prot === FALSE) {
    echo 'Протокол не найден';
} else {
    echo 'Протокол #' . $get_prot;
}
?>
