Imagick::animateImages

(Информация о версии неизвестна, возможно, только в SVN)

Imagick::animateImages — Анимация одного или нескольких изображений

Описание

bool Imagick::animateImages ( string $x_server )

Этот метод производит анимацию изображения на локальном или удалённом X сервере. Этот метод не доступен в Windows. Этот метод доступен, если Imagick был скомпилирован с версией ImageMagick 6.3.6 или старше.

Список параметров

x_server

Адрес X сервера

Возвращаемые значения

В случае успешной работы возвращает TRUE.
