GearmanWorker::unregister

(PECL gearman >= 0.6.0)

GearmanWorker::unregister — Удалить регистрацию имени функции на всех серверах заданий

Описание

public bool GearmanWorker::unregister ( string $function_name )

Снимает регистрацию имени функции на всех серверах заданий. Это значит, что этому обработчику (этой функции) задания посылаться больше не будут.

Список параметров

function_name

Имя функции, регистрацию которой требуется снять.

Возвращаемые значения

Стандартное возвращаемое значение Gearman.

Смотрите также

  • GearmanWorker::register() - Регистрация функции на сервере заданий
  • GearmanWorker::unregisterAll() - Удаление регистрации всех имен функций на серверах заданий
