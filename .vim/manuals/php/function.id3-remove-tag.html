id3_remove_tag

(PECL id3 >= 0.1)

id3_remove_tag — Удаляет существующий ID3-тег

Описание

bool id3_remove_tag ( string $filename [, int $version = ID3_V1_0 ] )

id3_remove_tag() используется для удаления ID3-тега.

Список параметров

filename

Путь к MP3-файлу

Вместо имени файла можно передать потоковый ресурс

version

Позволяет определять версию ID3-тега, так как MP3-файл может содержать теги версии 1.х и 1.2.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 id3_remove_tag() example

<?php
$result = id3_remove_tag( "path/to/example.mp3", ID3_V1_0 );
if ($result === true) {
    echo "Tag succesfully removed\n";
}
?>

Если файл доступен для записи и содержит тег версии 1.0, то вывод будет следующим: 

Tag succesfully removed

Примечания

Замечание: В настоящее время функция id3_remove_tag() поддерживает только теги версии 1.0 и 1.1. Если вы выберете для удаления тег версии 1.0, а файл содержит тег версии 1.1, то тег будет удалён, так как версия 1.1 является расширением версии 1.0.
