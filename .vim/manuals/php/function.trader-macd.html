trader_macd

(PECL trader >= 0.2.0)

trader_macd — Moving Average Convergence/Divergence

Описание

array trader_macd ( array $real [, integer $fastPeriod [, integer $slowPeriod [, integer $signalPeriod ]]] )

Список параметров

real

Массив, содержащий реальные значения.

fastPeriod

Номер периода для быстрого скользящего среднего. Допустимы значения от 2 до 100000.

slowPeriod

Номер периода для медленного скользящего среднего. Допустимы значения от 2 до 100000.

signalPeriod

Сглаживание для сигнальной линии (номер периода). Допустимы значения от 1 до 100000.

Возвращаемые значения

Returns an array with calculated data or false on failure.
