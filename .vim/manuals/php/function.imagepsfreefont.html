imagepsfreefont

(PHP 4, PHP 5)

imagepsfreefont — Освобождение памяти, занятой шрифтом PostScript Type

Описание

bool imagepsfreefont ( resource $font_index )

imagepsfreefont() освобождает память, которая была выделена для PostScript Type 1 шрифта.

Список параметров

font_index

Ресурс шрифта, возвращаемый функцией imagepsloadfont().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования imagepsfreefont()

<?php
// Загрузка файла шрифта .pfb
$font = imagepsloadfont('./bchbi.pfb');

// Тут могут быть какие-либо операции со шрифтом

// Освобождение памяти
imagepsfreefont($font);
?>

Примечания

Замечание: Эта функция доступна только в случае, если PHP был скомпилирован с опцией --with-t1lib[=DIR] .
