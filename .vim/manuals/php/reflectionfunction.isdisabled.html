ReflectionFunction::isDisabled

(PHP 5)

ReflectionFunction::isDisabled — Проверяет, что функция отключена

Описание

public bool ReflectionFunction::isDisabled ( void )

Проверяет, отключена ли функция, с помощью директивы disable_functions.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если функция отключена, FALSE противном случае.

Смотрите также

  • ReflectionFunctionAbstract::isUserDefined() - Проверяет, является ли функция определенной пользователем
  • Директива disable_functions
