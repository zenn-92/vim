Exception::__construct

(PHP 5 >= 5.1.0)

Exception::__construct — Создать исключение

Описание

public Exception::__construct ([ string $message = "" [, int $code = 0 [, Exception $previous = NULL ]]] )

Создает исключение.

Список параметров

message

Текст сообщения.

code

Код исключения.

previous

Предыдущее исключение. Используется при создания цепочки исключений.

Список изменений

Версия Описание
5.3.0 Был добавлен параметр previous.

Примечания

Замечание:

Параметр message не является бинарно-бесопасным.
