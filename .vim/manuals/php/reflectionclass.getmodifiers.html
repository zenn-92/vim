ReflectionClass::getModifiers

(PHP 5)

ReflectionClass::getModifiers — Возвращает информацию о модификаторах класса

Описание

public int ReflectionClass::getModifiers ( void )

Возвращает битовую маску модификаторов для данного метода.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает битовую маску из констант модификаторов.
