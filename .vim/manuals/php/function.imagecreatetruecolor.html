imagecreatetruecolor

(PHP 4 >= 4.0.6, PHP 5)

imagecreatetruecolor — Создание нового полноцветного изображения

Описание

resource imagecreatetruecolor ( int $width , int $height )

imagecreatetruecolor() возвращает идентификатор изображения, представляющий черное изображение заданного размера.

Определена эта функция или нет зависит от версий PHP и GD. В версиях PHP с 4.0.6 по 4.1.x она всегда существует, если GD модуль загружен, но вызов функции без установленной GD2 приведет к фатальной ошибке и выходу. В версиях PHP 4.2.x это поведение отличается тем, что вместо ошибки выбрасывается предупреждение. Другие версии определяют функцию, только если установлена корректная версия GD.

Список параметров

width

Ширина изображения.

height

Высота изображения.

Возвращаемые значения

Возвращает ресурс изображения в случае успеха, или FALSE в случае ошибки.

Примеры

Пример #1 Создание нового потока GD изображения и вывод картинки.

<?php
header ('Content-Type: image/png');
$im = @imagecreatetruecolor(120, 20)
      or die('Невозможно инициализировать GD поток');
$text_color = imagecolorallocate($im, 233, 14, 91);
imagestring($im, 1, 5, 5,  'Простая Текстовая Строка', $text_color);
imagepng($im);
imagedestroy($im);
?>
