CallbackFilterIterator::accept

(PHP 5 >= 5.4.0)

CallbackFilterIterator::accept — Вызывает callback-функцию и передает ей в качестве аргументов текущее значение, текущий ключ и внутренний указатель

Описание

public string CallbackFilterIterator::accept ( void )

Вызывает callback-функцию и передает ей в качестве аргументов текущее значение, текущий ключ и внутренний указатель.

Callback-функция должна возвращать TRUE, если текущий элемент прошел фильтр, FALSE в противном случае.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE, если текущий элемент прошел фильтр, FALSE в противном случае.
