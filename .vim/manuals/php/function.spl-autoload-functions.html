spl_autoload_functions

(PHP 5 >= 5.1.2)

spl_autoload_functions — Получение списка всех зарегистрированных фунций __autoload()

Описание

array spl_autoload_functions ( void )

Получает список всех зарегистрированных функций __autoload().

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Массив (array) имен всех зарегистрированных фунций __autoload. Если стэк функций автозагрузки не активирован, будет возвращено значение FALSE. Если зарегистрированных функций нет, будет возвращен пустой массив.
