realpath_cache_get

(PHP 5 >= 5.3.2)

realpath_cache_get — Получает записи из кэша реального пути

Описание

array realpath_cache_get ( void )

Получает содержимое кэша реального пути.

Возвращаемые значения

Возвращает массив записей из кэша реального пути. Ключи являются записями реального пути, а значения являются массивами данных, содержащих вычисленный путь, время жизни кэша и других данных, хранящихся в кэше.

Примеры

Пример #1 Пример использования realpath_cache_get()

<?php
var_dump(realpath_cache_get());
?>

Результатом выполнения данного примера будет что-то подобное:

array(2) {
  ["/test"] => array(4) {
    ["key"] => int(123456789)
    ["is_dir"] => bool(true)
    ["realpath"] => string(5) "/test"
    ["expires"] => int(1260318939)
  }
  ["/test/test.php"] => array(4) {
    ["key"] => int(987654321)
    ["is_dir"] => bool(false)
    ["realpath"] => string(12) "/root/test.php"
    ["expires"] => int(1260318939)
  }
}
