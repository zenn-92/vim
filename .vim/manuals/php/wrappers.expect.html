expect://

expect:// — Потоки для взаимодействия с процессами

Описание

Потоки, открытые с помощью обертки expect://, предоставляют доступ к stdio, stdout и stderr процессов через PTY.

Замечание: Эта обертка отключена по умолчанию
Для того, чтобы использовать обертку expect://, необходимо установить модуль » Expect, доступный в » PECL.

expect:// PHP 4.3.0 и выше (PECL)

Использование

  • expect://command
