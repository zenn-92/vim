trader_mavp

(PECL trader >= 0.2.0)

trader_mavp — Moving average with variable period

Описание

array trader_mavp ( array $real , array $periods [, integer $minPeriod [, integer $maxPeriod [, integer $mAType ]]] )

Список параметров

real

Массив, содержащий реальные значения.

periods

Массив, содержащий реальные значения.

minPeriod

Значение, меньшее минимума, будет изменено на минимальный период. Допустимы значения от 2 до 100000

maxPeriod

Значение, большее максимума, будет изменено на максимальный период. Допустимы значения от 2 до 100000

mAType

Тип скользящей средней. Следует использовать серию констант TRADER_MA_TYPE_*.

Возвращаемые значения

Returns an array with calculated data or false on failure.
