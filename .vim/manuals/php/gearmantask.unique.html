GearmanTask::unique

(PECL gearman >= 0.6.0)

GearmanTask::unique — Получение уникального идентификатора задания

Описание

public string GearmanTask::unique ( void )

Возвращает уникальный идентификатор этого задания. Этот идентификатор присваивает GearmanClient, в отличие от идентификатора объекта "работы", который присваивает сервер задач.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Уникальный идентификатор или FALSE, если никакого идентификатора не присвоено.
