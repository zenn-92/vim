acosh

(PHP 4 >= 4.1.0, PHP 5)

acosh — Гиперболический арккосинус

Описание

float acosh ( float $arg )

Возвращает гиперболический арккосинус arg, т.е. значение, чей гиперболический косинус равен arg.

Список параметров

arg

Входное значение

Возвращаемые значения

Гиперболический арккосинус arg

Список изменений

Версия Описание
5.3.0 Функция стала доступна для всех платформ
