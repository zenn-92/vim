Потоковые фильтры

Фильтр (filter) - рабочий код, который может осуществлять операции с данными, читаемыми из потока или записываемыми в него. На каждый поток можно надстроить несколько фильтров. Пользовательские фильтры можно регистрировать либо в PHP скрипте функцией stream_filter_register(), либо в расширении, используя API Reference Working with streams. Чтобы посмотреть список зарегистрированных на данный момент фильтров, воспользуйтесь функцией stream_get_filters().
