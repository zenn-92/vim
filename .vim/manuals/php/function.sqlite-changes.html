sqlite_changes

SQLiteDatabase::changes

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_changes -- SQLiteDatabase::changes — Возвращает количество записей, которые были изменены последним SQL-запросом

Описание

int sqlite_changes ( resource $dbhandle )

Объектно-ориентированный стиль (метод):

int SQLiteDatabase::changes ( void )

Возвращает количество записей, которые были изменены последним SQL-запросом к базе данных dbhandle.

Список параметров

dbhandle

Ресурс базы данных SQLite, полученный из функции sqlite_open() в случае использования процедурного подхода. Этот параметр не требуется при использовании объектно-ориентированного подхода.

Возвращаемые значения

Возвращает количество измененных записей.

Примеры

Пример #1 Процедурный стиль

<?php
$dbhandle = sqlite_open('mysqlitedb');
$query = sqlite_query($dbhandle, "UPDATE users SET email='jDoe@example.com' WHERE username='jDoe'");
if (!$query) {
    exit('Ошибка в запросе.');
} else {
    echo 'Количество измененных записей: ', sqlite_changes($dbhandle);
}
?>

Пример #2 Объектно-ориентированный стиль

<?php
$dbhandle = new SQLiteDatabase('mysqlitedb');
$query = $dbhandle->query("UPDATE users SET email='jDoe@example.com' WHERE username='jDoe'");
if (!$query) {
    exit('Ошибка в запросе.');
} else {
    echo 'Количество измененных записей: ', $dbhandle->changes();
}
?>
