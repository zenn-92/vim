stream_get_transports

(PHP 5)

stream_get_transports — Возвращает список зарегистрированных транспортов сокета

Описание

array stream_get_transports ( void )

Возвращает индексированный массив, содержащий названия всех транспортов сокета, доступных на работающей системе.

Возвращаемые значения

Возвращает индексированный массив названий транспортов сокета.

Примеры

Пример #1 Пример использования stream_get_transports()

<?php
$xportlist = stream_get_transports();
print_r($xportlist);
?>

Результатом выполнения данного примера будет что-то подобное:

Array (
  [0] => tcp
  [1] => udp
  [2] => unix
  [3] => udg
)
