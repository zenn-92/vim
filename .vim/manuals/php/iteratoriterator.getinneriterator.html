IteratorIterator::getInnerIterator

(PHP 5 >= 5.1.0)

IteratorIterator::getInnerIterator — Получает внутренний итератор

Описание

public Traversable IteratorIterator::getInnerIterator ( void )

Получает внутренний итератор.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Внутренний итератор, который был передан методу IteratorIterator::__construct().
