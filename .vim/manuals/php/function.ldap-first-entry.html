ldap_first_entry

(PHP 4, PHP 5)

ldap_first_entry — Возвратить первый идентификатор результата first result id

Описание

resource ldap_first_entry ( resource $link_identifier , resource $result_identifier )

Возвращает идентификатор для первой записи в результате. Этот идентификатор затем используется в функции ldap_next_entry(), для последовательного получения записей из результата.

Записи в LDAP результате считываются последовательно используя функции ldap_first_entry() и ldap_next_entry().

Список параметров

link_identifier

Идентификатор ссылки LDAP, возвращенный ldap_connect().

result_identifier

Возвращаемые значения

Возвращает результирующий идентификатор для первой записи в случае успеха и FALSE в случае ошибки.
