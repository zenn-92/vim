Изменения в модулях SAPI

  • Доступен новый модуль SAPI, называемый cli-server.
  • Добавлена CLI-опция --rz, которая отображает информацию о имени расширения Zend.
  • Добавлена ссылка #inisetting=value для изменения php.ini опций в процессе работы скрипта при интерактивной работе в консоли.
  • Добавлены функции, совместимые с Apache: apache_child_terminate(), getallheaders(), apache_request_headers() и apache_response_headers() для FastCGI SAPI.
  • PHP-FPM: Добавлена опция process.max для контроля количества ветвлений FPM-процессов.
