eio_nthreads

(PECL eio >= 0.0.1dev)

eio_nthreads — Возвращает количество используемых в данный момент потоков

Описание

int eio_nthreads ( void )

Список параметров

У этой функции нет параметров.

Возвращаемые значения

eio_nthreads() возвращает количество используемых в данный момент потоков.
