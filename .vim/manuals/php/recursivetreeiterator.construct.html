RecursiveTreeIterator::__construct

(PHP 5 >= 5.3.0)

RecursiveTreeIterator::__construct — Конструктор класса RecursiveTreeIterator

Описание

public RecursiveTreeIterator::__construct ( RecursiveIterator|IteratorAggregate $it [, int $flags = RecursiveTreeIterator::BYPASS_KEY [, int $cit_flags = CachingIterator::CATCH_GET_CHILD [, int $mode = RecursiveIteratorIterator::SELF_FIRST ]]] )

Создает новый объект класса RecursiveTreeIterator на основе рекурсивного объекта-итератора.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

it

Объект класса RecursiveIterator или класса IteratorAggregate.

flags

Флаги влияют на поведение некоторых методов класса. Список возможных флагов можно найти на странице Предопределенных констант RecursiveTreeIterator.

caching_it_flags

Флаги для настройки внутреннего объекта RecursiveCachingIterator.

mode

Флаги для настройки внутреннего объекта RecursiveIteratorIterator.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
