mb_split

(PHP 4 >= 4.2.0, PHP 5)

mb_split — Разделение строк в многобайтных кодировках, используя регулярное выражение

Описание

array mb_split ( string $pattern , string $string [, int $limit = -1 ] )

Разделяет многобайтную строку string, используя регулярное выражение pattern и возвращает массив array.

Список параметров

pattern

Шаблон регулярного выражения.

string

Разбиваемая строка string.

limit
Если необязятельный аргумент limit задан, функция разобьет строку не более, чем на limit частей.

Возвращаемые значения

Результат разбиения в виде массива array.

Примечания

Замечание:

Для этой функции будет использована внутренняя кодировка или кодировка, установленная функцией mb_regex_encoding().

Смотрите также

  • mb_regex_encoding() - Возвращает текущую кодировку для многобайтового регулярного выражения в виде строки
  • mb_ereg() - Совпадение с регулярным выражением с поддержкой многобайтовых символов
