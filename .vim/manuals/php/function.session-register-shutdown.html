session_register_shutdown

(PHP >=5.4.0)

session_register_shutdown — Функция завершения сессии

Описание

void session_register_shutdown ( void )

Регистрирует функцию session_write_close() в качестве функции завершения сессии.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Ошибки

При неудачной регистрации завершающей функции генерирует ошибку уровня E_WARNING.
