SplObserver::update

(PHP 5 >= 5.1.0)

SplObserver::update — Получает уведомление от субъекта

Описание

abstract public void SplObserver::update ( SplSubject $subject )

Этот метод вызывается, когда любой SplSubject, к которому присоединен наблюдатель, вызывает SplSubject::notify().

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

subject

SplSubject, уведомляющий наблюдателя об обновлении.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
