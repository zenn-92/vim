ReflectionClass::isInternal

(PHP 5)

ReflectionClass::isInternal — Проверяет, является ли класс встроенным в расширение или в ядро

Описание

public bool ReflectionClass::isInternal ( void )

Проверяет, является ли класс встроенным в расширение или в ядро, а не пользовательским.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования ReflectionClass::isInternal()

<?php
$internalclass = new ReflectionClass('ReflectionClass');

class Apple {}
$userclass = new ReflectionClass('Apple');

var_dump($internalclass->isInternal());
var_dump($userclass->isInternal());
?>

Результат выполнения данного примера:

bool(true)
bool(false)
