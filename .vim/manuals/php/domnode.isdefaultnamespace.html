DOMNode::isDefaultNamespace

(PHP 5)

DOMNode::isDefaultNamespace — Проверяет, совпадает ли URI пространства имен узла с пространством имен по умолчанию

Описание

public bool DOMNode::isDefaultNamespace ( string $namespaceURI )

Сообщает, является ли namespaceURI пространством имен по умолчанию.

Список параметров

namespaceURI

URI пространства имен.

Возвращаемые значения

Возвращает TRUE если namespaceURI является пространством имен по умолчанию или FALSE в противном случае.
