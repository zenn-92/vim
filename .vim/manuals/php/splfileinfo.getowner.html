SplFileInfo::getOwner

(PHP 5 >= 5.1.2)

SplFileInfo::getOwner — Определяет владельца файла

Описание

public int SplFileInfo::getOwner ( void )

Определяет владельца файла. ID владельца файла возвращается в виде числа.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

ID владельца файла в виде числа.

Ошибки

В случае ошибки выбрасывает исключение RuntimeException.

Примеры

Пример #1 Пример использования SplFileInfo::getOwner()

<?php
$info = new SplFileInfo('file.txt');
print_r(posix_getpwuid($info->getOwner()));
?>
