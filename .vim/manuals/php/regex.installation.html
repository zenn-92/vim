Установка

Внимание

Не изменяйте TYPE, если вы не знаете что делаете.

Для включения поддержки регулярных выражений необходимо скомпилировать PHP c ключом --with-regex[=TYPE] , где TYPE может принимать значение system, apache, php. По умолчанию используется php.

Версия PHP для Windows имеет встроенную поддержку данного расширения. Это означает, что для использования данных функций не требуется загрузка никаких дополнительных расширений.
