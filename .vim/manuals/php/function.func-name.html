GearmanWorker::setId

(Информация о версии неизвестна, возможно, только в SVN)

GearmanWorker::setId — Назначает обработчику идентификатор, чтобы в дальнейшем иметь возможность опросить все доступные обработчики

Описание

public bool GearmanWorker::setId ( string $id )

Назначает обработчику идентификатор.

Список параметров

id

Строковый идентификатор.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования GearmanWorker::setId()

Задание идентификатора для простого обработчика.

<?php
$worker= new GearmanWorker();
$worker->setId('test');
?>

Результатом выполнения данного примера будет что-то подобное:

Run the following command:
gearadmin --workers

Output:
30 ::3a3a:3361:3361:3a33%976303667 - : test
