getmygid

(PHP 4 >= 4.1.0, PHP 5)

getmygid — Получение GID владельца PHP скрипта

Описание

int getmygid ( void )

Получает идентификатор группы текущего скрипта.

Возвращаемые значения

Возвращает групповой идентификатор текущего скрипта или FALSE в случае ошибки.
