SplQueue::__construct

(PHP 5 >= 5.3.0)

SplQueue::__construct — Создает новую очередь, реализованную с использованием двусвязного списка

Описание

SplQueue::__construct ( void )

Создаёт новую пустую очередь.

Замечание:

Этот метод автоматически устанавливает режим итератора SplDoublyLinkedList::IT_MODE_FIFO.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования SplQueue::__construct()

<?php
$q = new SplQueue();

$q[] = 1;
$q[] = 2;
$q[] = 3;

foreach ($q as $elem)  {
 echo $elem."\n";
}
?>

Результат выполнения данного примера:

1
2
3

Пример #2 Эффективная обработка задач с помощью SplQueue

<?php
$q = new SplQueue();
$q->setIteratorMode(SplQueue::IT_MODE_DELETE);

// ... поставить несколько задач в очередь ...

// обработать их
foreach ($q as $task) {
    // ... обработать $task ...

    // добавить новые задачи в очередь
    $q[] = $newTask;
    // ...
}
?>
