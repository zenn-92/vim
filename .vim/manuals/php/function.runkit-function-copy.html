runkit_function_copy

(PECL runkit >= 0.7.0)

runkit_function_copy — Копирует функцию с новым именем

Описание

bool runkit_function_copy ( string $funcname , string $targetname )

Список параметров

funcname

Имя существующей функции

targetname

Имя новой функции, в которую необходимо копировать существующую

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример runkit_function_copy()

<?php
function original() {
  echo "В функции\n";
}
runkit_function_copy('original','duplicate');
original();
duplicate();
?>
