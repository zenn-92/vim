DOMElement::getElementsByTagNameNS

(PHP 5)

DOMElement::getElementsByTagNameNS — Получение элементов по локальному имени в заданном пространстве имен

Описание

public DOMNodeList DOMElement::getElementsByTagNameNS ( string $namespaceURI , string $localName )

Эта функция выбирает все элементы-потомки с заданным именем localName и пространством имен namespaceURI.

Список параметров

namespaceURI

URI пространства имен.

localName

Локальное имя. Используйте * для выбора всех элементов из дерева.

Возвращаемые значения

Эта функция возвращает новый объект класса DOMNodeList содержащий все подошедшие под шаблон выбора элементы в том порядке, в котором они появляются при прямом обходе дерева.
