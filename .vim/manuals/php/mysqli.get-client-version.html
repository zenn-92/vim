mysqli_get_client_version

mysqli::$client_version

(PHP 5)

mysqli_get_client_version -- mysqli::$client_version — Возвращает информацию о клиенте MySQL в виде строки

Описание

Объектно-ориентированный стиль

int $mysqli->client_version;

Процедурный стиль

int mysqli_get_client_version ( mysqli $link )

Возвращает версию клиента в виде целого числа.

Возвращаемые значения

Число, содержащее версию клиентской библиотеки MySQL в следующем формате: main_version*10000 + minor_version *100 + sub_version. К примеру, для версии 4.1.0 выведет 40100.

Целесообразно использовать для быстрого определения версии клиентской библиотеки и возможностей её применения.

Примеры

Пример #1 mysqli_get_client_version

<?php

/* Для определения версии клиентской библиотеки MySQL
 нет необходимости в создании соединения*/

printf("Версия клиентской библиотеки: %d\n", mysqli_get_client_version());
?>
