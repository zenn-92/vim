zip_entry_read

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PECL zip >= 1.0.0)

zip_entry_read — Читает данные из открытого ранее дескриптора директории

Описание

string zip_entry_read ( resource $zip_entry [, int $length ] )

Читает данные из открытого ранее дескриптора директории.

Список параметров

zip_entry

Дескриптор директории, возвращаемый функцией zip_read().

length

Число байт, которое нужно считать. Если параметр не определен, то функция будет пытаться считать 1024 байта.

Замечание:

В параметре необходимо указывать объем данных, которые вы хотите считать, без учета сжатия.

Возвращаемые значения

Возвращает считанные данные или FALSE если достигнут конец файла.

Смотрите также

  • zip_entry_open() - Открывает директорию для чтения
  • zip_entry_close() - Закрывает дескриптор директории
  • zip_entry_filesize() - Возвращает реальный размер файла для дескриптора директории
