SphinxClient::setIDRange

(PECL sphinx >= 0.1.0)

SphinxClient::setIDRange — Установка диапазона значений допустимых ID документов

Описание

public bool SphinxClient::setIDRange ( int $min , int $max )

Устанавливает допустимый диапазон ID документов. Диапазон по умолчанию - от 0 до 0, то есть без ограничений. В список результатов попадут только документы, имеющие ID документа в границах между min и max (включая ID, в точности равные min или max).

Список параметров

min

Минимальное значение ID.

max

Максимальное значение ID.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
