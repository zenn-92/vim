GearmanJob::workload

(PECL gearman >= 0.5.0)

GearmanJob::workload — Получение данных для обработки

Описание

public string GearmanJob::workload ( void )

Возвращает данные, переданные для обработки. Это сериализованные данные, с которыми будет работать обработчик.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Сериализованные данные.
