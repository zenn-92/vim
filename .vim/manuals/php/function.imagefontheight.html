imagefontheight

(PHP 4, PHP 5)

imagefontheight — Получение высоты шрифта

Описание

int imagefontheight ( int $font )

Возвращает высоту в пикселах символа заданного шрифта.

Список параметров

font

Может принимать значения 1, 2, 3, 4, 5 для встроенных шрифтов в кодировке latin2 (более высокое число соответствует большему шрифту) или любому из ваших собственных идентификаторов шрифтов, зарегистрированных с помощью imageloadfont().

Возвращаемые значения

Возвращает высоту в пикселах.

Примеры

Пример #1 Пример использования imagefontheight() на встроенных шрифтах

<?php
echo 'Font height: ' . imagefontheight(4);
?>

Результатом выполнения данного примера будет что-то подобное:

Font height: 16

Пример #2 Использование imagefontheight() вместе с imageloadfont()

<?php
// загрузка .gdf шрифта
$font = imageloadfont('anonymous.gdf');

echo 'Высота шрифта: ' . imagefontheight($font);
?>

Результатом выполнения данного примера будет что-то подобное:

Font height: 43

  • imageloadfont() - Загрузка шрифта
