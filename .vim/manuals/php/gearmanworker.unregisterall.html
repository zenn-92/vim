GearmanWorker::unregisterAll

(PECL gearman >= 0.6.0)

GearmanWorker::unregisterAll — Удаление регистрации всех имен функций на серверах заданий

Описание

public bool GearmanWorker::unregisterAll ( void )

Снимает регистрацию всех ранее зарегистрированных функций на всех серверах заданий. Это значит, что этому обработчику больше не будут поступать никакие задачи.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Стандартное возвращаемое значение Gearman.

Смотрите также

  • GearmanWorker::register() - Регистрация функции на сервере заданий
  • GearmanWorker::unregister() - Удалить регистрацию имени функции на всех серверах заданий
