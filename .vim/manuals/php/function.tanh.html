tanh

(PHP 4 >= 4.1.0, PHP 5)

tanh — Гиперболический тангенс

Описание

float tanh ( float $arg )

Возвращает гиперболический тангенс arg, определяемый как sinh(arg)/cosh(arg).

Список параметров

arg

Входное значение

Возвращаемые значения

Гиперболический тангенс arg
