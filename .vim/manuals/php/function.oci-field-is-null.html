oci_field_is_null

(PHP 5, PECL OCI8 >= 1.1.0)

oci_field_is_null — Проверяет, равняется ли поле NULL

Описание

bool oci_field_is_null ( resource $statement , mixed $field )

Проверяет, является ли значение, переданное в поле field из запроса statement равным NULL.

Список параметров

statement

Идентификатор допустимого OCI запроса.

field

Может быть порядковый номер поля или имя поля (в верхнем регистре).

Возвращаемые значения

Возвращает TRUE, если field равен NULL, и FALSE если нет.

Примечания

Замечание:

В версиях PHP ниже 5.0.0 эта функция называлась ocicolumnisnull(). В PHP 5.0.0 и выше ocicolumnisnull() является алиасом oci_field_is_null(), поэтому вы можете продолжать использовать это имя, однако это не рекомендуется.
