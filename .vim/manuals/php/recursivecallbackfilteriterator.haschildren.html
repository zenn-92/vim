RecursiveCallbackFilterIterator::hasChildren

(PHP 5 >= 5.4.0)

RecursiveCallbackFilterIterator::hasChildren — Проверяет, содержит ли текущий элемент внутреннего итератора дочерние элементы

Описание

public void RecursiveCallbackFilterIterator::hasChildren ( void )

Возвращает TRUE, если у текущего элемента есть дочерние, и FALSE, если нет.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE, если у текущего элемента есть дочерние, и FALSE, если нет.

Примеры

Пример #1 Пример использования RecursiveCallbackFilterIterator::hasChildren()

<?php

$dir = new FilesystemIterator(__DIR__);

// рекурсивный обход XML файлов
$files = new RecursiveCallbackFilterIterator($dir, function ($current, $key, $iterator) {
    // просматривать директории
    if ($iterator->hasChildren()) {
        return TRUE;
    }
    // отбор XML файлов
    if (!strcasecmp($current->getExtension(), 'xml')) {
        return TRUE;
    }
    return FALSE;
});

?>
