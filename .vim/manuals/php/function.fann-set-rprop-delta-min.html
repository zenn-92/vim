fann_set_rprop_delta_min

(PECL fann >= 1.0.0)

fann_set_rprop_delta_min — Sets the minimum step-size

Описание

bool fann_set_rprop_delta_min ( resource $ann , float $rprop_delta_min )

The minimum step-size is a small positive number determining how small the minimum step-size may be.

Список параметров

ann

Ресурс (resource) нейронной сети.

rprop_delta_min

The minimum step-size.

Возвращаемые значения

Возвращает TRUE в случае успешного выполнения, или FALSE в противном случае.
