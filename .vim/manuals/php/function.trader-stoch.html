trader_stoch

(PECL trader >= 0.2.0)

trader_stoch — Stochastic

Описание

array trader_stoch ( array $high , array $low , array $close [, integer $fastK_Period [, integer $slowK_Period [, integer $slowK_MAType [, integer $slowD_Period [, integer $slowD_MAType ]]]]] )

Список параметров

high

Высокая цена, массив реальных значений.

low

Низкая цена, массив реальных значений.

close

Цена закрытия, массив реальных значений.

fastK_Period

Временной период для построения линии Fast-K. Допустимы значения от 1 до 100000.

slowK_Period

Сглаживание для создания линии Slow-K. Допустимы значения от 1 до 100000, обычно установлено значение 3.

slowK_MAType

Тип скользящей средней для Slow-K. Следует использовать серию констант TRADER_MA_TYPE_*.

slowD_Period

Сглаживание для создания линии Slow-D. Допустимы значения от 1 до 100000.

slowD_MAType

Тип скользящей средней для Slow-D. Следует использовать серию констант TRADER_MA_TYPE_*.

Возвращаемые значения

Returns an array with calculated data or false on failure.
