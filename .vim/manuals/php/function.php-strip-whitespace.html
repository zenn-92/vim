php_strip_whitespace

(PHP 5)

php_strip_whitespace — Возвращает исходный код без комментариев и пробелов

Описание

string php_strip_whitespace ( string $filename )

Возвращает исходный код PHP в файл (filename) исключая комментарии и пробелы. Эта функция может быть полезной для определения актуального объема чистого кода в скриптах по сравнению с объемом комментариев. Функция аналогична использованию php -w из командной строки.

Список параметров

filename

Путь к PHP файлу.

Возвращаемые значения

Возвращает очищенный исходный код, или пустую строку в случае ошибки.

Замечание:

Эта функция работает как описано в PHP 5.0.1. Перед этим может быть возвращена пустая строка. Для более детальной информации смотрите отчет об ошибках » #29606.

Примеры

Пример #1 Пример использования php_strip_whitespace()

<?php
// PHP комментарий

/*
 * Другой PHP комментарий
 */

echo        php_strip_whitespace(__FILE__);
// Символы новой строки считаются пробелами, и также удаляются:
do_nothing();
?>

Результат выполнения данного примера:

<?php
 echo php_strip_whitespace(__FILE__); do_nothing(); ?>

В результате выполнения примера выведен исходный код PHP, исключая комментарии, пробелы и пустые строки.
