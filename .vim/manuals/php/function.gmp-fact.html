gmp_fact

(PHP 4 >= 4.0.4, PHP 5)

gmp_fact — Факториал

Описание

resource gmp_fact ( mixed $a )

Вычисляет факториал (a!) числа a.

Список параметров

a

Число, факториал которого вычисляется.

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

Возвращаемые значения

Число GMP.

Примеры

Пример #1 Пример использования gmp_fact()

<?php
$fact1 = gmp_fact(5); // 5 * 4 * 3 * 2 * 1
echo gmp_strval($fact1) . "\n";

$fact2 = gmp_fact(50); // 50 * 49 * 48, ... и т.д.
echo gmp_strval($fact2) . "\n";
?>
