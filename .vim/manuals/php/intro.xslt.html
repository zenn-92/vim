Введение

Это PHP расширение предоставляет независимый от процессора API для XSLT преобразований. На данный момент поддерживается только библиотека Sablotron от Ginger Alliance. Планируется поддержка и других библиотек, таких как Xalan или libxslt.

XSLT (Extensible Stylesheet Language (XSL) Transformations) - язык для преобразований XML документов в другие XML документы. Это стандарт, описанный Консорциумом Всемирной Паутины (W3C). Информацию о XSLT и сопутствующих технологиях можно найти на » http://www.w3.org/TR/xslt.

Замечание: Это расширение отличается от sablotron расширения выпущенного для версий PHP ниже 4.1.0. На данный момент в PHP 4.1.0 поддерживается только новое расширение. Если нужна поддержка старых расширений, обращайтесь в списки рассылки PHP.

Замечание:

Это расширение было перемещено в репозиторий » PECL и больше не поставляется с PHP 5.0.0.

Замечание: Если требуется поддержка xslt в PHP 5, можно воспользоваться XSL расширением.
