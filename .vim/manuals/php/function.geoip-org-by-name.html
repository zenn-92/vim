geoip_org_by_name

(PECL geoip >= 0.2.0)

geoip_org_by_name — Возвращает название организации, владеющей IP адресом

Описание

string geoip_org_by_name ( string $hostname )

Функция geoip_org_by_name() возвращает название организации, которая владеет указаным IP адресом.

Эта функция доступна только для тех, кто приобрел коммерческую версию GeoIP Organization, ISP ил AS . Если такая база отсутствует, выводится предупреждение.

Список параметров

hostname

Имя хоста или IP адрес.

Возвращаемые значения

Возвращает имя организации в случае успеха или FALSE, если адрес не может быть найден в базе данных.

Примеры

Пример #1 Пример использованияgeoip_org_by_name()

Будет выведено название организации, которой принадлежит IP адрес хоста example.com.

<?php
$org = geoip_org_by_name('www.example.com');
if ($org) {
    echo 'Владелец данного адреса: ' . $org;
}
?>

Результат выполнения данного примера:

Владелец данного адреса: ICANN c/o Internet Assigned Numbers Authority
