XMLWriter::startDTDEntity

xmlwriter_start_dtd_entity

(PHP 5 >= 5.1.2, PECL xmlwriter >= 0.1.0)

XMLWriter::startDTDEntity -- xmlwriter_start_dtd_entity — Создать начальную запись DTD

Описание

Объектно-ориентированный стиль

bool XMLWriter::startDTDEntity ( string $name , bool $isparam )

Процедурный стиль

bool xmlwriter_start_dtd_entity ( resource $xmlwriter , string $name , bool $isparam )

Начинает запись DTD.

Список параметров

xmlwriter

Только для процедурных вызовов. Изменяемый resource XMLWriter. Этот ресурс может быть получен из xmlwriter_open_uri() или xmlwriter_open_memory().

name

Имя записи.

isparam

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

  • XMLWriter::writeDTDEntity() - Записать полный тэг DTD записи
