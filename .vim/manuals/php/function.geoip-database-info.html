geoip_database_info

(PECL geoip >= 0.2.0)

geoip_database_info — Возвращает информацию о базе GeoIP

Описание

string geoip_database_info ([ int $database = GEOIP_COUNTRY_EDITION ] )

Функция geoip_database_info() возвращает версию базы GeoIP, соответствующую определению в бинарном файле.

Эта функция вызывается без параметров и возвращает версию базы GeoIP Free Country Edition.

Список параметров

database

Тип базы определяется целым числом (integer). Вы можете использовать разнообразные константы, определённые в этом расширении (ie: GEOIP_*_EDITION).

Возвращаемые значения

Возвращает подходящую версию, или NULL при ошибке.

Примеры

Пример #1 Пример использования geoip_database_info()

Пример демонстрирует вывод информации о базе данных.

<?php
print geoip_database_info(GEOIP_COUNTRY_EDITION);
?>
