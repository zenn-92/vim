InfiniteIterator::next

(PHP 5 >= 5.1.0)

InfiniteIterator::next — Перемещает итератор на одну позицию вперед или в начало

Описание

public void InfiniteIterator::next ( void )

Перемещает внутренний указатель объекта Iterator на следующий элемент или на первый элемент, если текущий элемент был последним.

Замечание:

Итератор InfiniteIterator останавливается, когда хранящийся внутри объект Iterator оказывается пустым.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
