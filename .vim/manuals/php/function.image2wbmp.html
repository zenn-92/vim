image2wbmp

(PHP 4 >= 4.0.5, PHP 5)

image2wbmp — Выводит изображение в браузер или пишет в файл

Описание

bool image2wbmp ( resource $image [, string $filename [, int $threshold ]] )

image2wbmp() выводит либо сохраняет WBMP версию заданного изображения image.

Список параметров

image

Ресурс изображения, полученный одной из функций создания изображений, например, такой как imagecreatetruecolor().

filename

Путь для сохранения файла. Если не задан, исходные данные изображения будут выведены напрямую в выходной поток.

threshold

Пороговое значение, может быть между 0 и 255 (включительно).

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования image2wbmp()

<?php
$file = 'php.png';
$image = imagecreatefrompng($file);

header('Content-Type: ' . image_type_to_mime_type(IMAGETYPE_WBMP));
image2wbmp($image); // вывод потока
imagedestroy($image);
?>

Примечания

Замечание:

Поддержка WBMP доступна, только если PHP скомпилирован с GD-1.8 или старше.
