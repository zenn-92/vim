GearmanTask::isKnown

(PECL gearman >= 0.5.0)

GearmanTask::isKnown — Определение, известно ли серверу об этом задании

Описание

public bool GearmanTask::isKnown ( void )

Определяет, известно ли серверу задач об этом задании.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если задание известно, FALSE в противном случае.
