php_logo_guid

(PHP 4, PHP 5 < 5.5)

php_logo_guid — Получение guid логотипа

Описание

string php_logo_guid ( void )

Функция возвращает ID, который можно использовать для вывода логотипа PHP, используя встроенное изображение. Логотип отображается, только если включена настройка expose_php.

Внимание

Эта функция была помечена УСТАРЕВШЕЙ и была УДАЛЕНА начиная с версии PHP 5.5.0.

Возвращаемые значения

Возвращает PHPE9568F34-D428-11d2-A769-00AA001ACF42.

Список изменений

Версия Описание
5.5.0 php_logo_guid() удалена из PHP.

Примеры

Пример #1 Пример использования php_logo_guid()

<?php

echo '<img src="' . $_SERVER['PHP_SELF'] .
     '?=' . php_logo_guid() . '" alt="Логотип PHP!" />';

?>
