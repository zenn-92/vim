ZipArchive::statName

(PHP 5 >= 5.2.0, PECL zip >= 1.5.0)

ZipArchive::statName — Получение детальной информации о элементе по его имени

Описание

mixed ZipArchive::statName ( name $name [, int $flags ] )

Получение детальной информации о элементе по его имени.

Список параметров

name

Имя элемента.

flags

Флаг, указывающий как должен происходить поиск имени. Также, может указываться ZIPARCHIVE::FL_UNCHANGED, чтобы запросить информацию об исходном файле в архиве, игнорируя любые внесенные изменения.

  • ZIPARCHIVE::FL_NOCASE

  • ZIPARCHIVE::FL_NODIR

  • ZIPARCHIVE::FL_UNCHANGED

Возвращаемые значения

Возвращает массив, содержащий детальную информацию о элементе или или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Получение статистической информации о элементе

<?php
$zip = new ZipArchive;
$res = $zip->open('test.zip');
if ($res === TRUE) {
    print_r($zip->statName('foobar/baz'));
    $zip->close();
} else {
    echo 'failed, code:' . $res;
}
?>

Результатом выполнения данного примера будет что-то подобное:

Array
(
    [name] => foobar/baz
    [index] => 3
    [crc] => 499465816
    [size] => 27
    [mtime] => 1123164748
    [comp_size] => 24
    [comp_method] => 8
)
