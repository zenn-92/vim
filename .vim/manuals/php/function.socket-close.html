socket_close

(PHP 4 >= 4.1.0, PHP 5)

socket_close — Закрывает ресурс сокета

Описание

void socket_close ( resource $socket )

Функция socket_close() закрывает ресурс сокета, указанный параметром socket. Эта функция работает только с сокетами и не может быть использована с каким-нибудь другим типом ресурсов.

Список параметров

socket

Действующий ресурс сокета, созданный при помощи функций socket_create() или socket_accept().

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Смотрите также

  • socket_bind() - Привязывает имя к сокету
  • socket_listen() - Listens for a connection on a socket
  • socket_create() - Create a socket (endpoint for communication)
  • socket_strerror() - Возвращает строку, описывающую ошибку сокета
