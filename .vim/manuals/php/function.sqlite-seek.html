sqlite_seek

SQLiteResult::seek

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_seek -- SQLiteResult::seek — Переходит к определенному номеру записи в буферизованном результирующем наборе

Описание

bool sqlite_seek ( resource $result , int $rownum )

Объектно-ориентированный стиль (метод):

bool SQLiteResult::seek ( int $rownum )

sqlite_seek() перемещает внутренний указатель к записи с порядковым номером, заданным аргументом rownum.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

rownum

Порядковый номер записи, к которой производится перемещение. Нумерация начинается с нуля (0 указывает на первую запись).

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Возвращаемые значения

Возвращает FALSE, если запись не существует, иначе возвращается TRUE.

Смотрите также

  • sqlite_next() - Переходит на следующую запись
  • sqlite_current() - Возвращает текущую запись из результирующего набора в виде массива
  • sqlite_rewind() - Переходит к первой записи
