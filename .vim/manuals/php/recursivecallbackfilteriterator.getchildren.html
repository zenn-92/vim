RecursiveCallbackFilterIterator::getChildren

(PHP 5 >= 5.4.0)

RecursiveCallbackFilterIterator::getChildren — Возвращает дочерние элементы итератора, хранящегося внутри RecursiveCallbackFilterIterator

Описание

public RecursiveCallbackFilterIterator RecursiveCallbackFilterIterator::getChildren ( void )

Выбирает дочерние элементы внутреннего итератора, подходящие под условия фильтра.

Прежде чем вызывать этот метод, необходимо удостовериться, что внутренний итератор содержит дочерние элементы. Сделать это можно с помощью метода RecursiveCallbackFilterIterator::hasChildren().

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает объект RecursiveCallbackFilterIterator содержащий дочерние элементы внутреннего итератора.
