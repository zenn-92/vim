mysqli_bind_result

(PHP 5 < 5.4.0)

mysqli_bind_result — Синоним функции mysqli_stmt_bind_result()

Описание

Эта функция является синонимом функции mysqli_stmt_bind_result().

Внимание

Данная функция была помечена УСТАРЕВШЕЙ начиная с версии PHP 5.3.0 и была УДАЛЕНА в версии PHP 5.4.0.
