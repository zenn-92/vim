SplFileInfo::getFileInfo

(PHP 5 >= 5.1.2)

SplFileInfo::getFileInfo — Получает объект SplFileInfo для файла

Описание

public SplFileInfo SplFileInfo::getFileInfo ([ string $class_name ] )

Этот метод получает объект SplFileInfo для указанного файла.

Список параметров

class_name

Имя производного от SplFileInfo класса для использования.

Возвращаемые значения

Объект SplFileInfo, созданный для файла.

Смотрите также

  • SplFileInfo::setInfoClass() - Задает имя класса, объекты которого будут создаваться методами getFileInfo и getPathInfo
