fann_destroy_train

(PECL fann >= 1.0.0)

fann_destroy_train — Destructs the training data

Описание

bool fann_destroy_train ( resource $train_data )

Destructs the training data

Список параметров

train_data

Ресурс (resource) обучающих данных нейронной сети.

Возвращаемые значения

Возвращает TRUE в случае успешного выполнения, или FALSE в противном случае.
