printer_logical_fontheight

(PECL printer SVN)

printer_logical_fontheight — Получает логическую высоту шрифта

Описание

int printer_logical_fontheight ( resource $printer_handle , int $height )

Функция вычисляет логическую высоту шрифта height.

Список параметров

printer_handle

printer_handle должен быть действительным дескриптором шрифта.

height

Высота шрифта.

Возвращаемые значения

Возвращает логическую высоту шрифта или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример printer_logical_fontheight()

<?php
$handle = printer_open();
echo printer_logical_fontheight($handle, 72);
printer_close($handle);
?>
