ReflectionExtension::__clone

(PHP 5)

ReflectionExtension::__clone — Клонирует объект

Описание

final private void ReflectionExtension::__clone ( void )

clone-метод предотвращает клонирование объекта. Reflection-объекты не могут быть клонированы.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Метод ничего не возвращает, в случае вызова возникает фатальная ошибка.
