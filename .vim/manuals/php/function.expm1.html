expm1

(PHP 4 >= 4.1.0, PHP 5)

expm1 — Возвращает exp(number) - 1, рассчитанное таким образом, что результат точен, даже если number близок к нулю.

Описание

float expm1 ( float $arg )

expm1() возвращает эквивалент выражения 'exp(arg) - 1', рассчитанный таким образом, что результат точен, даже если значение arg близко к нулю, тогда как 'exp (arg) - 1' будет неточно из-за вычитания двух почти равных чисел.

Список параметров

arg

Входное значение

Возвращаемые значения

'e' в степени arg минус один

Список изменений

Версия Описание
5.3.0 Теперь функция доступна на всех платформах
