ReflectionParameter::getDeclaringFunction

(PHP 5 >= 5.2.3)

ReflectionParameter::getDeclaringFunction — Получение объявляющей функции

Описание

public ReflectionFunction ReflectionParameter::getDeclaringFunction ( void )

Получает функцию, в которой аргумент объявлен.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Объект класса ReflectionFunction.
