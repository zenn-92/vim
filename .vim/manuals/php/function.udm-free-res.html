udm_free_res

(PHP 4 >= 4.0.5, PHP 5 <= 5.0.5, PECL mnogosearch >= 1.0.0)

udm_free_res — Освобождает память, выделенную для результатов mnoGoSearch

Описание

bool udm_free_res ( resource $res )

Освобождает память, выделенную для результатов.

Список параметров

res

Ссылка на идентификатор результата, полученная после вызова udm_find().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
