ReflectionExtension::getClassNames

(PHP 5)

ReflectionExtension::getClassNames — Получение имен классов

Описание

public array ReflectionExtension::getClassNames ( void )

Получает список имен классов, как они определены в расширении.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Массив array имен классов, определенных в расширении. Если никаких классов не определено, будет возвращен пустой массив.

Примеры

Пример #1 Пример использования ReflectionExtension::getClassNames()

<?php
$ext = new ReflectionExtension('XMLWriter');
var_dump($ext->getClassNames());
?>

Результатом выполнения данного примера будет что-то подобное:

array(1) {
  [0]=>
  string(9) "XMLWriter"
}
