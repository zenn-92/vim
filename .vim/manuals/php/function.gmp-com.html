gmp_com

(PHP 4 >= 4.0.4, PHP 5)

gmp_com — Вычисляет дополнение до единицы числа

Описание

resource gmp_com ( resource $a )

Возвращает дополнение до единицы числа a.

Список параметров

a

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

Возвращаемые значения

Возвращает дополнение до единицы числа a в виде числа GMP.

Примеры

Пример #1 Пример использования gmp_com()

<?php
$com = gmp_com("1234");
echo gmp_strval($com) . "\n";
?>
