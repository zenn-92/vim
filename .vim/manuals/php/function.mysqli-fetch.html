mysqli_fetch

(PHP 5 < 5.4.0)

mysqli_fetch — Синоним для mysqli_stmt_fetch()

Описание

Функция является синонимом mysqli_stmt_fetch().

Внимание

Данная функция была помечена УСТАРЕВШЕЙ начиная с версии PHP 5.3.0 и была УДАЛЕНА в версии PHP 5.4.0.
