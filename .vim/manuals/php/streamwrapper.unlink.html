streamWrapper::unlink

(PHP 5)

streamWrapper::unlink — Удаление файла

Описание

public bool streamWrapper::unlink ( string $path )

Этот метод вызывается в результате вызова unlink().

Замечание:

Чтобы сообщения об ошибках соответствовали реальным ошибкам, этот метод не нужно определять в случах, когда обертка не поддерживает удаление файлов.

Список параметров

path

URL удаляемого файла.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Ошибки

Вызывает ошибку уровня E_WARNING, если вызов к этому методу не удался (например, не реализован).

Примечания

Замечание:

Свойство streamWrapper::$context будет обновлено, если корректный контекст был передан в вызываемую функцию.
