SplFileObject::ftell

(PHP 5 >= 5.1.0)

SplFileObject::ftell — Определение текущей позиции файлового указателя

Описание

public int SplFileObject::ftell ( void )

Возвращает текущую позицию файлового указателя, или, другими словами, текущее смещение в файловом потоке.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает позицию файлового указателя в виде целого числа или FALSE в случае ошибки.

Примеры

Пример #1 Пример использования SplFileObject::ftell()

<?php
$file = new SplFileObject("/etc/passwd");

// читаем первую строку
$data = $file->fgets();

// определяем, где мы?
echo $file->ftell();
?>
