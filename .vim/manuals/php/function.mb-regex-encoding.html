mb_regex_encoding

(PHP 4 >= 4.2.0, PHP 5)

mb_regex_encoding — Возвращает текущую кодировку для многобайтового регулярного выражения в виде строки

Описание

mixed mb_regex_encoding ([ string $encoding ] )

Возвращает текущую кодировку для многобайтового регулярного выражения в виде строки string.

Список параметров

encoding

Параметр encoding представляет собой символьную кодировку. Если он опущен, вместо него будет использовано значение внутренней кодировки.

Возвращаемые значения

Возвращает кодировку символов, используемую функциями для работы с многобайтовыми регулярными выражениями.

Смотрите также

  • mb_internal_encoding() - Установка/получение внутренней кодировки скрипта
  • mb_ereg() - Совпадение с регулярным выражением с поддержкой многобайтовых символов
