cosh

(PHP 4 >= 4.1.0, PHP 5)

cosh — Гиперболический косинус

Описание

float cosh ( float $arg )

Возвращает гиперболический косинус arg, определяемый как (exp(arg) + exp(-arg))/2.

Список параметров

arg

Входное значение

Возвращаемые значения

Гиперболический косинус arg
