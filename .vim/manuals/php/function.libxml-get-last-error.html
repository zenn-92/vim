libxml_get_last_error

(PHP 5 >= 5.1.0)

libxml_get_last_error — Извлечение последней ошибки из libxml

Описание

LibXMLError libxml_get_last_error ( void )

Извлечение последней ошибки из libxml.

Возвращаемые значения

Возвращает объект LibXMLError, если в буфере есть ошибки, либо FALSE в противном случае.
