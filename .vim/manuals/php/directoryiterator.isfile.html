DirectoryIterator::isFile

(PHP 5)

DirectoryIterator::isFile — Определяет, является ли текущий элемент DirectoryIterator обычным файлом

Описание

public bool DirectoryIterator::isFile ( void )

Определяет, является ли текущий элемент DirectoryIterator обычным файлом.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE если файл существует и является обычным файлом (т.е., не является ссылкой или директорией), иначе FALSE

Примеры

Пример #1 Пример использования DirectoryIterator::isFile() example

Этот пример выведет список всех файлов в директории, содержащей выполняемый скрипт.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    if ($fileinfo->isFile()) {
        echo $fileinfo->getFilename() . "\n";
    }
}
?>

Результатом выполнения данного примера будет что-то подобное:

apple.jpg
banana.jpg
example.php
pears.jpg
