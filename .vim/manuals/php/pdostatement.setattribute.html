PDOStatement::setAttribute

(PHP 5 >= 5.1.0, PECL pdo >= 0.2.0)

PDOStatement::setAttribute — Присваивает атрибут объекту PDOStatement

Описание

bool PDOStatement::setAttribute ( int $attribute , mixed $value )

Задает атрибут объекта PDOStatement. На данный момент никаких общих атрибутов устанавливать нельзя, только характерные для конкретного драйвера:

  • PDO::ATTR_CURSOR_NAME (Firebird и ODBC): Задает имя курсора для запросов вида UPDATE ... WHERE CURRENT OF.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

  • PDO::setAttribute() - Присвоение атрибута
