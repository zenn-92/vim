tidy::body

tidy_get_body

(PHP 5, PECL tidy 0.5.2-1.0)

tidy::body -- tidy_get_body — Возвращает объект tidyNode, начиная с тега <body> разобранного tidy-дерева

Описание

Объектно-ориентированный стиль

tidyNode tidy::body ( void )

Процедурный стиль

tidyNode tidy_get_body ( tidy $object )

Возвращает объект tidyNode, начиная с тега <body> разобранного tidy-дерева.

Список параметров

object

Объект Tidy.

Возвращаемые значения

Возвращает объект tidyNode.

Внимание

Эта функция еще не реализована в самом Tidylib, поэтому всегда возвращается 0.

Примеры

Пример #1 Пример использования функции tidy::getBody()

<?php
$html = '
<html>
  <head>
    <title>test</title>
  </head>
  <body>
    <p>paragraph</p>
  </body>
</html>';

$tidy = tidy_parse_string($html);

$body = $tidy->Body();
echo $body->value;
?>

Результат выполнения данного примера:

<body>
<p>paragraph</p>
</body>

Примечания

Замечание: Эта функция доступна только в Zend Engine 2 (PHP >= 5.0.0).
