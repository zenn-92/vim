RecursiveIteratorIterator::beginIteration

(PHP 5 >= 5.1.0)

RecursiveIteratorIterator::beginIteration — Начало навигации

Описание

public void RecursiveIteratorIterator::beginIteration ( void )

Вызывается, чтобы начать навигацию (после первого вызова RecursiveIteratorIterator::rewind()).

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
