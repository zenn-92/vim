XMLReader::moveToAttributeNs

(PHP 5 >= 5.1.0)

XMLReader::moveToAttributeNs — Переместить курсор к именованному атрибуту

Описание

bool XMLReader::moveToAttributeNs ( string $localName , string $namespaceURI )

Позиционирует курсор на именованном атрибуте в указанном пространстве имен.

Список параметров

localName

Локальное имя.

namespaceURI

URI пространства имён.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

  • XMLReader::moveToAttributeNo() - Переместить курсор на атрибут по индексу
  • XMLReader::moveToFirstAttribute() - Переместить позицию курсора на первый атрибут
