printer_start_page

(PECL printer SVN)

printer_start_page — Начинает новую страницу

Описание

bool printer_start_page ( resource $printer_handle )

Эта функция создает новую страницу в активном документе. Для примера смотрите функцию printer_start_doc().

Список параметров

printer_handle

printer_handle должен быть действующим дескриптором устройства.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
