Imagick::getResource

(PECL imagick 2.0.0)

Imagick::getResource — Возвращает размер используемой памяти указанного ресурса

Описание

int Imagick::getResource ( int $type )

Возвращает размер используемой памяти указанного ресурса в мегабайтах.

Список параметров

type

Смотрите список констант типов ресурсов.

Возвращаемые значения

Возвращает размер используемой памяти указанного ресурса в мегабайтах.

Ошибки

Вызывает ImagickException при ошибке.
