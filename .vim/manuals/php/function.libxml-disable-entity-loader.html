libxml_disable_entity_loader

(PHP 5 >= 5.2.11)

libxml_disable_entity_loader — Отключение возможности загрузки сущностей из внешних источников

Описание

bool libxml_disable_entity_loader ([ bool $disable = true ] )

Отключение/включение возможности загружать внешние сущности.

Список параметров

disable

Отключение (TRUE) или включение (FALSE) libxml расширений (также как в DOM, XMLWriter и XMLReader) для загрузки внешних сущностей.

Возвращаемые значения

Возвращает предыдущее значение.
