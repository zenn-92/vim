SplFixedArray::offsetSet

(PHP 5 >= 5.3.0)

SplFixedArray::offsetSet — Устанавливает новое значение по заданному индексу

Описание

public void SplFixedArray::offsetSet ( int $index , mixed $newval )

По индексу index устанавливает значение newval.

Список параметров

index

Индекс, по которому устанавливается значение.

newval

Новое значение для индекса index.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Ошибки

Выбрасывает исключение RuntimeException, когда index превышает заданный размер массива или когда index нельзя распознать как целое число.
