SplObjectStorage::valid

(PHP 5 >= 5.1.0)

SplObjectStorage::valid — Определяет, допустимо ли текущее значение итератора

Описание

public bool SplObjectStorage::valid ( void )

Определяет, допустимо ли текущее значение итератора, то есть возможна ли работа с этим объектом.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE, если текущий объект допустим, и FALSE в противном случае.

Примеры

Пример #1 Пример использования SplObjectStorage::valid()

<?php
$s = new SplObjectStorage();

$o1 = new StdClass;
$o2 = new StdClass;

$s->attach($o1, "d1");
$s->attach($o2, "d2");

$s->rewind();
while($s->valid()) {
    echo $s->key()."\n";
    $s->next();
}
?>

Результатом выполнения данного примера будет что-то подобное:
