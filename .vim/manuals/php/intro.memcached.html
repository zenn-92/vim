Введение

» Memcached - это высокопроизводительная и распределенная система кеширования любых объектов в памяти, предназначенная в первую очередь для ускорения динамических веб-приложений за счет снижения нагрузки на базу данных.

Это расширение, используя библиотеку libmemcached, предоставляет API для связи с серверами memcached. Оно также предоставляет обработчик для хранения сессий в (memcached).

Информацию о libmemcached можно найти здесь: » http://libmemcached.org/libMemcached.html.
