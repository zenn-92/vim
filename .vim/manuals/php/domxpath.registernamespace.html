DOMXPath::registerNamespace

(PHP 5)

DOMXPath::registerNamespace — Ассоциирует пространство имен с объектом DOMXPath

Описание

public bool DOMXPath::registerNamespace ( string $prefix , string $namespaceURI )

Ассоциирует URI пространства имен namespaceURI и префикс prefix с объектом DOMXPath.

Список параметров

prefix

Префикс.

namespaceURI

URI пространства имен.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
