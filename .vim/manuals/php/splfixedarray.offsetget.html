SplFixedArray::offsetGet

(PHP 5 >= 5.3.0)

SplFixedArray::offsetGet — Возвращает значение по указанному индексу

Описание

public mixed SplFixedArray::offsetGet ( int $index )

Возвращает значение по индексу index.

Список параметров

index

Индекс.

Возвращаемые значения

Значение по индексу index.

Ошибки

Выбрасывает исключение RuntimeException, когда параметр index выходит за рамки заданного размера массива или когда index нельзя распознать как целое число.
