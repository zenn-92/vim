sqlite_num_fields

SQLiteResult::numFields

SQLiteUnbuffered::numFields

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_num_fields -- SQLiteResult::numFields -- SQLiteUnbuffered::numFields — Возвращает количество полей в результате запроса

Описание

int sqlite_num_fields ( resource $result )

Объектно-ориентированный стиль (метод):

int SQLiteResult::numFields ( void )
int SQLiteUnbuffered::numFields ( void )

Возвращает количество полей в результате запроса result.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

Возвращаемые значения

Возвращает количество полей в виде целого числа.

Смотрите также

  • sqlite_changes() - Возвращает количество записей, которые были изменены последним SQL-запросом
  • sqlite_num_rows() - Возвращает количество записей в результате запроса
