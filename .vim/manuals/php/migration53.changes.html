Что нового в PHP 5.3.x ?

Большинство изменений в PHP 5.3.x никак не повлияет на существующий код. Есть несколько несовместимостей и новых возможностей, которые должны быть учтены, и код должен быть протестирован перед тем как переключиться на новую версию PHP на рабочих серверах.

Для систем, которые обновились с более старой версии PHP, соответствующую документацию можно получить по адресам:

  • Переход с PHP 5.1.x на PHP 5.2.x.
  • Переход с PHP 5.0.x на PHP 5.1.x.
  • Переход с PHP 4 на PHP 5.0.x.
