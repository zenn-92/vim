imagecreate

(PHP 4, PHP 5)

imagecreate — Создание нового палитрового изображения

Описание

resource imagecreate ( int $width , int $height )

imagecreate() возвращает идентификатор изображения, представляющего из себя пустое изображение заданного размера.

Мы рекомендуем использовать функцию imagecreatetruecolor().

Список параметров

width

Ширина изображения.

height

Высота изображения.

Возвращаемые значения

Возвращает ресурс изображения в случае успеха, или FALSE в случае ошибки.

Примеры

Пример #1 Создание нового GD потока изображения и вывод изображения.

<?php
header("Content-Type: image/png");
$im = @imagecreate(110, 20)
    or die("Невозможно создать поток изображения");
$background_color = imagecolorallocate($im, 0, 0, 0);
$text_color = imagecolorallocate($im, 233, 14, 91);
imagestring($im, 1, 5, 5,  "Простая Текстовая Строка", $text_color);
imagepng($im);
imagedestroy($im);
?>
