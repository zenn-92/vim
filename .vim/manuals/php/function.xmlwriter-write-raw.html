XMLWriter::writeRaw

xmlwriter_write_raw

(PHP 5 >= 5.2.0, PECL xmlwriter >= 2.0.4)

XMLWriter::writeRaw -- xmlwriter_write_raw — Пишет необработанный XML-текст

Описание

Объектно-ориентированный стиль

bool XMLWriter::writeRaw ( string $content )

Процедурный стиль

bool xmlwriter_write_raw ( resource $xmlwriter , string $content )

Пишет необработанный XML-текст.

Список параметров

xmlwriter

Только для процедурных вызовов. Изменяемый resource XMLWriter. Этот ресурс может быть получен из xmlwriter_open_uri() или xmlwriter_open_memory().

content

Текстовая строка для записи.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
