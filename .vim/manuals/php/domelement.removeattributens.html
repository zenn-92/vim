DOMElement::removeAttributeNS

(PHP 5)

DOMElement::removeAttributeNS — Удаляет атрибут

Описание

public bool DOMElement::removeAttributeNS ( string $namespaceURI , string $localName )

Удаляет атрибут с пространством имен namespaceURI и локальным именем localName из элемента.

Список параметров

namespaceURI

URI пространства имен.

localName

Локальное имя.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Ошибки

DOM_NO_MODIFICATION_ALLOWED_ERR

Возникает, если узел доступен только для чтения.

Смотрите также

  • DOMElement::hasAttributeNS() - Проверяет, существует ли заданный атрибут
  • DOMElement::getAttributeNS() - Возвращает значение атрибута
  • DOMElement::setAttributeNS() - Добавляет новый атрибут
