SplObjectStorage::serialize

(PHP 5 >= 5.2.2)

SplObjectStorage::serialize — Сериализует контейнер

Описание

public string SplObjectStorage::serialize ( void )

Возвращает строковое представление контейнера.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Строковое представление контейнера.

Примеры

Пример #1 Пример использования SplObjectStorage::serialize()

<?php
$s = new SplObjectStorage;
$o = new StdClass;
$s[$o] = "data";

echo $s->serialize()."\n";
?>

Результатом выполнения данного примера будет что-то подобное:

x:i:1;O:8:"stdClass":0:{},s:4:"data";;m:a:0:{}
