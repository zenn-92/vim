mb_decode_mimeheader

(PHP 4 >= 4.0.6, PHP 5)

mb_decode_mimeheader — Декодирует строку в MIME-заголовке

Описание

string mb_decode_mimeheader ( string $str )

Декодирует закодированную строку string str в MIME-заголовке.

Список параметров

str

Строка string для декодирования.

Возвращаемые значения

Декодированная строка string во внутренней кодировке скрипта.
