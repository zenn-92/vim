Интерфейс JsonSerializable

(PHP 5 >= 5.4.0)

Введение

Объекты, реализующие интерфейс JsonSerializable, могут адаптировать свое JSON-представление когда кодируются с помощью json_encode().

Обзор интерфейсов

JsonSerializable {
/* Методы */
abstract public mixed jsonSerialize ( void )
}

Содержание

  • JsonSerializable::jsonSerialize — Задает данные, которые должны быть сериализованы в JSON
