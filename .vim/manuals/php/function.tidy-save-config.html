tidy_save_config

(PECL tidy >= 0.5.2)

tidy_save_config — Сохранить текущие настройки в файл

Описание

bool tidy_save_config ( string $filename )

Сохраняет текущие настройки в указанный файл. Записываются только те значения, которые отличаются от значений по умолчанию.

Список параметров

filename

Путь к файлу конфигурации.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примечания

Замечание: Эта функция доступна только в Tidy 1.0. С приходом Tidy 2.0 она стала ненужной и была удалена.
