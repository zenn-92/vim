PDOStatement::fetchObject

(PHP 5 >= 5.1.0, PECL pdo >= 0.2.4)

PDOStatement::fetchObject — Извлекает следующую строку и возвращает ее в виде объекта

Описание

mixed PDOStatement::fetchObject ([ string $class_name = "stdClass" [, array $ctor_args ]] )

Извлекает следующую строку и возвращает ее в виде объекта. Этот метод является альтернативой вызову PDOStatement::fetch() с параметром PDO::FETCH_CLASS или PDO::FETCH_OBJ.

Список параметров

class_name

Имя класса создаваемого объекта.

ctor_args

Элементы этого массива будут переданы в конструктор класса.

Возвращаемые значения

Возвращает новый объект указанного класса, имена свойств которого соответствуют именам столбцов результирующего набора или FALSE в случае возникновения ошибки.
