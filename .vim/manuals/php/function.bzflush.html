bzflush

(PHP 4 >= 4.0.4, PHP 5)

bzflush — Форсирует запись всех буферизированных данных

Описание

int bzflush ( resource $bz )

Форсирует запись всех буферизированных bzip2 данных в файл, на который ссылается указатель bz.

Список параметров

bz

Указатель на файл. Должен быть корректным и указывать на файл, успешно открытый bzopen().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

  • bzwrite() - Бинарно-безопасная запись bzip2 файла
