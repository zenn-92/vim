trader_cdldoji

(PECL trader >= 0.2.0)

trader_cdldoji — Doji

Описание

array trader_cdldoji ( array $open , array $high , array $low , array $close )

Список параметров

open

Цена открытия, массив реальных значений.

high

Высокая цена, массив реальных значений.

low

Низкая цена, массив реальных значений.

close

Цена закрытия, массив реальных значений.

Возвращаемые значения

Returns an array with calculated data or false on failure.
