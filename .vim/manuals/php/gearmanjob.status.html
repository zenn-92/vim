GearmanJob::status

(PECL gearman <= 0.5.0)

GearmanJob::status — Отправка статуса задания (устаревший метод)

Описание

public bool GearmanJob::status ( int $numerator , int $denominator )

Посылает на сервер задач и всем слушающим клиентам статус текущей работы. С помощью этого метода можно узнать, какой процент задания выполнен.

Замечание:

Этот метод был заменен на GearmanJob::sendStatus() в версии 0.6.0 расширения Gearman.

Список параметров

numerator

Доля выполненной работы.

denominator

Количество всей работы.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Смотрите также

  • GearmanClient::jobStatus() - Получение статуса выполнения фонового задания
  • GearmanTask::taskDenominator() - Получение знаменателя в дроби, отвечающей за процент выполненной работы
  • GearmanTask::taskNumerator() - Получение числителя дроби, отвечающей за процент выполненной работы
