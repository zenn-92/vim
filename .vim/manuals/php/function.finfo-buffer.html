finfo_buffer

finfo::buffer

(PHP 5 >= 5.3.0, PECL fileinfo >= 0.1.0)

finfo_buffer -- finfo::buffer — Возвращает информацию о строке буфера

Описание

Процедурный стиль

string finfo_buffer ( resource $finfo , string $string = NULL [, int $options = FILEINFO_NONE [, resource $context = NULL ]] )

Объектно-ориентированный стиль

string finfo::buffer ( string $string = NULL [, int $options = FILEINFO_NONE [, resource $context = NULL ]] )

Эта функция используется для получения информации о бинарных данных в строке.

Список параметров

finfo

Ресурс Fileinfo, полученный функцией finfo_open().

string

Содержимое проверяемого файла.

options

Одна или операция логическое "ИЛИ" из нескольких Fileinfo констант.

context

Возвращаемые значения

Возвращает текстовое описание для аргумента string или FALSE, в случае ошибки.

Примеры

Пример #1 A finfo_buffer() example

<?php
$finfo = new finfo(FILEINFO_MIME);
echo $finfo->buffer($_POST["script"]) . "\n";
?>

Результатом выполнения данного примера будет что-то подобное:

application/x-sh; charset=us-ascii
