ArrayIterator::uksort

(PHP 5 >= 5.2.0)

ArrayIterator::uksort — Сортировка, определенная пользователем

Описание

public void ArrayIterator::uksort ( string $cmp_function )

Сортирует записи в массиве по ключам, используя функцию сортировки, определенную пользователем.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

cmp_function

Функция сравнения элементов, используемая для сортировки.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
