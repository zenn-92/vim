Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Параметры конфигурации APD
Имя По умолчанию Меняемо Список изменений
apd.dumpdir NULL PHP_INI_ALL  
apd.statement_tracing "0" PHP_INI_ALL Доступно начиная с apd 0.9.
Для подробного описания констант PHP_INI_*, обратитесь к разделу Где могут быть установлены параметры конфигурации.

Краткое разъяснение конфигурационных директив.

apd.dumpdir string

Задает каталог, в который APD записывает дамп-файлы профилей. Можно указать абсолютный путь или относительный.

Можно указать другой каталог в качестве аргумента apd_set_pprof_trace().

apd.statement_tracing boolean

Указывает будет ли или нет проводиться построковая трассировка. Включение этого параметра (если он равен 1) будет влиять на производительность приложения.
