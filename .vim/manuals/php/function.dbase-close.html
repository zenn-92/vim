dbase_close

(PHP 4, PHP 5)

dbase_close — Закрывает базу данных

Описание

bool dbase_close ( int $dbase_identifier )

Закрывает базу данных согласно полученному идентификатору.

Список параметров

dbase_identifier

Идентификатор базы данных, возвращаемый функцией dbase_open() или dbase_create().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Закрытие файла базы данных dBase

<?php

// открыть БД в режиме чтения
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
  // получить некоторые данные
  
  dbase_close($db);
}

?>
