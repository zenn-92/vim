streamWrapper::stream_close

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::stream_close — Закрывает ресурс

Описание

public void streamWrapper::stream_close ( void )

Этот метод вызывается в результате вызова fclose().

Все ресурсы, выделенные или заблокированные оберткой, должны быть освобождены.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
