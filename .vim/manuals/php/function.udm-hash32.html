udm_hash32

(PHP 4 >= 4.3.3, PHP 5 <= 5.0.5, PECL mnogosearch >= 1.0.0)

udm_hash32 — Возвращает контрольную сумму Hash32 данной строки

Описание

int udm_hash32 ( resource $agent , string $str )

udm_hash32() принимает строку str и возвращает практически уникальную 32-битную хэш-сумму этой строки.

Список параметров

agent

Ссылка на идентификатор агента, полученная после вызова udm_alloc_agent().

str

Входная строка.

Возвращаемые значения

Возвращает 32-битную хэш-сумму.
