mysqli_get_metadata

(PHP 5 < 5.4.0)

mysqli_get_metadata — Синоним для mysqli_stmt_result_metadata()

Описание

Функция является синонимом mysqli_stmt_result_metadata().

Внимание

Данная функция была помечена УСТАРЕВШЕЙ начиная с версии PHP 5.3.0 и была УДАЛЕНА в версии PHP 5.4.0.
