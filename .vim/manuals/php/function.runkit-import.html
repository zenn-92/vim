runkit_import

(PECL runkit >= 0.7.0)

runkit_import — Обрабатывает PHP файл, импортируя функции и классы, перезаписывая при необходимости.

Описание

bool runkit_import ( string $filename [, int $flags = RUNKIT_IMPORT_CLASS_METHODS ] )

Функция похожа на include, за исключением того, что весь код вне функций и объявлений классов игнорируется. В зависимости от значения параметра flags существующие функции и классы будут автоматически заменены новыми реализациями.

Список параметров

filename

Имя файла, из которого будут импортированы классы и функции

flags

Значение побитового ИЛИ семейства констант RUNKIT_IMPORT_*/

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
