session_is_registered

(PHP 4, PHP 5 < 5.4.0)

session_is_registered — Определяет, зарегистрирована ли глобальная переменная в сессии

Описание

bool session_is_registered ( string $name )

Определяет, зарегистрирована ли глобальная переменная в сессии.

Внимание

Данная функция была помечена УСТАРЕВШЕЙ начиная с версии PHP 5.3.0 и была УДАЛЕНА в версии PHP 5.4.0.

Список параметров

name

Имя переменной.

Возвращаемые значения

session_is_registered() возвращает TRUE если глобальная переменная с именем name зарегистрирована в текущей сессии, либо FALSE в противном случае.

Примечания

Замечание:

Если применяется $_SESSION (или $HTTP_SESSION_VARS для PHP 4.0.6 и более ранних), следует использовать isset() для проверки регистрации переменных в $_SESSION.

Предостережение

Если используется $_SESSION (или $HTTP_SESSION_VARS), не следует использовать session_register(), session_is_registered() и session_unregister().
