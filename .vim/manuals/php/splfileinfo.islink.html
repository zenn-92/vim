SplFileInfo::isLink

(PHP 5 >= 5.1.2)

SplFileInfo::isLink — Сообщает, является ли файл ссылкой

Описание

public bool SplFileInfo::isLink ( void )

Используйте этот метод для проверки того, является ли файл, на который ссылается объект SplFileInfo, ссылкой.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE, если файл является ссылкой; в противном случае - FALSE.

Примеры

Пример #1 Пример использования SplFileInfo::isLink()

<?php
$info = new SplFileInfo('/path/to/symlink');
if ($info->isLink()) {
    echo 'Реальный путь: '.$info->getRealPath();
}
?>
