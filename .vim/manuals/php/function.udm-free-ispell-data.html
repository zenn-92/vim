udm_free_ispell_data

(PHP 4 >= 4.0.5, PHP 5 <= 5.0.5, PECL mnogosearch >= 1.0.0)

udm_free_ispell_data — Освобождает память, выделенную для ispell-данных

Описание

bool udm_free_ispell_data ( int $agent )

Освобождает память, выделенную для ispell-данных.

Список параметров

agent

Ссылка на идентификатор агента, полученная после вызова udm_alloc_agent().

Возвращаемые значения

udm_free_ispell_data() всегда возвращает TRUE.

Примечания

Замечание:

Эта функция поддерживается, начиная с mnoGoSearch версии 3.1.12, и ничего не делает в предыдущих версиях.
