date_sun_info

(PHP 5 >= 5.1.2)

date_sun_info — Возвращает массив с информацией о закате/рассвете и начале/окончании сумерек

Описание

array date_sun_info ( int $time , float $latitude , float $longitude )

Список параметров

time

Временная метка.

latitude

Широта в градусах.

longitude

Долгота в градусах.

Возвращаемые значения

Возвращает массив при успешном завершении или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования date_sun_info()

<?php
$sun_info = date_sun_info(strtotime("2006-12-12"), 31.7667, 35.2333);
foreach ($sun_info as $key => $val) {
    echo "$key: " . date("H:i:s", $val) . "\n";
}
?>
