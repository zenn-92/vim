gnupg_export

(PECL gnupg >= 0.1)

gnupg_export — Экспортирует ключ

Описание

string gnupg_export ( resource $identifier , string $fingerprint )

Экспортирует ключ fingerprint.

Список параметров

identifier

Идентификатор gnupg, полученный из gnupg_init() или gnupg.

fingerprint

Отпечаток ключа.

Возвращаемые значения

В случае успеха, эта функция возвращает ключ. В случае ошибки, эта функция возвращает FALSE.

Примеры

Пример #1 Процедурный пример использования gnupg_export()

<?php
$res = gnupg_init();
$export = gnupg_export($res, "8660281B6051D071D94B5B230549F9DC851566DC");
echo $export;
?>

Пример #2 Объектно-ориентированный пример использования gnupg_export()

<?php
$gpg = new gnupg();
$export = $gpg -> export("8660281B6051D071D94B5B230549F9DC851566DC");
?>
