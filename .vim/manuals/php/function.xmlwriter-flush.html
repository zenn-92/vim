XMLWriter::flush

xmlwriter_flush

(PHP 5 >= 5.1.2, PECL xmlwriter >= 1.0.0)

XMLWriter::flush -- xmlwriter_flush — Сбросить текущий буфер

Описание

Объектно-ориентированный стиль

mixed XMLWriter::flush ([ bool $empty = true ] )

Процедурный стиль

mixed xmlwriter_flush ( resource $xmlwriter [, bool $empty = true ] )

Сбрасывает текущий буфер.

Список параметров

xmlwriter

Только для процедурных вызовов. Изменяемый resource XMLWriter. Этот ресурс может быть получен из xmlwriter_open_uri() или xmlwriter_open_memory().

empty

Освободить ли буфер или нет. По умолчанию TRUE.

Возвращаемые значения

Если XML создается в памяти, то эта функция вернет сгенерированный буфер XML, иначе при использовании URI эта функция запишет буфер и возвратит количество записанных байт.
