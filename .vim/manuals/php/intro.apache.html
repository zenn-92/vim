Введение

Эти функции доступны только в том случае, если PHP работает в качестве модуля Apache.

Замечание: Начиная с PHP 4.3.2, переменная PATH_TRANSLATED больше не создаётся под SAPI Apache 2, в отличие от Apache 1, где она создаётся с тем же значением, что и SCRIPT_FILENAME (если другое значение не было установлено самим Apache). Это изменение было сделано для соответствия спецификации CGI, которая гласит, что PATH_TRANSLATED должна быть объявлена только тогда, когда определена переменная PATH_INFO. Пользователи Apache 2 могут использовать опцию AcceptPathInfo = On в httpd.conf для объявления переменной PATH_INFO.
