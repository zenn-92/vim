closelog

(PHP 4, PHP 5)

closelog — Завершает соединение с системным журналом

Описание

bool closelog ( void )

Функция closelog() закрывает дескриптор, используемый для записи в системный журнал. Использование closelog() не является обязательным.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Смотрите также

  • syslog() - Генерирует сообщение для системного журнала
  • openlog() - Открывает соединение к системному журналу
