DirectoryIterator::getOwner

(PHP 5)

DirectoryIterator::getOwner — Возвращает индентификатор владельца текущего элемента DirectoryIterator

Описание

public int DirectoryIterator::getOwner ( void )

Возвращает идентификатор владельца текущего элемента DirectoryIterator в числовом формате.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Владелец файла, в числовом формате.

Примеры

Пример #1 Пример использования DirectoryIterator::getOwner()

Этот пример показывает владельца директории, содержащей выполняемый скрипт.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
print_r(posix_getpwuid($iterator->getOwner()));
?>

Результатом выполнения данного примера будет что-то подобное:

Array
(
    [name] => tom
    [passwd] => x
    [uid] => 501
    [gid] => 42
    [gecos] => Tom Cat
    [dir] => /home/tom
    [shell] => /bin/bash
)
