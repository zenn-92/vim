SplObjectStorage::attach

(PHP 5 >= 5.1.0)

SplObjectStorage::attach — Добавляет объект в контейнер

Описание

public void SplObjectStorage::attach ( object $object [, mixed $data = NULL ] )

Добавляет объект object в контейнер и дополнительно может ассоциировать этот объект с какими-то данными.

Список параметров

object

Добавляемый объект object.

data

Данные, с которыми требуется ассоциировать объект object.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования SplObjectStorage::attach()

<?php
$o1 = new StdClass;
$o2 = new StdClass;
$s = new SplObjectStorage();
$s->attach($o1); // то же, что и $s[$o1] = NULL;
$s->attach($o2, "hello"); // то же, что и $s[$o2] = "hello";

var_dump($s[$o1]);
var_dump($s[$o2]);

?>

Результатом выполнения данного примера будет что-то подобное:

NULL
string(5) "hello"
