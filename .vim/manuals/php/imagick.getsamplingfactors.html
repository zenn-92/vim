Imagick::getSamplingFactors

(PECL imagick 2.0.0)

Imagick::getSamplingFactors — Возвращает горизонтальный и вертикальный фактор выборки

Описание

array Imagick::getSamplingFactors ( void )

Возвращает горизонтальный и вертикальный фактор выборки.

Возвращаемые значения

Возвращает ассоциативный массив с горизонтальным и вертикальным фактором выборки изображения.

Ошибки

Вызывает ImagickException при ошибке.
