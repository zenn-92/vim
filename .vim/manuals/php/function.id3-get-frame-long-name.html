id3_get_frame_long_name

(PECL id3 >= 0.2)

id3_get_frame_long_name — Возвращает длинное имя ID3v2 кадра

Описание

string id3_get_frame_long_name ( string $frameId )

id3_get_frame_long_name() возвращает длинное имя ID3v2 кадра.

Список параметров

frameId

кадр ID3v2

Возвращаемые значения

Возвращает длинное имя или FALSE в случае ошибки.

Примеры

Пример #1 Пример использования id3_get_frame_long_name()

<?php
$longName = id3_get_frame_long_name("TOLY");
echo $longName;
?>

Результат выполнения данного примера:

Original lyricist(s)/text writer(s)
