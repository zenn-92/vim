get_cfg_var

(PHP 4, PHP 5)

get_cfg_var — Извлекает значение настройки конфигурации PHP

Описание

string get_cfg_var ( string $option )

Извлекает значение настройки конфигурации PHP option.

Функция не вернет никакой информации, если PHP скомпилирован из конфигурационного файла Apache.

Удостовериться в том, что система использует файл конфигурации, попробуйте получить значение настройки конфигурации cfg_file_path. Если эта настройка доступна, это значит, что файл конфигурации используется.

Список параметров

option

Имя настройки конфигурации.

Возвращаемые значения

Возвращает текущее значение настройки конфигурации PHP option, или FALSE в случае ошибки.

Список изменений

Версия Описание
5.3.0 get_cfg_var() изменена, чтобы иметь возможность возвращать ini настройки содержащие массивы.
