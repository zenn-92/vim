FilesystemIterator::next

(PHP 5 >= 5.3.0)

FilesystemIterator::next — Перемещение указателя на следующий файл

Описание

public void FilesystemIterator::next ( void )

Переход к следующему файлу.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования FilesystemIterator::next()

Вывод списка содержимого директории, используя цикл while.

<?php
$iterator = new FilesystemIterator(dirname(__FILE__));
while($iterator->valid()) {
    echo $iterator->getFilename() . "\n";
    $iterator->next();
}
?>

Результатом выполнения данного примера будет что-то подобное:

apple.jpg
banana.jpg
example.php
