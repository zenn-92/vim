Введение

Этот модуль позволяет прозрачно читать и писать сжатые файлы gzip (.gz) стандартным методом наподобие функций файловой системы, которые работают с обычными несжатыми файлами (исключение составляют только сокеты).

Замечание:

В версии 4.0.4 появилась новая fopen-обертка для .gz-файлов, и вы можете использовать специальный URL zlib: для прозрачного доступа к сжатым файлам, используя обычные функции f*() доступа к файлам. Для этого необходимо добавить префикс zlib: в начало пути к файлу при вызове fopen(). Эта возможность требует наличия библиотеки С, в которой есть функция fopencookie(). До настоящего момента только библиотека GNU libc предоставляет эту возможность.

В версии 4.3.0, префикс zlib: изменился на compress.zlib:// для предотвращения многозначности с именами файлов, содержащих ':' в имени. Функция fopencookie() больше не требуется. Больше информации доступно в разделе о zlib://.
