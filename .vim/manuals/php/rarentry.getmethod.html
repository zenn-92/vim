RarEntry::getMethod

(PECL rar >= 0.1)

RarEntry::getMethod — Возвращает метод компрессии элемента

Описание

public int RarEntry::getMethod ( void )

RarEntry::getMethod() возвращает число соответствующее методу компрессии, использовавшемуся при создании текущего элемента архива.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Вовзращает номер метода или FALSE в случае ошибки.

Примеры

Пример #1 Пример с RarEntry::getMethod()

<?php

$rar_file = rar_open('example.rar') or die("Failed to open Rar archive");

$entry = rar_entry_get($rar_file, 'Dir/file.txt') or die("Failed to find such entry");

echo "Method number: " . $entry->getMethod();

?>
