Новые параметры

К некоторым функциям были добавлены новые необязательные параметры в PHP 5.3:

Ядро PHP:

  • clearstatcache() - Добавлен параметр clear_realpath_cache и filename.
  • copy() - Добавлен параметр контекста потока context.
  • fgetcsv() - Добавлен параметр escape.
  • ini_get_all() - Добавлен параметр details.
  • nl2br() - Добавлен параметр is_xhtml.
  • parse_ini_file() - Добавлен параметр scanner_mode.
  • round() - Добавлен параметр mode.
  • stream_context_create() - Добавлен параметр params.
  • strstr() и stristr() - Добавлен параметр before_needle.

json:

  • json_encode() - Добавлен параметр options.
  • json_decode() - Добавлен параметр depth.

Потоки:

  • stream_select(), stream_set_blocking(), stream_set_timeout() и stream_set_write_buffer() теперь работают с пользовательскими обертками потоков.

sybase_ct:

  • sybase_connect() - Добавлен параметр new.

Новые параметры методов в PHP 5.3.0:

Ядро PHP:

  • Exception::__construct() - Добавлен параметр previous.
