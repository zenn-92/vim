Константы Memcache
Имя Описание
MEMCACHE_COMPRESSED (integer) Используется для включения сжатия данных "на лету" для следующих методов: Memcache::set(), Memcache::add() и Memcache::replace().
MEMCACHE_HAVE_SESSION (integer) 1, если соединение с Memcache установлено, иначе 0.
MEMCACHE_USER1 (integer) Используется для включения пользовательского флага приложения для Memcache::set(), Memcache::add() и Memcache::replace().
MEMCACHE_USER2 (integer) Используется для включения пользовательского флага приложения для Memcache::set(), Memcache::add() и Memcache::replace().
MEMCACHE_USER3 (integer) Используется для включения пользовательского флага приложения для Memcache::set(), Memcache::add() и Memcache::replace().
MEMCACHE_USER4 (integer) Используется для включения пользовательского флага приложения для Memcache::set(), Memcache::add() и Memcache::replace().
