connection_aborted

(PHP 4, PHP 5)

connection_aborted — Проверяет подключение клиента

Описание

int connection_aborted ( void )

Проверяет подключен ли клиент.

Возвращаемые значения

Возвращает 1, если клиент отключен, 0 в другом случае.

Смотрите также

  • connection_status() - Возвращает статус соединения в битах
  • ignore_user_abort() - Устанавливает, необходимо ли прерывать работу скрипта при отключении клиента
  • См. Управление подключениями для получения полного описания управления подключениями в PHP.
