mysqli::thread_safe

mysqli_thread_safe

(PHP 5)

mysqli::thread_safe -- mysqli_thread_safe — Показывает, безопасна ли работа с процессами

Описание

Процедурный стиль

bool mysqli_thread_safe ( void )

Показывает, клиентская библиотека скомпилирована как потокобезопасная или нет.

Возвращаемые значения

TRUE, если клиентская библиотека потокобезопасна, в противном случае FALSE.
