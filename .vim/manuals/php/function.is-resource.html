is_resource

(PHP 4, PHP 5)

is_resource — Проверяет, является ли переменная ресурсом

Описание

bool is_resource ( mixed $var )

Проверяет, является ли данная переменная ресурсом.

Список параметров

var

Проверяемая переменная.

Возвращаемые значения

Возвращает TRUE, если var является ресурсом, или FALSE в противном случае.

Примеры

Пример #1 Пример использования is_resource()

<?php

$db_link = @mysql_connect('localhost', 'mysql_user', 'mysql_pass');
if (!is_resource($db_link)) {
    die('Ошибка соединения : ' . mysql_error());
}

?>
