SplObjectStorage::contains

(PHP 5 >= 5.1.0)

SplObjectStorage::contains — Проверяет, содержит ли контейнер заданный объект

Описание

public bool SplObjectStorage::contains ( object $object )

Проверяет, содержит ли контейнер заданный объект object.

Список параметров

object

Искомый объект object.

Возвращаемые значения

Возвращает TRUE, если object находится в контейнере, и FALSE в противном случае.

Примеры

Пример #1 Пример использования SplObjectStorage::contains()

<?php
$o1 = new StdClass;
$o2 = new StdClass;

$s = new SplObjectStorage();

$s[$o1] = "hello";
var_dump($s->contains($o1));
var_dump($s->contains($o2));
?>

Результатом выполнения данного примера будет что-то подобное:

bool(true)
bool(false)
