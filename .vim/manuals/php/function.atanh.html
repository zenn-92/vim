atanh

(PHP 4 >= 4.1.0, PHP 5)

atanh — Гиперболический арктангенс

Описание

float atanh ( float $arg )

Возвращает гиперболический арктангенс arg, т.е. значение, чей гиперболический тангенс равен arg.

Список параметров

arg

Входное значение

Возвращаемые значения

Гиперболический арктангенс arg

Список изменений

Версия Описание
5.3.0 Функция стала доступна для всех платформ
