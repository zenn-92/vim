SplFileInfo::getPath

(PHP 5 >= 5.1.2)

SplFileInfo::getPath — Получение пути без имени файла

Описание

public string SplFileInfo::getPath ( void )

Возвращает путь к файлу, исключая само имя файла и замыкающий слэш.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает путь к файлу.

Примеры

Пример #1 Пример использования SplFileInfo::getPath()

<?php
$info = new SplFileInfo('/usr/bin/php');
var_dump($info->getPath());


$info = new SplFileInfo('/usr/');
var_dump($info->getPath());?>

Результатом выполнения данного примера будет что-то подобное:

string(8) "/usr/bin"
string(4) "/usr"
