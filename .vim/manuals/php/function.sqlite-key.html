sqlite_key

SQLiteResult::key

(PHP 5 >= 5.1.0 and < 5.4.0)

sqlite_key -- SQLiteResult::key — Возвращает текущую позицию указателя

Описание

int sqlite_key ( resource $result )

Объектно-ориентированный стиль (метод):

int SQLiteResult::key ( void )

sqlite_key() возвращает текущую позицию указателя в буферизованном результирующем наборе result.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Возвращаемые значения

Возвращает текущую позицию указателя в буферизованном результирующем наборе result.

Список изменений

Версия Описание
5.0.4 До версии PHP 5.0.4 sqlite_key() можно было вызывать только в качестве метода объекта SQLiteResult, процедурная альтернатива не существовала.

  • sqlite_rewind() - Переходит к первой записи
