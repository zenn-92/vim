gmp_divexact

(PHP 4 >= 4.0.4, PHP 5)

gmp_divexact — Деление чисел без остатка

Описание

resource gmp_divexact ( resource $n , resource $d )

Делит число n на d, используя быстрый алгоритм деления без остатка. Функция выдает корректный результат, только если заведомо известно, что число d делит n нацело.

Список параметров

n

Делимое.

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

d

Делитель числа a.

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

Возвращаемые значения

Число GMP.

Примеры

Пример #1 Пример использования gmp_divexact()

<?php
$div1 = gmp_divexact("10", "2");
echo gmp_strval($div1) . "\n";

$div2 = gmp_divexact("10", "3"); // некорректный результат
echo gmp_strval($div2) . "\n";
?>
