oci_client_version

(PHP 5.3.7, PECL OCI8 >= 1.4.6)

oci_client_version — Возвращает версию клиентской библиотеки

Описание

string oci_client_version ( void )

Возвращает номер версии в виде строки клиентской C-библиотеки Oracle, с которой связан PHP.

Список параметров

Отсутствуют.

Возвращаемые значения

Возвращает номер версии в виде строки.

Примеры

Пример #1 Пример использования oci_client_version()

<?php
    echo "Версия клиента: " . oci_client_version(); // Версия клиента: 11.2.0.2
?>

Примечания

Замечание:

Библиотеки Oracle версий до 10gR2 не имеют необходимого функционала, чтобы выдать номер версии. В этих случаях функция вернет строку "Unknown".
