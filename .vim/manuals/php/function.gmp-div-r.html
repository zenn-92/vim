gmp_div_r

(PHP 4 >= 4.0.4, PHP 5)

gmp_div_r — Остаток от деления чисел

Описание

resource gmp_div_r ( resource $n , resource $d [, int $round = GMP_ROUND_ZERO ] )

Вычисляет остаток от деления нацело числа n на d. Если число n отлично от нуля, результат будет иметь знак этого числа.

Список параметров

n

Делимое.

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

d

Делитель числа n.

Может быть как числом GMP, так и строкой, при условии, что ее значение можно конвертировать в число.

round

В документации к функции gmp_div_q() приведено описание аргумента round.

Возвращаемые значения

Остаток в виде GMP числа.

Примеры

Пример #1 Пример использования gmp_div_r()

<?php
     $div = gmp_div_r("105", "20");
     echo gmp_strval($div) . "\n";
     ?>

Результат выполнения данного примера:

     5
