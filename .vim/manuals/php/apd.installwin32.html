Сборка на Win32

Для сборки APD под Windows вам нужна рабочая скомпилированная PHP среда, как описано на http://php.net/ - в основом это требует наличия Microsoft Visual C++, win32build.zip, bison/flex и грамотного специалиста. Также необходимо убедиться в том, что adp.dsp имеет DOS конвертацию конца строк, если конвертация конца строк Unix, то Microsoft Visual C++ сообщит об этом.
