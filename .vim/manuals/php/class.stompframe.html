Класс StompFrame

(PECL stomp >= 0.1.0)

Введение

Представляет сообщение, которое было отправлено или получено от Stomp-совместимого брокера сообщений (Message Broker).

Обзор классов

StompFrame {
/* Свойства */
public $command ;
public $headers ;
public $body ;
/* Методы */
__construct ([ string $command [, array $headers [, string $body ]]] )
}

Свойства

command

Команда фрейма.

headers

Заголовки фрейма (array).

body

Тело фрейма.
