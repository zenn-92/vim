cyrus_bind

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_bind — Привязать обратные вызовы для Cyrus IMAP соединения

Описание

bool cyrus_bind ( resource $connection , array $callbacks )

Привязывает обратные вызовы для Cyrus IMAP соединения.

Список параметров

connection

Дескриптор соединения.

callbacks

Массив обратных вызовов.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
