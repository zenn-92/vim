XSLTProcessor::removeParameter

(PHP 5)

XSLTProcessor::removeParameter — Удаляет параметр

Описание

bool XSLTProcessor::removeParameter ( string $namespaceURI , string $localName )

Удаляет параметр, если он установлен. Это позволяет процессору использовать значение в таблице стилей, установленное по умолчанию.

Список параметров

namespaceURI

Пространство имён URI параметра XSLT.

localName

Локальное название XSLT параметра.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

  • XSLTProcessor::getParameter() - Возвращает значение параметра
