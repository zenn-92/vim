SplDoublyLinkedList::offsetUnset

(PHP 5 >= 5.3.0)

SplDoublyLinkedList::offsetUnset — Удаляет значение по указанному индексу $index

Описание

void SplDoublyLinkedList::offsetUnset ( mixed $index )

Удаляет значение по указанному индексу.

Список параметров

index

Индекс, для которого удаляется значение.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Ошибки

Бросает исключение OutOfRangeException, когда index выходит за границы, или когда index не может быть представлен в виде целого числа.
