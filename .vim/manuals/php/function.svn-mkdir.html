svn_mkdir

(PECL svn >= 0.4.0)

svn_mkdir — Создает директорию в рабочей копии или в репозитарии

Описание

bool svn_mkdir ( string $path [, string $log_message ] )

Создает директорию в рабочей копии или в репозитарии.

Список параметров

path

Путь к рабочей копии или к репозитарию.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Смотрите также

  • svn_add() - Добавляет элементы в список запланированных для добавления в рабочую копию
  • svn_copy()
