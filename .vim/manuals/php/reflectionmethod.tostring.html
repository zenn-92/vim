ReflectionMethod::__toString

(PHP 5)

ReflectionMethod::__toString — Возвращает строковое представление объекта Reflection method

Описание

public string ReflectionMethod::__toString ( void )

Возвращает строковое представление объекта Reflection method.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Строковое представление объекта ReflectionMethod.

Примеры

Пример #1 Пример использования ReflectionMethod::__toString()

<?php
class HelloWorld {

    public function sayHelloTo($name) {
        return 'Hello ' . $name;
    }

}

$reflectionMethod = new ReflectionMethod(new HelloWorld(), 'sayHelloTo');
echo $reflectionMethod;
?>
