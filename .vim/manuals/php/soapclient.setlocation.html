SoapClient::__setLocation

(PHP 5 >= 5.0.1)

SoapClient::__setLocation — Устанавливает адрес используемой веб-службы

Описание

public string SoapClient::__setLocation ([ string $new_location ] )

Устанавливает URL, затрагивающий последующие SOAP-запросы. Эквивалентна настройке location, указываемой в конструкторе SoapClient.

Замечание:

Вызов метода необязателен. По умолчанию SoapClient использует адрес из WSDL-файла.

Список параметров

new_location

Новый URL.

Возвращаемые значения

Старый URL.

Примеры

Пример #1 Пример использования SoapClient::__setLocation()

<?php
$client = new SoapClient('http://example.com/webservice.php?wsdl');

$client->__setLocation('http://www.somethirdparty.com');

$old_location = $client->__setLocation(); // сбрасывает настройку адреса (location)

echo $old_location;

?>

Результатом выполнения данного примера будет что-то подобное:

http://www.somethirdparty.com
