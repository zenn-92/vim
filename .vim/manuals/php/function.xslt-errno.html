xslt_errno

(PHP 4 >= 4.0.3)

xslt_errno — Возвращает код ошибки

Описание

int xslt_errno ( resource $xh )

Возвращает код последней ошибки, произошедшей в работе заданного XSLT процессора.

Список параметров

xh

Идентификатор ссылки XSLT-процессора, созданный с помощью функции xslt_create().

Возвращаемые значения

Возвращает целое число, соответствующее коду ошибки.
