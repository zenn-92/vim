Imagick::getPointSize

(Информация о версии неизвестна, возможно, только в SVN)

Imagick::getPointSize — Возвращает размер точки

Описание

float Imagick::getPointSize ( void )

Возвращает значение размера точки объекта. Этот метод доступен, если Imagick был скомпилирован с версией ImageMagick 6.3.7 или старше.

Возвращаемые значения

Возвращает float содержащее размер точки.
