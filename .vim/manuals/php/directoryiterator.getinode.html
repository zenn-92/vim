DirectoryIterator::getInode

(PHP 5)

DirectoryIterator::getInode — Возвращает inode текущего элемента DirectoryIterator

Описание

public int DirectoryIterator::getInode ( void )

Возвращает inode текущего элемента DirectoryIterator.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает inode файла.

Примеры

Пример #1 Пример использования DirectoryIterator::getInode()

Этот пример выведет inode директории, содержащей выполняемый скрипт.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
echo $iterator->getInode();
?>
