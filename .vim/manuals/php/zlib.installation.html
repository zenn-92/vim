Установка

Поддержка Zlib в PHP отключена по умолчанию. Чтобы включить её, используйте опцию --with-zlib[=DIR] во время сборки.

Версия PHP для Windows имеет встроенную поддержку данного расширения. Это означает, что для использования данных функций не требуется загрузка никаких дополнительных расширений.

Замечание: Под Windows поддержка zlib включена начиная с PHP версии 4.3.0.
