Установка

Этот модуль включен по умолчанию начиная с PHP 5.3.0. До этого fileinfo являлся PECL модулем, но работа над ним больше не ведется. Тем не менее, версии до 5.3+ могут использовать » данный PECL модуль.

Windows-пользователи должны подключить идущий в комплекте поставки php_fileinfo.dll DLL файл в php.ini для включения данного модуля.

Библиотека libmagic поставляется в комплекте с PHP, но может содержать специфичные для PHP изменения, которые можно добавлять с помощью патча libmagic.patch. Этот патч поддерживается в актуальном состоянии, его можно найти там же, где находятся файлы расширений fileinfo.
