SplFileObject::hasChildren

(PHP 5 >= 5.1.2)

SplFileObject::hasChildren — Класс SplFileObject не имеет наследников

Описание

public bool SplFileObject::hasChildren ( void )

Класс SplFileObject не имеет наследников, поэтому метод всегда возвращает FALSE.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает FALSE
