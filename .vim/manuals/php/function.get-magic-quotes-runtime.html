get_magic_quotes_runtime

(PHP 4, PHP 5)

get_magic_quotes_runtime — Получение текущего значения настройки конфигурации magic_quotes_runtime

Описание

bool get_magic_quotes_runtime ( void )

Возвращает текущее значение настройки magic_quotes_runtime.

Возвращаемые значения

Возвращает 0, если magic_quotes_runtime выключена, 1 в противном случае. С версии PHP 5.4.0 всегда возвращает FALSE.

Список изменений

Версия Описание
5.4.0 Всегда возвращает FALSE, так как функционал магических кавычек удален из PHP.

Примеры

Пример #1 Пример использования get_magic_quotes_runtime()

<?php
// Проверка, работает ли magic_quotes_runtime
if(get_magic_quotes_runtime())
{
    // отключение
    set_magic_quotes_runtime(false);
}
?>
