SimpleXMLIterator::current

(PHP 5 >= 5.1.0)

SimpleXMLIterator::current — Возвращает текущий элемент

Описание

public mixed SimpleXMLIterator::current ( void )

Этот метод возвращает текущий элемент как объект класса SimpleXMLIterator или NULL.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает текущий элемент как объект класса SimpleXMLIterator или NULL в случае ошибки.

Примеры

Пример #1 Возвращение текущего элемента

<?php
$xmlIterator = new SimpleXMLIterator('<books><book>Основы PHP</book><book>Основы XML</book></books>');
var_dump($xmlIterator->current());

$xmlIterator->rewind(); // сбрасывает курсор к первому элементу
var_dump($xmlIterator->current());
?>

Результат выполнения данного примера:

NULL
object(SimpleXMLIterator)#2 (1) {
  [0]=>
  string(10) "Основы PHP"
}
