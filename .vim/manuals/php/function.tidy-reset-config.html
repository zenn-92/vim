tidy_reset_config

(PECL tidy >= 0.7.0)

tidy_reset_config — Сбрасывает конфигурацию Tidy в стандартные значения

Описание

bool tidy_reset_config ( void )

Эта функция сбрасывает конфигурацию Tidy в стандартные значения.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примечания

Замечание: Эта функция доступна только в Tidy 1.0. С приходом Tidy 2.0 она стала ненужной и была удалена.
