ReflectionExtension::info

(PHP 5)

ReflectionExtension::info — Вывод информации о расширении

Описание

public void ReflectionExtension::info ( void )

Выводит отрывок"phpinfo()", касающийся заданного расширения.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Информация о расширении.

Примеры

Пример #1 Пример использования ReflectionExtension::info()

<?php
$ext = new ReflectionExtension('mysqli');
$ext->info();
?>
