AppendIterator::getArrayIterator

(PHP 5 >= 5.1.0)

AppendIterator::getArrayIterator — Возвращает класс итератора массива ArrayIterator

Описание

public void AppendIterator::getArrayIterator ( void )

Этот метод получает класс ArrayIterator, который используется для хранения итераторов, добавленных с помощью метода AppendIterator::append().

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает класс ArrayIterator, содержащий добавленные итераторы.
