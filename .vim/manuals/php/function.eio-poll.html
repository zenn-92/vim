eio_poll

(PECL eio >= 0.0.1dev)

eio_poll — Может быть вызвана когда имеются запросы, ожидающие выполнения

Описание

int eio_poll ( void )

eio_poll() может быть использована для реализации особых циклов обработки запросов. При этом eio_nreqs() может быть использована для нахождения невыполненных запросов.

Замечание:

Применяется только для пользовательской реализации цикла обработки запросов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Если любой вызов запроса возвращает ненулевое значение, возвращается это значение. Иначе возвращает 0.

Примеры

Пример #1 Пример использования eio_poll()

<?php
function res_cb($data, $result) {
    var_dump($data);
    var_dump($result);
}

eio_nop(EIO_PRI_DEFAULT, "res_cb", "1");
eio_nop(EIO_PRI_DEFAULT, "res_cb", "2");
eio_nop(EIO_PRI_DEFAULT, "res_cb", "3");

while (eio_nreqs()) {
    // Некоторый специфичный IPC или что-то еще
    eio_poll();
}
?>

Результатом выполнения данного примера будет что-то подобное:

string(1) "1"
int(0)
string(1) "3"
int(0)
string(1) "2"
int(0)
