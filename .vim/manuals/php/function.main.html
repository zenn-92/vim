main

main — Заглушка для main()

Описание

Функция main() существует только в исходных кодах PHP. Пользовательской функции с таким именем нет. В коде PHP 4.3.0 был введен новый способ обработки ошибок php_error_docref. Одной из его возможностей является вывод ссылок на документацию в сообщениях об ошибках. Эта возможность задействована, если PHP заданы директивы html_errors (задана по умолчанию) и docref_root (задана по умолчанию в версиях PHP до 4.3.2).

Иногда сообщения об ошибках ссылаются на страницу документации к функции main(). Собственно поэтому эта страница и существует. Если вам попалась ссылка на эту страницу в каком-либо сообщении, пожалуйста » создайте баг-репорт с указанием PHP функции, выдавшей сообщение об ошибке со ссылкой на main(). Команда разработчиков в свою очередь исправит ошибку и внесет исправления в документацию.

Известные ошибки, которые ссылаются на main()
