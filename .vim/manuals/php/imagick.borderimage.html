Imagick::borderImage

(PECL imagick 2.0.0)

Imagick::borderImage — Окружает изображение рамкой

Описание

bool Imagick::borderImage ( mixed $bordercolor , int $width , int $height )

Окружает изображение рамкой с цветом установленным в объекте ImagickPixel.

Список параметров

bordercolor

Объект ImagickPixel или строка содержащая цвет рамки

width

Ширина рамки

height

Высота рамки

Возвращаемые значения

В случае успешной работы возвращает TRUE.

Список изменений

Версия Описание
2.1.0 В качестве цвета разрешено передавать строку. Предыдущие версии допускали только объект ImagickPixel.
