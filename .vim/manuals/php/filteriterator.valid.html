FilterIterator::valid

(PHP 5 >= 5.1.0)

FilterIterator::valid — Проверяет, является ли текущий элемент допустимым

Описание

bool FilterIterator::valid ( void )
Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Проверяет, является ли текущий элемент допустимым.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE если текущий элемент допустим, в противном случае FALSE
