GearmanTask::taskDenominator

(PECL gearman >= 0.5.0)

GearmanTask::taskDenominator — Получение знаменателя в дроби, отвечающей за процент выполненной работы

Описание

public int GearmanTask::taskDenominator ( void )

Получает знаменатель дроби, который используется для определения процента выполненной работы.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Число от 0 до 100 или FALSE, если определить значение не удалось.
