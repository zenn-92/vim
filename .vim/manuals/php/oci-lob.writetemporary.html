OCI-Lob::writeTemporary

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob::writeTemporary — Записывает временный большой объект (LOB)

Описание

bool OCI-Lob::writeTemporary ( string $data [, int $lob_type = OCI_TEMP_CLOB ] )

Создает временный большой объект и записывает в него данные data.

После завершения работы с этим объектом его желательно закрыть функцией OCI-Lob::close.

Список параметров

data

Данные для записи.

lob_type

Один из следующих вариантов:

  • OCI_TEMP_BLOB для создания временных BLOBов
  • OCI_TEMP_CLOB для создания временных CLOBов

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
