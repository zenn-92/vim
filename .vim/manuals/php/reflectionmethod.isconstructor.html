ReflectionMethod::isConstructor

(PHP 5)

ReflectionMethod::isConstructor — Проверяет, является ли метод конструктором

Описание

public bool ReflectionMethod::isConstructor ( void )

Проверяет, является ли метод конструктором.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если метод является конструктором, FALSE в противном случае.
