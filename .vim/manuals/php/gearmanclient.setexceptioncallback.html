GearmanClient::setExceptionCallback

(PECL gearman >= 0.5.0)

GearmanClient::setExceptionCallback — Задание функции обратного вызова для перехвата исключений обработчика заданий

Описание

public bool GearmanClient::setExceptionCallback ( callable $callback )

Задает функцию, которая будет вызываться, когда обработчик задания пошлет исключение.

Список параметров

callback

Функция для перехвата исключения, выброшенного обработчиком заданий.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
