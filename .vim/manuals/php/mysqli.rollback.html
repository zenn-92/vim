mysqli::rollback

mysqli_rollback

(PHP 5)

mysqli::rollback -- mysqli_rollback — Откат текущей транзакции

Описание

Объектно-ориентированный стиль

bool mysqli::rollback ( void )

Процедурный стиль

bool mysqli_rollback ( mysqli $link )

Откатывает текущую транзакцию.

Список параметров

link

Только для процедурного стиля: Идентификатор соединения, полученный с помощью mysqli_connect() или mysqli_init()

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования mysqli::rollback()

Объектно-ориентированный стиль

<?php
$mysqli = new mysqli("localhost", "my_user", "my_password", "world");

/* проверка соединения */
if (mysqli_connect_errno()) {
    printf("Не удалось подключиться: %s\n", mysqli_connect_error());
    exit();
}

/* выключаем автофиксацию */
$mysqli->autocommit(FALSE);

$mysqli->query("CREATE TABLE myCity LIKE City");
$mysqli->query("ALTER TABLE myCity Type=InnoDB");
$mysqli->query("INSERT INTO myCity SELECT * FROM City LIMIT 50");

/* фиксируем вставленные строки */
$mysqli->commit();

/* удаляем все строки */
$mysqli->query("DELETE FROM myCity");

if ($result = $mysqli->query("SELECT COUNT(*) FROM myCity")) {
    $row = $result->fetch_row();
    printf("%d строк в таблице myCity.\n", $row[0]);
    /* очищаем результат */
    $result->close();
}

/* откат */
$mysqli->rollback();

if ($result = $mysqli->query("SELECT COUNT(*) FROM myCity")) {
    $row = $result->fetch_row();
    printf("%d строк в таблице myCity (after rollback).\n", $row[0]);
    /* очищаем результат */
    $result->close();
}

/* удаляем таблицу myCity */
$mysqli->query("DROP TABLE myCity");

$mysqli->close();
?>

Процедурный стиль

<?php
$link = mysqli_connect("localhost", "my_user", "my_password", "world");

/* проверка соединения */
if (mysqli_connect_errno()) {
    printf("Не удалось подключиться: %s\n", mysqli_connect_error());
    exit();
}

/* выключаем автофиксацию */
mysqli_autocommit($link, FALSE);

mysqli_query($link, "CREATE TABLE myCity LIKE City");
mysqli_query($link, "ALTER TABLE myCity Type=InnoDB");
mysqli_query($link, "INSERT INTO myCity SELECT * FROM City LIMIT 50");

/* фиксируем вставленные строки */
mysqli_commit($link);

/* удаляем все строки */
mysqli_query($link, "DELETE FROM myCity");

if ($result = mysqli_query($link, "SELECT COUNT(*) FROM myCity")) {
    $row = mysqli_fetch_row($result);
    printf("%d строк в таблице myCity.\n", $row[0]);
    /* очищаем результат */
    mysqli_free_result($result);
}

/* откат */
mysqli_rollback($link);

if ($result = mysqli_query($link, "SELECT COUNT(*) FROM myCity")) {
    $row = mysqli_fetch_row($result);
    printf("%d строк в таблице myCity (после отката).\n", $row[0]);
    /* очищаем результат */
    mysqli_free_result($result);
}

/* удаляем таблицу myCity */
mysqli_query($link, "DROP TABLE myCity");

mysqli_close($link);
?>

Результат выполнения данных примеров:

0 строк в таблице myCity.
50 строк в таблице myCity (после отката).
