SplFileInfo::isReadable

(PHP 5 >= 5.1.2)

SplFileInfo::isReadable — Сообщает, является ли файл доступным для чтения

Описание

public bool SplFileInfo::isReadable ( void )

Проверяет, является ли файл доступным для чтения.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE если файл доступен для чтения; в противном случае - FALSE.

Примеры

Пример #1 Пример использования SplFileInfo::isReadable()

<?php
$info = new SplFileInfo(__FILE__);
var_dump($info->isReadable());

$info = new SplFileInfo('foo');
var_dump($info->isReadable());
?>

Результатом выполнения данного примера будет что-то подобное:

bool(true)
bool(false)
