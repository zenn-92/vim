streamWrapper::rename

(PHP 5)

streamWrapper::rename — Переименовывает файл или директорию

Описание

public bool streamWrapper::rename ( string $path_from , string $path_to )

Этот метод вызывается в результате вызова rename().

Должен попытаться переименовать path_from в path_to

Замечание:

Чтобы сообщения об ошибках соответствовали реальным ошибкам, этот метод не нужно определять в случах, когда обертка не поддерживает переименование файлов.

Список параметров

path_from

URL текущего файла.

path_to

URL в который path_from нужно переименовать.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Ошибки

Вызывает ошибку уровня E_WARNING, если вызов к этому методу не удался (например, не реализован).

Примечания

Замечание:

Свойство streamWrapper::$context будет обновлено, если корректный контекст был передан в вызываемую функцию.
