eio_init

(PECL eio = 1.0.0)

eio_init — (Re-)initialize Eio

Описание

void eio_init ( void )

eio_init() (ре-)инициализирует Eio. Резервируется память для внутренних структур libeio и Eio. Можно вызывать eio_init() перед использованием Eio-функций. В любом случае, инициализация будет выполнена при первом вызове Eio-функции.

Внимание

Эта возможность устарела и будет обязательно удалена в будущем. Начиная с Eio 1.1.0 eio_init() считается устаревшей. В Eio 1.0.0 потому что » ограничения libeio необходим вызов eio_init() в дочерних процессах или в других случаях. Необходимо избегать использование Eio в родительских процессах если оно используется в дочерних.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
