DirectoryIterator::isExecutable

(PHP 5)

DirectoryIterator::isExecutable — Определяет, является ли текущий элемент DirectoryIterator исполняемым

Описание

public bool DirectoryIterator::isExecutable ( void )

Определяет, является ли текущий элемент DirectoryIterator исполняемым.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE, если файл является исполняемым, иначе FALSE

Примеры

Пример #1 DirectoryIterator::isExecutable() example

Этот пример показывает список исполняемых файлов в директории, содержащей выполняемый скрипт.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    if ($fileinfo->isExecutable()) {
        echo $fileinfo->getFilename() . "\n";
    }
}
?>

Результатом выполнения данного примера будет что-то подобное:

example.php
myscript.sh
