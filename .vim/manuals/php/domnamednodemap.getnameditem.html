DOMNamedNodeMap::getNamedItem

(PHP 5)

DOMNamedNodeMap::getNamedItem — Извлекает узел с заданным именем

Описание

DOMNode DOMNamedNodeMap::getNamedItem ( string $name )

Извлекает узел с заданным значением тэга nodeName.

Список параметров

name

Значение тэга nodeName искомого узла.

Возвращаемые значения

Узел (любого типа) с заданным значением тэга nodeName или NULL, если таковых не найдено.
