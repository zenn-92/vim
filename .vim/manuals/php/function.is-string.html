is_string

(PHP 4, PHP 5)

is_string — Проверяет, является ли переменная строкой

Описание

bool is_string ( mixed $var )

Проверяет, является ли данная переменная строкой.

Список параметров

var

Проверяемая переменная.

Возвращаемые значения

Возвращает TRUE, если var является строкой, и FALSE в противном случае.

Примеры

Пример #1 Пример использования is_string()

<?php
if (is_string("23")) {
    echo "строка\n";
} else {
    echo "не строка\n";
}
var_dump(is_string('abc'));
var_dump(is_string("23"));
var_dump(is_string(23.5));
var_dump(is_string(true));
?>

Результат выполнения данного примера:

строка
bool(true)
bool(true)
bool(false)
bool(false)

Смотрите также

  • is_int() - Проверяет, является ли переменная переменной целочисленного типа
  • is_bool() - Проверяет, является ли переменная булевой
  • is_object() - Проверяет, является ли переменная объектом
  • is_array() - Определяет, является ли переменная массивом
