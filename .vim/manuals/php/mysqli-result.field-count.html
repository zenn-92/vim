mysqli_result::$field_count

mysqli_num_fields

(PHP 5)

mysqli_result::$field_count -- mysqli_num_fields — Получение количества полей в результирующем наборе

Описание

Объектно-ориентированный стиль

int $mysqli_result->field_count;

Процедурный стиль

int mysqli_num_fields ( mysqli_result $result )

Возвращает количество полей в результирующем наборе.

Список параметров

result

Только для процедурного стиля: Идентификатор результата запроса, полученный с помощью mysqli_query(), mysqli_store_result() или mysqli_use_result().

Возвращаемые значения

Количество столбцов в результирующей таблице.

Примеры

Пример #1 Объектно-ориентированный стиль

<?php
$mysqli = new mysqli("localhost", "my_user", "my_password", "world");

/* проверка подключения */
if (mysqli_connect_errno()) {
    printf("Не удалось подключиться: %s\n", mysqli_connect_error());
    exit();
}

if ($result = $mysqli->query("SELECT * FROM City ORDER BY ID LIMIT 1")) {

    /* определяем количество стобцов в результирующей таблице */
    $field_cnt = $result->field_count;

    printf("Результат содержит %d полей.\n", $field_cnt);

    /* закрываем результирующий набор */
    $result->close();
}

/* закрываем подключение */
$mysqli->close();
?>

Пример #2 Процедурный стиль

<?php
$link = mysqli_connect("localhost", "my_user", "my_password", "world");

/* проверка подключения */
if (mysqli_connect_errno()) {
    printf("Не удалось подключиться: %s\n", mysqli_connect_error());
    exit();
}

if ($result = mysqli_query($link, "SELECT * FROM City ORDER BY ID LIMIT 1")) {

    /* определяем количество стобцов в результирующей таблице */
    $field_cnt = mysqli_num_fields($result);

    printf("Результат содержит %d полей.\n", $field_cnt);

    /* закрываем результирующий набор */
    mysqli_free_result($result);
}

/* закрываем подключение */
mysqli_close($link);
?>

Результат выполнения данных примеров:

Результат содержит 5 полей.
