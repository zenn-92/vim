FilesystemIterator::setFlags

(PHP 5 >= 5.3.0)

FilesystemIterator::setFlags — Задание настроечных флагов

Описание

public void FilesystemIterator::setFlags ([ int $flags ] )

Задает настройки итератора.

Список параметров

flags

Флаги, которые необходимо установить. См. Константы FilesystemIterator.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования FilesystemIterator::key()

Этот пример демонстрирует различия между флагами FilesystemIterator::KEY_AS_PATHNAME и FilesystemIterator::KEY_AS_FILENAME.

<?php
$iterator = new FilesystemIterator(dirname(__FILE__), FilesystemIterator::KEY_AS_PATHNAME);
echo "Ключом является путь к файлу:\n";
foreach ($iterator as $key => $fileinfo) {
    echo $key . "\n";
}

$iterator->setFlags(FilesystemIterator::KEY_AS_FILENAME);
echo "\nКлючом является имя файла:\n";
foreach ($iterator as $key => $fileinfo) {
    echo $key . "\n";
}
?>

Результатом выполнения данного примера будет что-то подобное:

Ключом является путь к файлу:
/www/examples/apple.jpg
/www/examples/banana.jpg
/www/examples/example.php

nКлючом является имя файла:
apple.jpg
banana.jpg
example.php
