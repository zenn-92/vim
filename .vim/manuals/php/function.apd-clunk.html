apd_clunk

(PECL apd 0.2-0.4)

apd_clunk — Выводит предупреждение и стек вызова

Описание

void apd_clunk ( string $warning [, string $delimiter ] )

В поведении аналогичен Carp::cluck в PERL. Выводит предупреждение и стек вызова.

Список параметров

warning

Предупреждение для вывода.

delimiter

Разделитель. По умолчанию <BR />.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования apd_clunk()

<?php
apd_clunk("Some Warning", "<br/>");
?>
