mysqli::refresh

mysqli_refresh

(PHP 5 <= 5.3.0)

mysqli::refresh -- mysqli_refresh — Обновление

Описание

Объектно-ориентированный стиль

public bool mysqli::refresh ( int $options )

Процедурный стиль

int mysqli_refresh ( resource $link , int $options )

Очищает таблицы или кэш, либо сбрасывает копию серверной информации.

Список параметров

link

Только для процедурного стиля: Идентификатор соединения, полученный с помощью mysqli_connect() или mysqli_init()

options

Настройки обновления задаются константами MYSQLI_REFRESH_*, которые описаны в разделе документации Константы MySQLi.

См. также официальную документацию » MySQL Refresh.

Возвращаемые значения

TRUE в случае успешного выполнения, FALSE в противном случае.
