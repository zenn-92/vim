DOMAttr::isId

(PHP 5)

DOMAttr::isId — Проверяет, является ли атрибут описанным в DTD ID

Описание

public bool DOMAttr::isId ( void )

Эта функция проверяет, является ли атрибут ID, описанным в DTD.

В соответствии со стандартом DOM это требует наличия DTD, которая предписывает ID атрибуту иметь соответствующий ID тип. Перед применением этой функции необходимо проверять документ на действительность с помощью DOMDocument::validate или DOMDocument::validateOnParse.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования DOMAttr::isId()

<?php

$doc = new DomDocument;

// Необходимо проверить документ на действительность перед тем как ссылаться на id
$doc->validateOnParse = true;
$doc->Load('book.xml');

// Достаем атрибут с именем id из элемента chapter 
$attr = $doc->getElementsByTagName('chapter')->item(0)->getAttributeNode('id');

var_dump($attr->isId()); // bool(true)

?>
