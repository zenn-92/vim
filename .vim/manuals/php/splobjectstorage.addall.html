SplObjectStorage::addAll

(PHP 5 >= 5.3.0)

SplObjectStorage::addAll — Добавляет все объекты из другого контейнера

Описание

public void SplObjectStorage::addAll ( SplObjectStorage $storage )

Добавляет все пары объект-данные из другого контейнера в текущий.

Список параметров

storage

Контейнер объектов, из которого требуется импортировать данные.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования SplObjectStorage::addAll()

<?php
$o = new StdClass;
$a = new SplObjectStorage();
$a[$o] = "hello";

$b = new SplObjectStorage();
$b->addAll($a);
echo $b[$o]."\n";
?>

Результатом выполнения данного примера будет что-то подобное:

hello
