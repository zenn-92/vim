ReflectionMethod::isProtected

(PHP 5)

ReflectionMethod::isProtected — Проверяет, является ли метод защищенным (protected)

Описание

public bool ReflectionMethod::isProtected ( void )

Проверяет, является ли метод защищенным (protected).

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если метод защищенный (protected), FALSE в противном случае.
