maxdb_close_long_data

maxdb::close_long_data

(PECL maxdb >= 1.0)

maxdb_close_long_data -- maxdb::close_long_data — Псевдоним maxdb_stmt_close_long_data()

Описание

Эта функция является псевдонимом: maxdb_stmt_close_long_data().

Эта функция-псевдоним устарела и существует только по причине обратной совместимости. Использование этой функции не рекомендуется, так как в будущем она может быть удалена из PHP.
