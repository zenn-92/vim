GearmanJob::functionName

(PECL gearman >= 0.5.0)

GearmanJob::functionName — Получение имени функции

Описание

public string GearmanJob::functionName ( void )

Возвращает имя функции, обрабатывающей данные, то есть функции, которая будет вызвана для выполнения работы.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Имя функции.
