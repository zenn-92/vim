Новые расширения

Следующие расширения были добавлены начиная с PHP 5.3.0:

  • Enchant - Слой абстракции над различными библиотеками правописания
  • Fileinfo - Усовершенствованная и более серьезная замена (с полной бинарной совместимостью) расширению Mimetype, которое было удалено.
  • INTL - Расширение для интернационализации. INTL - это обертка вокруг библиотеки » ICU.
  • Phar - Реализация файлов-архивов PHP.
  • SQLite3 - Поддержка БД SQLite версии 3.

Mysqlnd - это новая основная библиотека, поставляемая с PHP. Это адаптированная для PHP замена для libmysql. Mysqlnd будет использована для сборки расширения mysql, mysqli и PDO_MySQL, если libmysql не будет найдена в системе. Она также может быть использована вместо libmysql даже когда libmysql присутствует. Mysqlnd рекомендуется для всех инсталляций PHP из соображений производительности.
