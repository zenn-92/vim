ReflectionClass::getConstants

(PHP 5)

ReflectionClass::getConstants — Возвращает константы

Описание

public array ReflectionClass::getConstants ( void )

Возвращает определённые в классе константы.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Массив (array) констант. Имя константы - ключ, значение константы - значение.
