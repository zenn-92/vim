stream_context_get_params

(PHP 5 >= 5.3.0)

stream_context_get_params — Извлекает параметры из контекста

Описание

array stream_context_get_params ( resource $stream_or_context )

Возвращает информацию о параметрах и опциях из потока или контекста.

Список параметров

stream_or_context

Ресурс resource потока или контекстовый ресурс resource

Возвращаемые значения

Возвращает ассоциативный массив, содержащий все опции и параметры контекста.

Примеры

Пример #1 Пример использования stream_context_get_params()

Пример простого использования

<?php
$ctx = stream_context_create();
$params = array("notification" => "stream_notification_callback");
stream_context_set_params($ctx, $params);

var_dump(stream_context_get_params($ctx));
?>

Результатом выполнения данного примера будет что-то подобное:

array(2) {
  ["notification"]=>
  string(28) "stream_notification_callback"
  ["options"]=>
  array(0) {
  }
}
