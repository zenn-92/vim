SimpleXMLElement::getName

(PHP 5 >= 5.1.3)

SimpleXMLElement::getName — Получение имени XML элемента

Описание

public string SimpleXMLElement::getName ( void )

Получает имя XML элемента.

Возвращаемые значения

Метод getName возвращает имя тега XML в виде строки string, на который ссылается SimpleXMLElement объект.

Примеры

Замечание:

Перечисленные примеры могут включать example.php, в котором определяется XML строка, расположенная в первом примере руководства по базовому использованию.

Пример #1 Получение имен XML элемента

<?php
include 'example.php';
$sxe = new SimpleXMLElement($xmlstr);

echo $sxe->getName() . "\n";

foreach ($sxe->children() as $child)
{
    echo $child->getName() . "\n";
}

?>

Результат выполнения данного примера:

movies
movie
