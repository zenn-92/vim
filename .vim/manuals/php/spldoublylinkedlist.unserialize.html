SplDoublyLinkedList::unserialize

(PHP 5 >= 5.4.0)

SplDoublyLinkedList::unserialize — Десериализует хранилище

Описание

public void SplDoublyLinkedList::unserialize ( string $serialized )

Десериализует хранилище из строковго представления, полученного с помощью SplDoublyLinkedList::serialize().

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

serialized

Сериализованная строка.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
