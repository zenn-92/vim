Введение

XMLWriter является расширением PHP. Оно создает XmlWriter API на основе библиотеки libxml.

Это расширение представляет собой класс-писатель, который является некэшируемым, однонаправленным средством генерирования потоков или файлов, содержащих данные XML.

Это расширение может быть использовано в объектно-ориентированном либо в процедурном стилях. Каждый документированный метод описывает альтернативные процедурные вызовы.
