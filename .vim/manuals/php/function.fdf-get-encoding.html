fdf_get_encoding

(PHP 4 >= 4.3.0, PHP 5)

fdf_get_encoding — Получает значение ключа /Encoding

Описание

string fdf_get_encoding ( resource $fdf_document )

Получает значение ключа /Encoding.

Список параметров

fdf_document

Дескриптор FDF-документа, возвращенный функциями fdf_create(), fdf_open() или fdf_open_string().

Возвращаемые значения

Возвращает кодировку в виде строки. Возвращается пустая строка, если используется схема по умолчанию PDFDocEncoding/Unicode.

Смотрите также

  • fdf_set_encoding() - Sets FDF character encoding
