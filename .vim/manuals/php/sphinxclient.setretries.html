SphinxClient::setRetries

(PECL sphinx >= 0.1.0)

SphinxClient::setRetries — Установка количества повторных попыток и задержки между ними

Описание

public bool SphinxClient::setRetries ( int $count [, int $delay = 0 ] )

Устанавливает на searchd количество повторных попыток и задержку между ними.

При временных проблемах searchd будет делать повторные попытки вплоть до count на одного агента. delay - это задержка между попытками, в миллисекундах. По умолчанию повторные попытки отключены. Обратите внимание, что этот запрос не дает команды самому API делать повторные попытки в случае неудачи. Он лишь передает новые параметры демону searchd.

Список параметров

count

Количество повторных попыток.

delay

Задержка между попытками, в миллисекундах.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
