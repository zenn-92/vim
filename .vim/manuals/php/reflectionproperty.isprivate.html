ReflectionProperty::isPrivate

(PHP 5)

ReflectionProperty::isPrivate — Проверяет, является ли свойство частным (private)

Описание

public bool ReflectionProperty::isPrivate ( void )

Проверяет, является ли свойство частным (private).

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если свойство частное, FALSE в противном случае.

Смотрите также

  • ReflectionProperty::isPublic() - Проверяет, является ли свойство общедоступным (public)
