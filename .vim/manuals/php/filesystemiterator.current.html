FilesystemIterator::current

(PHP 5 >= 5.3.0)

FilesystemIterator::current — Текущий файл

Описание

public mixed FilesystemIterator::current ( void )

Извлекает информацию о текущем файле.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Имя файла, информация о файле, либо $this, в зависимости от установленных флагов. См. Константы FilesystemIterator.

Примеры

Пример #1 Пример использования FilesystemIterator::current()

В этом примере будет выведен список содержимого директории, в которой находится выполняемый скрипт.

<?php
$iterator = new FilesystemIterator(dirname(__FILE__), FilesystemIterator::CURRENT_AS_PATHNAME);
foreach ($iterator as $fileinfo) {
    echo $iterator->current() . "\n";
}
?>

Результатом выполнения данного примера будет что-то подобное:

/www/examples/apple.jpg
/www/examples/banana.jpg
/www/examples/example.php
