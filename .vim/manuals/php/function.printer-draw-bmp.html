printer_draw_bmp

(PECL printer SVN)

printer_draw_bmp — Рисует bmp

Описание

bool printer_draw_bmp ( resource $printer_handle , string $filename , int $x , int $y [, int $width ], int $height )

Эта функция рисует картинку из файла с расширением .bmp.

Список параметров

printer_handle

printer_handle должен быть действительным дескриптором принтера.

filename

Путь к рисуемому файлу bmp.

x

Верхняя левая координата x картинки.

y

Верхняя левая координата y картинки.

width

Ширина картинки.

height

Высота картинки.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример printer_draw_bmp()

<?php
$handle = printer_open();
printer_start_doc($handle, "My Document");
printer_start_page($handle);

printer_draw_bmp($handle, "c:\\image.bmp", 1, 1);

printer_end_page($handle);
printer_end_doc($handle);
printer_close($handle);
?>
