Предопределенные константы

Перечисленные ниже константы определены данным расширением и могут быть доступны только в том случае, если PHP был собран с поддержкой этого расширения или же в том случае, если данное расширение было подгружено во время выполнения.

GD_VERSION (string)
Версия GD, с которой был скомпилирован PHP. (Доступна с PHP 5.2.4)
GD_MAJOR_VERSION (integer)
Полнофункциональная версия GD, с которой был скомпилирован PHP. (Доступна с PHP 5.2.4)
GD_MINOR_VERSION (integer)
Сокращенная версия GD, с которой был скомпилирован PHP. (Доступна с PHP 5.2.4)
GD_RELEASE_VERSION (integer)
Рабочая версия GD, с которой был скомпилирован PHP. (Доступна с PHP 5.2.4)
GD_EXTRA_VERSION (string)
Дополнительная (beta/rc..) версия GD, с которой был скомпилирован PHP. (Доступна с PHP 5.2.4)
GD_BUNDLED (integer)
Если используется распространяемая с PHP версия GD, константа принимает значение 1, в противном случае 0.
IMG_GIF (integer)
Используется в качестве возвращаемого значения функцией imagetypes()
IMG_JPG (integer)
Используется в качестве возвращаемого значения функцией imagetypes()
IMG_JPEG (integer)
Используется в качестве возвращаемого значения функцией imagetypes()

Замечание:

Эта константа имеет то же значение, что и IMG_JPG
