gzdecode

(PHP 5 >= 5.4.0)

gzdecode — Распаковывает строку, упакованную с помощью gzip

Описание

string gzdecode ( string $data [, int $length ] )

Эта функция возвращает распакованную версию входных данных. data.

Список параметров

data

Данные для распаковки, упакованные с помощью gzencode().

length

Максимальный размер строки для распаковки.

Возвращаемые значения

Распакованная строка или FALSE, если произошла ошибка.

  • gzencode() - Сжимает строку в формате gzip
