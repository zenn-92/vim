timezone_version_get

(PHP 5 >= 5.3.0)

timezone_version_get — Получение номера версии базы данных временных зон

Описание

string timezone_version_get ( void )

Возвращает номер версии базы данных временных зон.

Возвращаемые значения

Возвращает string.

Примеры

Пример #1 Определение версии базы данных временных зон

<?php
echo timezone_version_get();
?>

Результатом выполнения данного примера будет что-то подобное:

2009.7
