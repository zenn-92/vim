Изменения в PHP 5.2.x

Большая часть изменений в PHP 5.2.x не затронет существующий код. Тем не менее, есть несколько несовместимых изменений и новых сообщений об ошибках, которые должны быть учтены при переходе. Также рекомендуется оттестировать код перед переходом на новую версию PHP в боевой установке.

Прочтите раздел "Переход на PHP 5.1.x", если предыдущей версией PHP была PHP 5.0.x.

Аналогично, если предыдущей версией была PHP 4, прочтите также раздел "Переход с PHP 4 на PHP 5.0.x".
