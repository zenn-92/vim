stream_resolve_include_path

(PHP 5 >= 5.3.2)

stream_resolve_include_path — Составляет полное имя файла, используя пути включения

Описание

string stream_resolve_include_path ( string $filename )

Составляет полное имя файла из параметра filename, используя пути включения в соответствии с теми же правилами, что и функции fopen()/include.

Список параметров

filename

Имя файла, которое необходимо преобразовать.

Возвращаемые значения

Возвращает строку string, содержащую полученное абсолютное имя файла, или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования stream_resolve_include_path()

Основной пример использования.

<?php
var_dump(stream_resolve_include_path("test.php"));
?>

Результатом выполнения данного примера будет что-то подобное:

string(22) "/var/www/html/test.php"
