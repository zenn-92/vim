ArrayIterator::append

(PHP 5 >= 5.0.0)

ArrayIterator::append — Добавить элемент

Описание

public void ArrayIterator::append ( mixed $value )

Добавляет значение value, как последний элемент массива.

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

value

Значение, которое нужно добавить.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примечания

Замечание:

Этот метод не может быть вызван, если ArrayIterator указывает на object.
