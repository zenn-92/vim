DirectoryIterator::next

(PHP 5)

DirectoryIterator::next — Перемещает указатель на следующий элемент DirectoryIterator

Описание

public void DirectoryIterator::next ( void )

Перемещает указатель на следующий элемент DirectoryIterator.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования DirectoryIterator::next()

Вывод списка содержимого директории с помощью цикла while

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
while($iterator->valid()) {
    echo $iterator->getFilename() . "\n";
    $iterator->next();
}
?>

Результатом выполнения данного примера будет что-то подобное:

.
..
apple.jpg
banana.jpg
index.php
pear.jpg

  • Iterator::next() - Переходит к следующему элементу
