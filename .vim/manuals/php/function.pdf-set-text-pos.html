PDF_set_text_pos

(PHP 4, PECL pdflib >= 1.0.0)

PDF_set_text_pos — Устанавливает позицию текста

Описание

bool PDF_set_text_pos ( resource $p , float $x , float $y )

Устанавливает позицию вывода текста на страницу. Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
