SplStack::__construct

(PHP 5 >= 5.3.0)

SplStack::__construct — Создатет новый стек, реализованный с использованием двусвязного списка

Описание

SplStack::__construct ( void )

Создаёт новый пустой стек.

Замечание:

Этот метод автоматически устанавливает режим итератора SplDoublyLinkedList::IT_MODE_LIFO.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример использования SplStack::__construct()

<?php
$q = new SplStack();

$q[] = 1;
$q[] = 2;
$q[] = 3;

foreach ($q as $elem)  {
 echo $elem."\n";
}
?>

Результат выполнения данного примера:

3
2
1
