trader_macdext

(PECL trader >= 0.2.0)

trader_macdext — MACD with controllable MA type

Описание

array trader_macdext ( array $real [, integer $fastPeriod [, integer $fastMAType [, integer $slowPeriod [, integer $slowMAType [, integer $signalPeriod [, integer $signalMAType ]]]]]] )

Список параметров

real

Массив, содержащий реальные значения.

fastPeriod

Номер периода для быстрого скользящего среднего. Допустимы значения от 2 до 100000.

fastMAType

Тип скользящей средней для быстрого скользящего среднего. Следует использовать серию констант TRADER_MA_TYPE_*.

slowPeriod

Номер периода для медленного скользящего среднего. Допустимы значения от 2 до 100000.

slowMAType

Тип скользящей средней для медленного скользящего среднего. Следует использовать серию констант TRADER_MA_TYPE_*.

signalPeriod

Сглаживание для сигнальной линии (номер периода). Допустимы значения от 1 до 100000.

signalMAType

Тип скользящей средней для сигнальной линии. Следует использовать серию констант TRADER_MA_TYPE_*.

Возвращаемые значения

Returns an array with calculated data or false on failure.
