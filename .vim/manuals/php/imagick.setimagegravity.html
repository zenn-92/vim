Imagick::setImageGravity

(Информация о версии неизвестна, возможно, только в SVN)

Imagick::setImageGravity — Sets the image gravity

Описание

bool Imagick::setImageGravity ( int $gravity )

Sets the gravity property for the current image. This method can be used to set the gravity property for a single image sequence. Этот метод доступен, если Imagick был скомпилирован с версией ImageMagick 6.4.4 или старше.

Список параметров

gravity

The gravity property. Refer to the list of gravity constants.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.
