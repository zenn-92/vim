SplFileInfo::getType

(PHP 5 >= 5.1.2)

SplFileInfo::getType — Получает тип файла

Описание

public string SplFileInfo::getType ( void )

Возвращает тип указанного файла.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Строка (string), представляющая тип элемента. Возможны следующие значения: file, link или dir.

Ошибки

Выбрасывает RuntimeException в случае ошибки.

Примеры

Пример #1 Пример использования SplFileInfo::getType()

<?php

$info = new SplFileInfo(__FILE__);
echo $info->getType().PHP_EOL;

$info = new SplFileInfo(dirname(__FILE__));
echo $info->getType();

?>

Результатом выполнения данного примера будет что-то подобное:

file
dir
