use_soap_error_handler

(Unknown)

use_soap_error_handler — Указывает, нужно ли использовать обработчик ошибок SOAP

Описание

bool use_soap_error_handler ([ bool $handler = true ] )

Эта функция определяет, могут ли использоваться обработчики ошибок SOAP в SOAP-сервере. Функция возвращает предыдущее значение. Если установлено в TRUE, то подробности ошибок в приложении с классом SoapServer будут отправлены клиентам. Если установлено в FALSE, то информация не отсылается.

Список параметров

handler

Для отправки подробностей ошибок клиентам необходимо установить в TRUE.

Возвращаемые значения

Возвращает предыдущее значение.

  • set_exception_handler() - Задает пользовательский обработчик исключений
