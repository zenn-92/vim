gnupg_setarmor

(PECL gnupg >= 0.1)

gnupg_setarmor — Переключает вывод в текстовом или бинарном режиме

Описание

bool gnupg_setarmor ( resource $identifier , int $armor )

Переключает вывод в текстовом или бинарном режиме.

Список параметров

identifier

Идентификатор gnupg, полученный из gnupg_init() или gnupg.

armor

В случае не нулевого целого числа, функция включает текстовый режим вывода (по умолчанию). В случае 0, вывод в бинарном режиме.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Процедурный пример использования gnupg_setarmor()

<?php
$res = gnupg_init();
gnupg_setarmor($res,1); // текстовый режим вывода;
gnupg_setarmor($res,0); // бинарный режим вывода;
?>

Пример #2 Объектно-ориентированный пример использования gnupg_setarmor()

<?php
$gpg = new gnupg();
$gpg -> setarmor(1); // текстовый режим вывода;
$gpg -> setarmor(0); // бинарный режим вывода;
?>
