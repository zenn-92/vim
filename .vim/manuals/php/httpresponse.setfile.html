HttpResponse::setFile

(PECL pecl_http >= 0.10.0)

HttpResponse::setFile — Set file

Описание

static bool HttpResponse::setFile ( string $file )

Set the file to be sent.

Замечание: Вычисленные или определенные ранее ETag и Last-Modified будут пересчитаны и переопределены заново.

Список параметров

file

the path to the file to send

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
