filter_id

(PHP 5 >= 5.2.0)

filter_id — Возвращает идентификатор, принадлежащий именованному фильтру

Описание

int filter_id ( string $filtername )

Список параметров

filtername

Имя фильтра.

Возвращаемые значения

Идентификатор фильтра в случае успеха или FALSE, если фильтр не существует.

  • filter_list() - Возвращает список всех поддерживаемых фильтров
