Требования

Для использования функций cURL в PHP необходимо установить пакет » libcurl. Необходимо использовать версию libcurl 7.0.2-beta или выше. Для PHP 4.2.3 необходимо использовать версию libcurl 7.9.0 или выше. Начиная с версии PHP 4.3.0, необходима версия libcurl 7.9.8 или выше. PHP 5.0.0 требует libcurl не младше 7.10.5.
