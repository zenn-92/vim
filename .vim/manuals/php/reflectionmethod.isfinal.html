ReflectionMethod::isFinal

(PHP 5)

ReflectionMethod::isFinal — Проверяет, может ли метод иметь наследников (final)

Описание

public bool ReflectionMethod::isFinal ( void )

Проверяет, может ли метод иметь потомков (final).

Список параметров

У этой функции нет параметров.

Возвращаемые значения

TRUE, если создать наследника метода нельзя, FALSE в противном случае.
