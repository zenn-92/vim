SplFileInfo::getMTime

(PHP 5 >= 5.1.2)

SplFileInfo::getMTime — Получает время последнего изменения

Описание

public int SplFileInfo::getMTime ( void )

Возвращает время, когда содержимое файла было изменено. Время возвращается в формате Unix timestamp.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает время последнего изменения файла в формате Unix timestamp.

  • filemtime() - Возвращает время последнего изменения файла
