XMLWriter::writeAttributeNS

xmlwriter_write_attribute_ns

(PHP 5 >= 5.1.2, PECL xmlwriter >= 0.1.0)

XMLWriter::writeAttributeNS -- xmlwriter_write_attribute_ns — Записать полный атрибут пространства имен

Описание

Объектно-ориентированный стиль

bool XMLWriter::writeAttributeNS ( string $prefix , string $name , string $uri , string $content )

Процедурный стиль

bool xmlwriter_write_attribute_ns ( resource $xmlwriter , string $prefix , string $name , string $uri , string $content )

Записывает полный атрибут пространства имен.

Список параметров

xmlwriter

Только для процедурных вызовов. Изменяемый resource XMLWriter. Этот ресурс может быть получен из xmlwriter_open_uri() или xmlwriter_open_memory().

prefix

Префикс пространства имен.

name

Имя атрибута.

uri

URI пространства имен.

content

Значение атрибута.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
