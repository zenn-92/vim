GearmanJob::data

(PECL gearman <= 0.5.0)

GearmanJob::data — Отправка данных (устаревший метод)

Описание

public bool GearmanJob::data ( string $data )

Отправляет данные серверу заданий (и всем слушающим клиентам).

Замечание:

Этот метод был заменен на GearmanJob::sendData() в выпуске 0.6.0 расширения Gearman.

Список параметров

data

Сериализованные данные.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Смотрите также

  • GearmanJob::workload() - Получение данных для обработки
  • GearmanTask::data() - Получение данных из обработчика
