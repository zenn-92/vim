OAuth::getLastResponseHeaders — Get headers for last response

Описание

public string OAuth::getLastResponseHeaders ( void )

Get headers for last response.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

A string containing the last response's headers или FALSE в случае возникновения ошибки
