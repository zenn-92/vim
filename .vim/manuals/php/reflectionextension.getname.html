ReflectionExtension::getName

(PHP 5)

ReflectionExtension::getName — Получение имени расширения

Описание

public string ReflectionExtension::getName ( void )

Получает имя расширения.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Имя расширения.

Примеры

Пример #1 Пример использования ReflectionExtension::getName()

<?php
$ext = new ReflectionExtension('mysqli');
var_dump($ext->getName());
?>

Результатом выполнения данного примера будет что-то подобное:

string(6) "mysqli"
