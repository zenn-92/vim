ReflectionParameter::getPosition

(PHP 5 >= 5.2.3)

ReflectionParameter::getPosition — Получение позиции аргумента

Описание

public int ReflectionParameter::getPosition ( void )

Получает позицию аргумента в списке аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Позиция аргумента, слева направо, первый аргумент стоит на позиции #0.
