streamWrapper::stream_flush

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::stream_flush — Сохранение данных вывода

Описание

public bool streamWrapper::stream_flush ( void )

Этот метод вызывается в результате вызова fflush().

Если в потоке есть кэшированые и еще неиспользованые данные, в этом методе самое место передать их на нижележащий уровень.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Должен возвращать TRUE, если кэшированые данные успешно сохранены (или их вообще нет), либо FALSE, если данные не могут быть сохранены.

Примечания

Замечание:

Если метод не реализован, в качестве возвращаемого значения полагается FALSE.
